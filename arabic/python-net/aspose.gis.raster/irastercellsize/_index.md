**Summary:** Describes the size, scale and rotation of a raster cell on a map.

**Module:** [aspose.gis.raster](/psd/python-net/aspose.gis.raster/)

**Full Name:** aspose.gis.raster.IRasterCellSize

**Aspose.PSD Version:** 25.9.0

## **Properties**
| **Name** | **Type** | **Access** | **Description** |
| :- | :- | :- | :- |
| الارتفاع | double | r | يحصل على ارتفاع الخلية أو البكسل، دائمًا موجب. |
| scale_x | double | r | يحصل على المكوّن x لعرض الخلية أو البكسل (مقياس x). |
| scale_y | double | r | يحصل على المكوّن y لارتفاع الخلية أو البكسل (مقياس y)، عادةً سالب. |
| skew_x | double | r | يحصل على المكوّن x لارتفاع الخلية أو البكسل (انحراف x). |
| skew_y | double | r | يحصل على المكوّن y لعرض الخلية أو البكسل (انحراف y). |
| width | double | r | يحصل على عرض الخلية أو البكسل، دائمًا موجب. |
