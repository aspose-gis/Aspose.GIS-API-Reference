## FeatureAttribute.HasCustomDefaultValue property

يحصل على قيمة تشير إلى ما إذا تم استبدال القيمة الافتراضية المحددة مسبقاً لهذه السمة بقيمة مخصصة.

```csharp
public bool HasCustomDefaultValue { get; }
```

### Property Value

`true` إذا كان لهذا الكائن قيمة افتراضية مخصصة؛ وإلا `false`.
