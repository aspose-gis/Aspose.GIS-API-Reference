`Aspose.Gis` مساحة الاسم توفر فئات لإنشاء وتحويل وتعديل بيانات GIS.

## الفئات

| فئة | الوصف |
| --- | --- |
| [AbstractPath](./abstractpath/) | `AbstractPath` هي فئة أساسية للفئات التي تحدد موقعًا فريدًا في بيئة مشابهة لنظام الملفات، مثل نظام ملفات محلي أو تخزين ملفات بعيد أو أرشيف ZIP، من بين أمور أخرى. |
| [AttributesConverterActions](./attributesconverteractions/) | إجراءات اختيارية مع سمات طبقة الوجهة. |
| [ConversionOptions](./conversionoptions/) | خيارات تحويل البيانات بين الصيغ. |
| [DatabaseDriver](./databasedriver/) | محرك لتنسيق يعتمد على قاعدة بيانات محددة. |
| [DatabaseDriverOptions](./databasedriveroptions/) | خيارات لـ [`DatabaseDriver`](../aspose.gis/databasedriver/). |
| [Dataset](./dataset/) | مجموعة البيانات هي مجموعة من مثيلات [`VectorLayer`](../aspose.gis/vectorlayer/). |
| [Driver](./driver/) | فئة أساسية لمحركات بيانات GIS. |
| [DriverOptions](./driveroptions/) | خيارات لـ [`Driver`](../aspose.gis/driver/). |
| [Drivers](./drivers/) | محركات لجميع الصيغ المدعومة. |
| [Extent](./extent/) | صندوق حدود فضائي ثنائي الأبعاد. |
| [Feature](./feature/) | ميزة جغرافية مكوّنة من هندسة وسمات معرفة من قبل المستخدم. |
| [FeatureAttribute](./featureattribute/) | سمة لـ [`Feature`](../aspose.gis/feature/). |
| [FeatureAttributeCollection](./featureattributecollection/) | تحدد [`FeatureAttributeCollection`](../aspose.gis/featureattributecollection/) ما هي السمات المتاحة لـ [`Feature`](../aspose.gis/feature/). |
| [FeaturesSequence](./featuressequence/) | [`FeaturesSequence`](../aspose.gis/featuressequence/) يمثل مجموعة من الميزات المتجهية. |
| [FeatureStyle](./featurestyle/) | الفئة الجذرية المجردة لهرمية أنماط الميزة. |
| [FileDriver](./filedriver/) | محرك لتنسيق مبني على ملف محدد. |
| [GeoConvert](./geoconvert/) | يحوّل الإحداثيات إلى/من الصيغ المختلفة. |
| [GisException](./gisexception/) | الاستثناء الذي يُرمى عندما يحدث خطأ أثناء معالجة بيانات GIS. |
| [License](./license/) | يوفر طرقًا لتفعيل الترخيص للمكوّن. |
| [Metered](./metered/) | يوفر طرقًا لتعيين المفتاح المقيس. |
| [MultiStreamPath](./multistreampath/) | هذه الفئة تعمل مع الصيغ التي تحتوي على عدة ملفات. |
| [NumericFormat](./numericformat/) | [`NumericFormat`](../aspose.gis/numericformat/) تُستخدم لتنسيق الأنواع الرقمية الشائعة في النص. |
| [PrecisionModel](./precisionmodel/) | [`PrecisionModel`](../aspose.gis/precisionmodel/) يحدد عدد الأرقام ذات الدقة في الإحداثي. |
| [RasterDriver](./rasterdriver/) | محرك لتنسيق مبني على raster محدد. |
| [RasterDriverOptions](./rasterdriveroptions/) | خيارات لـ [`RasterDriver`](../aspose.gis/rasterdriver/). |
| [SavingOptions](./savingoptions/) | خيارات لحفظ [`FeaturesSequence`](../aspose.gis/featuressequence/) إلى ملف. |
| [VectorLayer](./vectorlayer/) | يمثل طبقة متجهية. الطبقة المتجهية هي مجموعة من الميزات الجغرافية، مخزنة في ملف. |
