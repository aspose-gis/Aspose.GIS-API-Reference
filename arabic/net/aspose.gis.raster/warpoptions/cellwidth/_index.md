## WarpOptions.CellWidth property

يحدد عرضًا جديدًا لخلية الراستر (بوحدات الإحداثيات المستهدفة). إذا تم تعيين القيمة إلى 0، يتم حساب `CellWidth` تلقائيًا. القيمة الافتراضية هي "0".

```csharp
public double CellWidth { get; set; }
```

## ملاحظات

إذا تم تعيين عرض الخلية إلى 0، ستؤخذ القيمة من عرض الخلية الأصلي أو تُحسب من [`Width`](../width/). لاحظ أن `CellWidth` لا يمكن استخدامه مع [`Width`](../width/).
