## GeoJsonOptions.WriteBoundingBoxes property

يحدد ما إذا كان يجب تضمين معلومات عن نطاق الإحداثيات لأشكال GeoJSON في الكائنات. إذا تم تعيينه إلى `true`، يتم إنشاء عضو \"bbox\" لكل شكل (ليس null) عند إضافته إلى الطبقة. القيمة الافتراضية هي `false`.

```csharp
public bool WriteBoundingBoxes { get; set; }
```
