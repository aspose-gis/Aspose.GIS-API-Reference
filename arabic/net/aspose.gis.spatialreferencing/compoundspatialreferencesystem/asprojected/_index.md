## CompoundSpatialReferenceSystem.AsProjected property

إرجاع تمثيل إسقاطي لهذا النظام. إذا كان هذا النظام المركب يمثل بالفعل نظام إحداثيات إسقاطي، فستكون النتيجة نظام إحداثيات إسقاطي ثلاثي الأبعاد (مع أبعاد X وY والارتفاع). وإلا سيتم إلقاء استثناء.

```csharp
public override ProjectedSpatialReferenceSystem AsProjected { get; }
```
