## CompoundSpatialReferenceSystem.AsGeographic property

إرجاع تمثيل جغرافي لهذا النظام. إذا كان هذا النظام المركب يمثل بالفعل نظام إحداثيات جغرافي، فستكون النتيجة نظام إحداثيات جغرافي ثلاثي الأبعاد (مع أبعاد الطول والعرض والارتفاع). وإلا سيتم إلقاء استثناء.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
