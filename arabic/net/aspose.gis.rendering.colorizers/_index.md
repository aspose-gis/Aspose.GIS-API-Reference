مساحة الاسم `Aspose.Gis.Rendering.Colorizers` توفر فئات لتكوين خلايا الراستر أثناء التصيير.

## الفئات

| فئة | الوصف |
| --- | --- |
| [BandColor](./bandcolor/) | يربط قيم النطاق ومكوّن اللون للفهرس المحدد للنطاق. سيتم استيفاء القيم بين الحد الأدنى والحد الأقصى خطيًا. |
| [MultiBandColor](./multibandcolor/) | المُلوّن متعدد النطاقات يحدد مكوّنات الأحمر والأخضر والأزرق للراستر. |
| [NullColorizer](./nullcolorizer/) | `NullColorizer` لا يرسم شيئًا ويتخطى فعليًا عرض خلية الراستر التي يُطبّق عليها. |
| [RasterColorizer](./rastercolorizer/) | الفئة الجذرية المجردة للمُلوّن الذي يعرض راستر. |
| [SingleBandColor](./singlebandcolor/) | المُلوّن أحادي النطاق يحدد مكوّن رمادي للراستر. |
