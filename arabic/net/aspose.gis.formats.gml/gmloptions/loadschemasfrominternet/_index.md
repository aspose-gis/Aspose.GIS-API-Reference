## GmlOptions.LoadSchemasFromInternet property

يحدد ما إذا كان يُسمح لـ Aspose.GIS بتحميل مخطط XML من الإنترنت. إذا تم تعيينه إلى `false`، فإن المخططات ذات عناوين URI المطلقة التي لا تبدأ بـ 'file://' لن يتم تحميلها. القيمة الافتراضية هي `false`.

```csharp
public bool LoadSchemasFromInternet { get; set; }
```

## ملاحظات

يستخدم Aspose.GIS مخطط XML لملف GML لإنشاء [`FeatureAttributeCollection`](../../../aspose.gis/featureattributecollection/). يمكن توزيع ملفات مخطط XML جنبًا إلى جنب مع ملفات GML أو يمكن أن تكون موجودة على الإنترنت. في الحالة الأولى، تحتاج إلى ضبط هذا الخيار إلى `true` للسماح لـ Aspose.GIS بتحميل مخطط XML.
