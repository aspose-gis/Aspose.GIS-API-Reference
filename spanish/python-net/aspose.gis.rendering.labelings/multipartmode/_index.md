Especifica cómo se renderizan las etiquetas para entidades que contienen geometrías multipartes.

**Module:** [aspose.gis.rendering.labelings](/psd/python-net/aspose.gis.rendering.labelings/)

**Full Name:** aspose.gis.rendering.labelings.MultipartMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nombre del miembro** | **Descripción** |
| :- | :- |
| ALL | Coloca una etiqueta cerca de cada parte de la geometría siempre que haya espacio cerca de la parte. |
| ANY | Coloca una etiqueta para toda la geometría. |
| LARGEST | Coloca una etiqueta para la parte más grande de la geometría siempre que haya espacio para la etiqueta. |
