Especifica un modo de escritura del Sistema de Referencia Espacial (SRS) en la base de datos<br/>            si es un SRS desconocido.

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.SpatialReferenceSystemMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nombre del miembro** | **Descripción** |
| :- | :- |
| SETUP_TO_ZERO | Configura el SRID de una geometría a 'cero' si es un SRS desconocido para la base de datos. |
| THROW_EXCEPTION | Lanza excepciones si es un SRS desconocido para la base de datos. |
| WRITE_IN_SYSTEM_TABLE | Escribe la información del SRS en la tabla del sistema si es un SRS desconocido para la base de datos. |
