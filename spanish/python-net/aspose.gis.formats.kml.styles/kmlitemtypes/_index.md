Especifica cómo se debe mostrar un kml:Feature y su contenido como elementos en una vista de lista.

**Module:** [aspose.gis.formats.kml.styles](/psd/python-net/aspose.gis.formats.kml.styles/)

**Full Name:** aspose.gis.formats.kml.styles.KmlItemTypes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nombre del miembro** | **Descripción** |
| :- | :- |
| CHECK | La visibilidad está vinculada al checkbox del elemento |
| CHECK_HIDE_CHILDREN | Utiliza un checkbox normal para la visibilidad pero no muestra los hijos del elemento en la vista de lista. |
| CHECK_OFF_ONLY | Impide que todos los elementos se vuelvan visibles de una vez—es decir,<br/>            el usuario puede desactivar todo en el Grupo de Contenedores pero no puede activar todo al mismo tiempo. |
| RADIO_FOLDER | Solo uno de los elementos será visible a la vez. |
