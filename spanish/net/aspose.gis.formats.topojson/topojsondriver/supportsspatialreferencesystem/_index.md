
### Valor_devuelto

Valor booleano, que indica si el sistema de referencia espacial especificado es compatible con el controlador.

### Observaciones

Para TopoJSON, el único sistema de referencia espacial admitido es 'nulo', ya que no hay forma de almacenar información del sistema de referencia espacial en el archivo TopoJSON y la especificación TopoJSON no especifica cuál es el sistema de referencia espacial de geometrías en el archivo TopoJSON.
