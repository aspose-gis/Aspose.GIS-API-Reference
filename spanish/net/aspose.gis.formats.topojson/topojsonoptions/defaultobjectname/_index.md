## TopoJsonOptions.DefaultObjectName property

Nombre del objeto donde se ponen las características por defecto.

```csharp
public string DefaultObjectName { get; set; }
```

### Observaciones

Esta es una opción de escritura, no afecta la lectura. TopoJSON puede contener cualquier cantidad de objetos con nombre. Cada uno de estos objetos puede contener múltiples características. Para especificar en qué objeto colocar su característica, use [`ObjectNameAttribute`](../objectnameattribute) propiedad. Si atributo con nombre[`ObjectNameAttribute`](../objectnameattribute) es`null` desarmado para alguna función, esta función se agrega al objeto con el nombre`DefaultObjectName` . Si atributo con nombre[`ObjectNameAttribute`](../objectnameattribute) no está presente en[`Attributes`](../../../aspose.gis/vectorlayer/attributes) colección , todas las características se colocan en el objeto con nombre[`ObjectNameAttribute`](../objectnameattribute) . El valor predeterminado es "sin nombre".
