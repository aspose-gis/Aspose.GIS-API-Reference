## TopoJsonOptions.Transform property

Especifica el objeto de transformación que se usará para cuantificar las coordenadas y los arcos de codificación delta en la salida TopoJSON.

```csharp
public TopoJsonTransform Transform { get; set; }
```

### Observaciones

Esta es una opción de escritura, no afecta la lectura. Esta opción es mutuamente excluyente con[`QuantizationNumber`](../quantizationnumber) - solo una de estas dos opciones puede ser no`null` . Si esto no es`null` - Las coordenadas TopoJSON de salida se cuantifican y los arcos se codifican en delta con el objeto de transformación especificado. Consulte la especificación de TopoJSON para obtener más detalles sobre el objeto de transformación. El valor predeterminado es`null` .
