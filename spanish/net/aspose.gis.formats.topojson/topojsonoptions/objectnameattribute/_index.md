## TopoJsonOptions.ObjectNameAttribute property

Nombre del atributo, que refleja el nombre del objeto que contiene una característica.

```csharp
public string ObjectNameAttribute { get; set; }
```

### Observaciones

TopoJSON puede contener cualquier cantidad de objetos con nombre. Cada uno de estos objetos puede contener múltiples funciones. Consulte la especificación de TopoJSON para obtener más detalles sobre los objetos con nombre. Al leer TopoJSON, esta propiedad especifica qué atributo debe reflejar el nombre del objeto que contiene una característica. Al escribir TopoJSON, esta propiedad especifica qué atributo debe usarse para agrupar características en objects. El valor predeterminado es "topojson_object_name".
