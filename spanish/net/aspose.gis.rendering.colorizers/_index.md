`Aspose.Gis.Rendering.Colorizers` El espacio de nombres proporciona clases para configurar la representación de celdas ráster.

## Clases

| Clase | Descripción |
| --- | --- |
| [BandColor](./bandcolor/) | Asocia valores de banda y un componente de color para el índice de banda especificado. Hay valores de banda entre el mínimo y el máximo que se interpolarán linealmente. |
| [MultiBandColor](./multibandcolor/) | El colorizador multibanda especifica los componentes rojo, verde y azul para un ráster. |
| [NullColorizer](./nullcolorizer/) | El`NullColorizer` no dibuja nada y efectivamente omite la representación de una celda ráster a la que se aplica. |
| [RasterColorizer](./rastercolorizer/) | La clase raíz abstracta para el colorizador que representa un ráster. |
| [SingleBandColor](./singlebandcolor/) | El colorizador de banda única especifica un componente gris para un ráster. |
