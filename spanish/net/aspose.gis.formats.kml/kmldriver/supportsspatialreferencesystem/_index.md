## KmlDriver.SupportsSpatialReferenceSystem method

Determina si el sistema de referencia espacial especificado es compatible con el controlador.

```csharp
public override bool SupportsSpatialReferenceSystem(SpatialReferenceSystem spatialReferenceSystem)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| spatialReferenceSystem | SpatialReferenceSystem | Sistema de referencia espacial. |

### Valor_devuelto

Valor booleano, que indica si el sistema de referencia espacial especificado es compatible con el controlador.

### Observaciones

Para KML, el único sistema de referencia espacial compatible es WGS84.
