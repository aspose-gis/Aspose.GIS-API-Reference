## AltitudeModes enumeration

Modo altitud para geometrías KML.

```csharp
public enum AltitudeModes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | Sin modo altitud. |
| RelativeToGround | `1` | Interpreta la altitud como un valor en metros sobre el suelo. |
| ClampToGround | `2` | Para una cámara, esta configuración también coloca la cámara en relación con el suelo, ya que colocar la cámara exactamente a la altura del terreno significaría que el ojo intersectaría el terreno (y la vista se bloquearía). |
| Absolute | `3` | Interpreta la altitud como un valor en metros sobre el nivel del mar. |
