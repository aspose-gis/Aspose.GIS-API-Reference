## GmlOptions.LoadSchemasFromInternet property

Determina si Aspose.GIS puede cargar el esquema XML de Internet. Si se establece en`false` , los esquemas con URI absolutos que no comiencen con 'file://' no se cargarán. El valor predeterminado es`false` .

```csharp
public bool LoadSchemasFromInternet { get; set; }
```

### Observaciones

Aspose.GIS utiliza el esquema XML del archivo GML para crear[`FeatureAttributeCollection`](../../../aspose.gis/featureattributecollection) . Los archivos de esquema XML se pueden distribuir junto con los archivos GML o se pueden ubicar en Internet. En el primer caso, debe establecer esta opción en`true` para permitir que Aspose.GIS cargue el esquema XML.
