## SldImportOptions.LoadInternetResources property

Determina si Aspose.GIS puede cargar fuentes desde Internet. Si se establece en`false` los esquemas con URI absolutos que no comiencen con 'file://' no se cargarán. El valor predeterminado es`false` .

```csharp
public bool LoadInternetResources { get; set; }
```
