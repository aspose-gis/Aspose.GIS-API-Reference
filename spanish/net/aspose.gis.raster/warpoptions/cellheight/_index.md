## WarpOptions.CellHeight property

Especifica una nueva altura de la celda ráster (en unidades objetivo georreferenciadas). Si el valor se establece en 0, el`CellHeight` se calcula automáticamente. El valor predeterminado es "0".

```csharp
public double CellHeight { get; set; }
```

### Observaciones

Si la altura de la celda se establece en 0, el valor se tomará de la altura de la celda original o se calculará a partir de[`Height`](../height) . Tenga en cuenta que`CellHeight` no se puede usar con[`Height`](../height) .
