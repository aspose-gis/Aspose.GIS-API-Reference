## WarpOptions.Width property

Especifica el ancho del ráster de salida en píxeles y columnas. Si el valor se establece en 0, el ancho se calcula automáticamente. El valor predeterminado es "0".

```csharp
public int Width { get; set; }
```

### Observaciones

Si el ancho se establece en 0, el valor se tomará del ancho original o se calculará a partir de[`CellWidth`](../cellwidth) . Tenga en cuenta que`Width` no se puede usar con[`CellWidth`](../cellwidth) .
