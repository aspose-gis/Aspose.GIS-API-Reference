## WarpOptions.CellWidth property

Especifica un nuevo ancho de la celda ráster (en unidades objetivo georreferenciadas). Si el valor se establece en 0, el`CellWidth` se calcula automáticamente. El valor predeterminado es "0".

```csharp
public double CellWidth { get; set; }
```

### Observaciones

Si el ancho de celda se establece en 0, el valor se tomará del ancho de celda original o se calculará a partir de[`Width`](../width/) . Tenga en cuenta que`CellWidth` no se puede usar con[`Width`](../width/) .
