## WarpOptions.Height property

Especifica la altura del ráster de salida en píxeles y columnas. Si el valor se establece en 0, la altura se calcula automáticamente. El valor predeterminado es "0".

```csharp
public int Height { get; set; }
```

### Observaciones

Si la altura se establece en 0, el valor se tomará de la altura original o se calculará a partir de[`CellHeight`](../cellheight) . Tenga en cuenta que`Height` no se puede usar con[`CellHeight`](../cellheight) .
