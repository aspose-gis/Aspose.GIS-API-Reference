## GeoJsonOptions.WriteBoundingBoxes property

Determina si los objetos GeoJSON deben incluir información sobre el rango de coordenadas para sus geometrías. Si se establece en`true` , se genera un miembro "bbox" para cada geometría (no nula) cuando se agrega a la capa. El valor predeterminado es`false` .

```csharp
public bool WriteBoundingBoxes { get; set; }
```

### Observaciones

Esta es una opción de creación - no afecta la apertura.
