## MapInfoTabOptions.TextStringAttribute property

Especifica el nombre del atributo que representa el texto del objeto gráfico 'Texto'.

```csharp
public string TextStringAttribute { get; set; }
```

### Observaciones

El formato de ficha MapInfo especifica un objeto gráfico de tipo 'Texto'. El objeto gráfico 'Texto' representa una etiqueta en un mapa. Exportamos objetos gráficos 'Text' como a[`Feature`](../../../aspose.gis/feature/) con[`Polygon`](../../../aspose.gis.geometries/polygon/) geometría que delimita la etiqueta. El texto de la etiqueta se exporta como[`FeatureAttribute`](../../../aspose.gis/featureattribute/) . Esta propiedad especifica el nombre del atributo que se utiliza para exportar el texto de la etiqueta. El valor predeterminado es`"cadena de texto"` .
