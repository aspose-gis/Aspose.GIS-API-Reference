## ProjectedSpatialReferenceSystem.DimensionsCount property

Devuelve el recuento de dimensiones en este SRS. Para el SRS proyectado, esto puede ser: dos: si se trata de un SRS proyectado simple. tres: si se trata de un SRS compuesto, que consiste en un SRS proyectado simple, bidimensional y un SRS vertical, que agrega una tercera dimensión.

```csharp
public abstract int DimensionsCount { get; }
```
