## CompoundSpatialReferenceSystem.AsProjected property

Devuelve la representación proyectada de este SRS. Si este SRS compuesto realmente representa un SRS proyectado, el resultado será un SRS proyectado tridimensional (con dimensiones de altura X, Y). De lo contrario, se lanzará una excepción.
