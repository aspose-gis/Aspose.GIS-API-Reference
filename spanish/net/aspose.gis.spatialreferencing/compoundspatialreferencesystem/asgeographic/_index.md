## CompoundSpatialReferenceSystem.AsGeographic property

Devuelve la representación geográfica de este SRS. Si este SRS compuesto realmente representa un SRS geográfico, el resultado será un SRS geográfico tridimensional (con dimensiones de longitud, latitud y altura). De lo contrario, se lanzará una excepción.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
