## SpatialReferenceSystem.TryCreateFromWkt method

```csharp
public static bool TryCreateFromWkt(string wkt, out SpatialReferenceSystem value)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| wkt | String | Cadena WKT. |
| value | SpatialReferenceSystem& | Cuando este método regresa`true` , contiene un SRS creado a partir de WKT; de lo contrario, contiene`null` . |
