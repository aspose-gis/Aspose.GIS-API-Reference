## MultipartMode enumeration

Especifica cómo se representan las etiquetas para entidades que contienen geometrías multiparte.

```csharp
public enum MultipartMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| All | `0` | Coloca una etiqueta cerca de cada parte de la geometría siempre que haya espacio cerca de la parte. |
| Any | `1` | Coloca una etiqueta para toda la geometría. |
| Largest | `2` | Coloca una etiqueta para la mayor parte de la geometría siempre que haya espacio para la etiqueta. |
