## KmlItemTypes enumeration

Especifica cómo se mostrarán una característica kml:Feature y su contenido como elementos en una vista de lista.

```csharp
public enum KmlItemTypes
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| RadioFolder | `0` | Solo uno de los artículos será visible a la vez. |
| Check | `1` | La visibilidad está ligada a la casilla de verificación de su elemento. |
| CheckHideChildren | `2` | Utilice una casilla de verificación normal para la visibilidad, pero no muestre los elementos secundarios del elemento en la vista de lista. |
| CheckOffOnly | `3` | Evita que todos los elementos se hagan visibles a la vez, es decir, el usuario puede apagar todo en el grupo de contenedores pero no puede encenderlo todo al mismo tiempo. |
