## ToLinearGeometry() {#tolineargeometry}

Obtiene una versión no curva aproximada o equivalente de esta geometría usando el valor predeterminado`tolerancia` .

```csharp
public ILineString ToLinearGeometry()
```

### Valor_devuelto

A[`ILineString`](../../ilinestring) que se aproxima o equivale a esta curva. Este es el equivalente de[`ToLinearGeometry`](../tolineargeometry) con predeterminado`tolerancia` . Defecto`tolerancia` el valor de s depende de[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem) de esta geometría:  Para SRS proyectado, la tolerancia es de 0,001 metros (en unidades SRS) Para la tolerancia SRS geográfica es`1e-5` grados (en unidades SRS) Para SRS desconocido, la tolerancia es`1e-5` Para obtener más detalles sobre qué transformaciones se aplican, consulte[`ToLinearGeometry`](../tolineargeometry) especificación.

## ToLinearGeometry(double) {#tolineargeometry_1}

Obtiene una versión no curva aproximada o equivalente de esta geometría usando el`tolerancia` .

```csharp
public ILineString ToLinearGeometry(double tolerance)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| tolerance | Double | El`tolerancia`usar. Se garantiza que el resultado será inferior a`tolerancia` lejos de la geometría curvada , a menos que el número de puntos necesarios para linealizar la geometría exceda el máximo por cuadrante, actualmente igual a 10000 puntos. |

### Valor_devuelto

A[`ILineString`](../../ilinestring) que se aproxima o equivale a esta curva:  Si este objeto es[`ILineString`](../../ilinestring) mismo el resultado es equivalente a este objeto Si este objeto es[`ICircularString`](../../icircularstring) el resultado es la cadena circular linealizada con el especificado*tolerance* Si este objeto es[`ICompoundCurve`](../../icompoundcurve) - todas las curvas de él se linealizan y luego se unen en[`ILineString`](../../ilinestring)

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentOutOfRangeException | `tolerancia` es menor o igual que`0` . |
| InvalidOperationException | Esta geometría no es válida por lo que la operación no se puede completar. |
