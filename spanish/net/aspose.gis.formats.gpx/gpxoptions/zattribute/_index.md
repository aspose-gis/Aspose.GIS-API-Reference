## GpxOptions.ZAttribute property

Determina qué atributo GPX se exportará como coordenada 'Z' de waypoints, puntos de ruta y puntos de track. Si`null` ningún atributo se exportará como coordenada 'Z'. El valor predeterminado es "ele". Los valores posibles son nombres de todos los atributos XML GPX que se pueden representar como dobles (por ejemplo, "velocidad", "magvar", "altura geoide", etc. )

```csharp
public string ZAttribute { get; set; }
```
