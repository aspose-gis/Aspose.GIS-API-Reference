## VectorLayer.ConstructFeature method

Crea (pero no agrega a la capa) una nueva entidad con atributos que coinciden con la colección de atributos de esta capa. Cuando termine de configurar los datos para la entidad, use[`Add`](../add/) para agregar la entidad a la capa.

```csharp
public Feature ConstructFeature()
```

### Valor_devuelto

Una nueva característica.

### Observaciones

Cambios en[`Attributes`](../attributes/) no están permitidos después de llamar a este método.
