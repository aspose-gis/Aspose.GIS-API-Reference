## UseSpatialIndex(string, bool) {#usespatialindex_1}

Carga el índice espacial para acelerar el filtrado por valor de atributos en métodos de filtrado como[`WhereIntersects`](../../featuressequence/whereintersects) y[`NearestTo`](../nearestto). Si el índice no existe lo crea primero. Usar*forceRebuild* para forzar el índice de recreación.

```csharp
public void UseSpatialIndex(string indexPath, bool forceRebuild = false)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| indexPath | String | Ruta al archivo de índice. |
| forceRebuild | Boolean | Ya sea para volver a crear el índice incluso si ya existe. |

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentNullException | camino es`null`. |
| IOException | Se produjo un error de E/S. |
| InvalidOperationException | El índice espacial ya está cargado para esta capa. |
| [GisException](../../gisexception) | El archivo existe y no es un archivo de índice espacial creado por Aspose.GIS. |
