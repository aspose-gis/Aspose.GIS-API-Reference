Determina si el sistema de referencia espacial especificado es compatible con el controlador.

### Valor_devuelto

Valor booleano, que indica si el sistema de referencia espacial especificado es compatible con el controlador. `null` se considera compatible con cualquier controlador.

### Observaciones

Si el sistema de referencia espacial no es compatible y lo pasa a[`CreateLayer`](../createlayer) método, aNotSupportedException será lanzado.
