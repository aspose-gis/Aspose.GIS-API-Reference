## AbstractPath.GetExtension method

Devuelve la extensión de este[`AbstractPath`](../) .

```csharp
public string GetExtension()
```

### Valor_devuelto

La extensión de este[`AbstractPath`](../) (incluyendo el punto ".") o una cadena vacía si el[`AbstractPath`](../) no tiene extensión.

### Ejemplos

Para un`ruta abstracta` con[`Location`](../location/) igual a`"/directorio/archivo.txt"` esta propiedad devuelve`".TXT"` .
