## SpatialReferenceSystemMode enumeration

Especifica un modo de escritura del Sistema de referencia espacial (SRS) en la base de datos si es un SRS desconocido.

```csharp
public enum SpatialReferenceSystemMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| ThrowException | `0` | Genera excepciones si se trata de un SRS desconocido para la base de datos. |
| WriteInSystemTable | `1` | Escriba la información de SRS en la tabla del sistema si es un SRS desconocido para la base de datos. |
| SetupToZero | `2` | Configure el SRID de una geometría en 'cero' si es un SRS desconocido para la base de datos. |
