## NumericFormat.General method

Convierte un número a la notación más compacta de punto fijo o científica, según el tipo de número y si está presente un especificador de precisión. Recomendado para usar.

```csharp
public static NumericFormat General(int precision = 0)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| precision | Int32 | La precisión define el número máximo de dígitos significativos que pueden aparecer en la cadena de resultado. Si la precisión es cero, se utiliza el valor "15". La máxima precisión disponible es "17". |

### Valor_devuelto

El especificador de formato general.

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentOutOfRangeException | El número de dígitos significativos es menor que 0 o mayor que 17. |

### Observaciones

El código interno está generando cadenas de números para WKT a través de:ordinar.ToString("G", CultureInfo.InvariantCulture).
