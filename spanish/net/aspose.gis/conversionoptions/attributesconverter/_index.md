## ConversionOptions.AttributesConverter property

Un convertidor personalizado para atributos. Nos permite renombrar o excluir atributos de destino. Si no`null` se llama para cada atributo de la capa de origen y se espera que lo cambie si es necesario.

```csharp
public IAttributesConverter AttributesConverter { get; set; }
```

### Observaciones

El tipo de datos del atributo también se puede cambiar.
