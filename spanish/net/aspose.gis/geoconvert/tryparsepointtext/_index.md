## GeoConvert.TryParsePointText method

Convierte la cadena que contiene las coordenadas en un objeto IPoint. Un valor de retorno indica si la conversión tuvo éxito o falló.

```csharp
public static bool TryParsePointText(string text, out IPoint point)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| text | String | Una cadena que contiene coordenadas para convertir. La cadena debe contener coordenadas tanto de latitud como de longitud. Las coordenadas deben estar separadas por espacios en blanco, por coma o por punto y coma. |
| point | IPoint& | Cuando este método regresa, contiene el objeto IPoint con coordenadas analizadas, si la conversión fue exitosa, o nulo si la conversión falló. |

### Valor_devuelto

Verdadero si el texto se analizó correctamente; de lo contrario, falso.

### Observaciones

Ejemplos: "80° 151°", "74°50,82', 172°08,21'", "80°;151°", "2CMB", "2CMB6682893142", "2C MB 66828 93142", "WMAQ12405535".
