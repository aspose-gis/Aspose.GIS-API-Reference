## FileGdbOptions.MTolerance property

Tolerancia de ajuste M.

```csharp
public double? MTolerance { get; set; }
```

### Observaciones

Esta es una opción de creación y no afecta la lectura. Este parámetro controla una tolerancia de ajuste utilizada para características avanzadas de ArcGIS. No afecta el comportamiento de Aspose.GIS, pero puede ser utilizado por ArcGIS. La unidad del parámetro es la unidad del sistema de referencia espacial. Si se establece en`null` , se utiliza el valor predeterminado. El valor predeterminado es 0.001.
