## XyzTiles.GetTile method

Carga el mosaico especificado.

```csharp
public WebTile GetTile(int zoom, int x, int y)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| zoom | Int32 | El nivel de zoom para cargar mosaicos. El nivel de zoom más alto es 0. La mayoría de los proveedores de mosaicos tienen alrededor de 22 niveles de zoom máximos. |
| x | Int32 | Una columna x de un mosaico. |
| y | Int32 | Una fila Y de un mosaico. |

### Valor_devuelto

El mosaico web.
