## GeoConvert.TryParsePointText method

Konverterar strängen som innehåller koordinater till IPoint-objekt. Ett returvärde anger om konverteringen lyckades eller misslyckades.

```csharp
public static bool TryParsePointText(string text, out IPoint point)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| text | String | En sträng som innehåller koordinater att konvertera. Strängen bör innehålla både koordinatlatitud och longitud. Koordinater ska avgränsas med blanksteg, komma eller semikolon. |
| point | IPoint& | När den här metoden returnerar, innehåller den IPoint-objektet med analyserade koordinater, om konverteringen lyckades, eller null om konverteringen misslyckades. |

### Returvärde

Sant om text tolkades framgångsrikt, annars False.

### Anmärkningar

Exempel: "80° 151°", "74°50.82', 172°08.21'", "80°;151°", "2CMB", "2CMB6682893142", "2C MB 66828 93142", "050305_d4".
