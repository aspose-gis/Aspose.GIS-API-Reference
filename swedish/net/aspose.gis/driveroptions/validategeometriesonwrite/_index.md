## DriverOptions.ValidateGeometriesOnWrite property

Bestämmer om geometrier ska valideras när de läggs till i lagret. Om satt till`true` ,[`IsValid`](../../../aspose.gis.geometries/geometry/isvalid/) anropas för each geometri när den läggs till i lagret och om valideringen misslyckas ([`IsValid`](../../../aspose.gis.geometries/geometry/isvalid/) är`false` ),[`GisException`](../../gisexception/) kastas.

```csharp
public bool ValidateGeometriesOnWrite { get; set; }
```

### Anmärkningar

Detta är ett skapande alternativ - det påverkar inte öppningen.
