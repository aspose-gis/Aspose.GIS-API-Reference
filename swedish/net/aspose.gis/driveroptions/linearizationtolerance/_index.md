## DriverOptions.LinearizationTolerance property

En tolerans att använda för att linjärisera kurvgeometrier.

```csharp
public double LinearizationTolerance { get; set; }
```

### Fastighetsvärde

Toleransen att övergå till[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/) innan du lägger till geometrier.

### Anmärkningar

Detta är ett skapande alternativ - det påverkar inte öppning. Om drivrutinen inte stöder icke-linjära geometrier linjäriseras de automatiskt. Den här egenskapen anger ett argument till[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/) metod som används för linjärisering.
