`Aspose.Gis` namnutrymme tillhandahåller klasser för att generera, konvertera och ändra GIS-data.

| Klass | Beskrivning |
| --- | --- |
| [AbstractPath](./abstractpath/) | An`AbstractPath` är en basklass för klasser som anger en unik plats i en miljö som liknar ett filsystem, som ett lokalt filsystem, en fjärrfillagring eller ett ZIP-arkiv, bland annat. |
| [AttributesConverterActions](./attributesconverteractions/) | Valfria åtgärder med attribut för destinationslagret. |
| [ConversionOptions](./conversionoptions/) | Alternativ för att konvertera data mellan format. |
| [DatabaseDriver](./databasedriver/) | En drivrutin för ett specifikt databasbaserat format. |

| [Driver](./driver/) | En basklass för drivrutiner till GIS-data. |
| [DriverOptions](./driveroptions/) | Alternativ för en[`Driver`](../aspose.gis/driver/) . |
| [Drivers](./drivers/) | Drivrutiner för alla format som stöds. |
| [Extent](./extent/) | En tvådimensionell rumslig begränsningslåda. |
| [Feature](./feature/) | En geografisk egenskap som består av en geometri och användardefinierade attribut. |
| [FeatureAttribute](./featureattribute/) | Ett attribut för en[`Feature`](../aspose.gis/feature/) . |

| [License](./license/) | Tillhandahåller metoder för att licensiera komponenten. |

| [RasterDriverOptions](./rasterdriveroptions/) | Alternativ för en[`RasterDriver`](../aspose.gis/rasterdriver/) . |
| [SavingOptions](./savingoptions/) | Alternativ för att spara[`FeaturesSequence`](../aspose.gis/featuressequence/) till fil. |
| [VectorLayer](./vectorlayer/) | Representerar ett vektorlager. Ett vektorlager är en samling geografiska egenskaper som lagras i en fil. |
