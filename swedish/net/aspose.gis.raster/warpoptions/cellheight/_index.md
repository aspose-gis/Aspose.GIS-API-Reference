## WarpOptions.CellHeight property

Anger en ny höjd för rastercellen (i målgeorefererade enheter). Om värdet är satt till 0,`CellHeight` beräknas automatiskt. Standardvärdet är "0".

```csharp
public double CellHeight { get; set; }
```

### Anmärkningar

Om cellhöjden är inställd på 0 kommer värdet att tas från den ursprungliga cellhöjden eller beräknas från[`Height`](../height) . Observera att`CellHeight` kan inte användas med[`Height`](../height) .
