## WarpOptions.CellWidth property

Anger en ny bredd på rastercellen (i målgeorefererade enheter). Om värdet är satt till 0,`CellWidth` beräknas automatiskt. Standardvärdet är "0".

```csharp
public double CellWidth { get; set; }
```

### Anmärkningar

Om cellbredden är inställd på 0 kommer värdet att tas från den ursprungliga cellbredden eller beräknas från[`Width`](../width) . Observera att`CellWidth` kan inte användas med[`Width`](../width) .
