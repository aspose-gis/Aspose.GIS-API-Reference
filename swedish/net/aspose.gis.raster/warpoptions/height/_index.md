## WarpOptions.Height property

Anger utmatningsrasterhöjd i pixlar och kolumner. Om värdet är satt till 0, beräknas höjden automatiskt. Standardvärdet är "0".

```csharp
public int Height { get; set; }
```

### Anmärkningar

Om höjden är satt till 0 kommer värdet att tas från den ursprungliga höjden eller beräknas från[`CellHeight`](../cellheight/) . Observera att`Height` kan inte användas med[`CellHeight`](../cellheight/) .
