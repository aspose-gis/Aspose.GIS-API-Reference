## WarpOptions.Width property

Anger utmatningsrasterbredd i pixlar och kolumner. Om värdet är inställt på 0, beräknas bredden automatiskt. Standardvärdet är "0".

```csharp
public int Width { get; set; }
```

### Anmärkningar

Om bredden är inställd på 0 kommer värdet att tas från den ursprungliga bredden eller beräknas från[`CellWidth`](../cellwidth/) . Observera att`Width` kan inte användas med[`CellWidth`](../cellwidth/) .
