## FileGdbOptions.EnsureValidCoordinatesRange property

Om koordinaterna ska vara inom giltigt område.

```csharp
public bool EnsureValidCoordinatesRange { get; set; }
```

### Anmärkningar

Detta är ett skapande alternativ och det påverkar inte läsningen. Om inställt på`true` ett undantag kommer att kastas vid försök att skriva en koordinat med värden utanför giltigt intervall. Om inställt på`false` sådan koordinat kommer att skrivas tyst. Giltigt intervall definieras av[`CoordinatePrecisionGrid`](../coordinateprecisiongrid) . Hänvisa till[`CoordinatePrecisionGrid`](../coordinateprecisiongrid) dokumentation för att läsa om hur giltigt område bestäms från koordinatprecisionsrutnät. Standard är`false` .
