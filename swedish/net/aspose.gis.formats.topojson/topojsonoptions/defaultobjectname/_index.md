## TopoJsonOptions.DefaultObjectName property

Namn på objektet där funktioner placeras som standard.

```csharp
public string DefaultObjectName { get; set; }
```

### Anmärkningar

Det här är ett skrivalternativ - det påverkar inte läsningen. TopoJSON kan innehålla valfritt antal namngivna objekt. Varje sådant objekt kan innehålla flera funktioner. Använd för att ange i vilket objekt din funktion ska placeras[`ObjectNameAttribute`](../objectnameattribute) egenskap. Om attribut med namn[`ObjectNameAttribute`](../objectnameattribute) är`null`eller avaktivera för någon funktion, denna funktion läggs till objekt med namnet`DefaultObjectName` . Om attribut med namn[`ObjectNameAttribute`](../objectnameattribute) finns inte i[`Attributes`](../../../aspose.gis/vectorlayer/attributes) samling, alla funktioner sätts i objekt med namn[`ObjectNameAttribute`](../objectnameattribute) . Standardvärdet är "unnamed".
