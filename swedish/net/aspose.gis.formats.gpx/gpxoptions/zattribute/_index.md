## GpxOptions.ZAttribute property

Bestämmer vilket GPX-attribut som kommer att exporteras som 'Z'-koordinat för waypoints, ruttpunkter och spårpunkter. Om`null` - inget attribut kommer att exporteras som 'Z'-koordinat. Default är "ele". Möjliga värden är namn på alla GPX XML-attribut som kan representeras som dubbla (t.ex. "hastighet", "magvar", "geoidheight" etc. )
