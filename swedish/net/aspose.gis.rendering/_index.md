`Aspose.Gis.Rendering` namnområdet tillhandahåller klasser för att rendera kartor.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [Map](./map/) | Karta är en samling lager som kan renderas ovanpå varandra via[`Renderer`](../aspose.gis.rendering/renderer/) . |
| [MapLayer](./maplayer/) | A`MapLayer` är en basklass för lager inuti[`Map`](../aspose.gis.rendering/map/). |
| [RasterMapLayer](./rastermaplayer/) | Ett lager inuti[`Map`](../aspose.gis.rendering/map/) som representerar ett rasterlagerdata. |
| [RasterMapResampling](./rastermapresampling/) | Den här klassen beskriver hur man samplar om ett rasterlager när man renderar en karta. |
| [Renderer](./renderer/) | En basklass för renderare. |
| [Renderers](./renderers/) | Renderers för alla format som stöds. |
| [VectorMapLayer](./vectormaplayer/) | Ett lager inuti[`Map`](../aspose.gis.rendering/map/) som representerar ett vektorlagerdata. |
