## ProjectedSpatialReferenceSystemParameters.ProjectionMethodIdentifier property

Identifierare av projektionsmetod. Det finns inget standardvärde, du kan ställa in den här parametern till inte`null` värde, om du vill bifoga identifierare till projektion. Om du gör det - det är upp till dig att se till att identifieraren i konsekvent projektionsmetod namn (projektionsmetodnamnet kommer inte att ändras när du ställer in den här egenskapen).

```csharp
public Identifier ProjectionMethodIdentifier { get; set; }
```
