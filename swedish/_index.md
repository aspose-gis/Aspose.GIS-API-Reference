## Aspose.GIS för .NET

{{% alert color="primary" %}} 

![Aspose.GIS för .NET-produktlogotypen](../home_1.png)

Aspose.GIS för .NET är designat specifikt för att göra det enkelt för .NET-utvecklare att arbeta med geospatial data lagrad i olika filformat. Den låter dig skapa, läsa, konvertera geografiska data och rendera kartor utan att behöva installera någon annan GIS-programvara. Aspose.GIS stöder Shapefile, GeoJSON, KML, ESRI File Geodatabase och andra format.
