**Summary:** Describes the size, scale and rotation of a raster cell on a map.

**Module:** [aspose.gis.raster](/psd/python-net/aspose.gis.raster/)

**Full Name:** aspose.gis.raster.IRasterCellSize

**Aspose.PSD Version:** 25.9.0

## **Properties**
| **Name** | **Type** | **Access** | **Beskrivning** |
| :- | :- | :- | :- |
| höjd | double | r | Hämtar cellens eller pixelns höjd, alltid positiv. |
| scale_x | double | r | Hämtar x-komponenten av cellens eller pixelns bredd (x-skala). |
| scale_y | double | r | Hämtar y-komponenten av cellens eller pixelns höjd (y-skala), vanligtvis negativ. |
| skew_x | double | r | Hämtar x-komponenten av cellens eller pixelns höjd (x-skevning). |
| skew_y | double | r | Hämtar y-komponenten av cellens eller pixelns bredd (y-skevning). |
| bredd | double | r | Hämtar cellens eller pixelns bredd, alltid positiv. |
