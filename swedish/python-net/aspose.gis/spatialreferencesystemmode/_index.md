Specificerar ett läge för att skriva Spatial Reference System (SRS) i databasen<br/>            om det är ett okänt SRS.

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.SpatialReferenceSystemMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Medlemsnamn** | **Beskrivning** |
| :- | :- |
| SETUP_TO_ZERO | Ställ in SRID för en geometri till 'zero' om det är ett okänt SRS för databasen. |
| THROW_EXCEPTION | Kasta undantag om det är ett okänt SRS för databasen. |
| WRITE_IN_SYSTEM_TABLE | Skriv SRS-information i systemtabellen om det är ett okänt SRS för databasen. |
