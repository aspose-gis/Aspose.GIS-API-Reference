Specificerar hur etiketter renderas för objekt som innehåller multipart-geometrier.

**Module:** [aspose.gis.rendering.labelings](/psd/python-net/aspose.gis.rendering.labelings/)

**Full Name:** aspose.gis.rendering.labelings.MultipartMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Medlemsnamn** | **Beskrivning** |
| :- | :- |
| ALL | Placera en etikett nära varje del av geometrin så länge det finns utrymme nära delen. |
| ANY | Placera en etikett för hela geometrin. |
| LARGEST | Placera en etikett för den största delen av geometrin så länge det finns utrymme för etiketten. |
