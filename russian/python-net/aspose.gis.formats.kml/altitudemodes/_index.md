Режим высоты для геометрий KML.

**Module:** [aspose.gis.formats.kml](/psd/python-net/aspose.gis.formats.kml/)

**Full Name:** aspose.gis.formats.kml.AltitudeModes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Имя члена** | **Описание** |
| :- | :- |
| ABSOLUTE | Интерпретирует высоту как значение в метрах над уровнем моря. |
| CLAMP_TO_GROUND | Для камеры эта настройка также размещает камеру relativeToGround, <br/>            поскольку размещение камеры точно на высоте местности означало бы, что <br/>            глаз пересекал бы поверхность (и вид был бы заблокирован). |
| NONE | Без режима высоты. |
| RELATIVE_TO_GROUND | Интерпретирует высоту как значение в метрах над землёй. |
