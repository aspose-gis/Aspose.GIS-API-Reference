Указывает, как метки отображаются для объектов, содержащих многосоставные геометрии.

**Module:** [aspose.gis.rendering.labelings](/psd/python-net/aspose.gis.rendering.labelings/)

**Full Name:** aspose.gis.rendering.labelings.MultipartMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Имя члена** | **Описание** |
| :- | :- |
| ALL | Размещает метку рядом с каждой частью геометрии, если рядом с частью есть место. |
| ANY | Размещает одну метку для всей геометрии. |
| LARGEST | Размещает метку для наибольшей части геометрии, если есть место для метки. |
