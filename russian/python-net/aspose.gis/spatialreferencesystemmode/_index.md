Указывает режим записи системы пространственных ссылок (SRS) в базу данных<br/>            если она неизвестна.

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.SpatialReferenceSystemMode

**Aspose.PSD Version:** 25.9.0

| :- | :- |
| SETUP_TO_ZERO | Установить SRID геометрии в 'zero', если для базы данных неизвестна SRS. |
| THROW_EXCEPTION | Выбрасывать исключения, если для базы данных неизвестна SRS. |
| WRITE_IN_SYSTEM_TABLE | Записать информацию о SRS в системную таблицу, если для базы данных неизвестна SRS. |
