Указывает, как kml:Feature и его содержимое должны отображаться в виде элементов в представлении списка.

**Module:** [aspose.gis.formats.kml.styles](/psd/python-net/aspose.gis.formats.kml.styles/)

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Имя члена** | **Описание** |
| :- | :- |
| CHECK | Видимость связана с флажком элемента |
| CHECK_HIDE_CHILDREN | Используйте обычный флажок для управления видимостью, но не отображайте дочерние элементы в представлении списка. |
| CHECK_OFF_ONLY | Предотвращает одновременное отображение всех элементов — то есть,<br/>            пользователь может отключить всё в группе контейнеров, но не может включить всё одновременно. |
| RADIO_FOLDER | Только один из элементов может быть видимым одновременно. |
