## LayeredSymbolizer.RenderingOrder property

Определяет порядок рендеринга. ByFeatures - визуализировать все символизаторы для объекта, затем перейти к следующему объекту.ByLayers - визуализировать все объекты с помощью символизатора, затем перейти к следующему символизатору.

public RenderingOrder RenderingOrder { get; set; }
