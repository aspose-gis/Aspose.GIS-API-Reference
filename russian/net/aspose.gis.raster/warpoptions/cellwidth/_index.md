## WarpOptions.CellWidth property

Указывает новую ширину растровой ячейки (в целевых единицах с географической привязкой). Если установлено значение 0,`CellWidth`автоматически вычисляется. Значение по умолчанию — «0».

```csharp
public double CellWidth { get; set; }
```

### Примечания

Если для ширины ячейки установлено значение 0, значение будет взято из исходной ширины ячейки или вычисляется из[`Width`](../width). Обратите внимание, что`CellWidth`нельзя использовать с[`Width`](../width).
