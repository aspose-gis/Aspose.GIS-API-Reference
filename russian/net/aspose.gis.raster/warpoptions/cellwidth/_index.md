## WarpOptions.CellWidth property

Определяет новую ширину растровой ячейки (в целевых единицах с географической привязкой). Если установлено значение 0,`CellWidth` вычисляется автоматически. Значение по умолчанию — «0».

```csharp
public double CellWidth { get; set; }
```

### Примечания

Если для ширины ячейки установлено значение 0, значение будет взято из исходной ширины ячейки или вычислено из[`Width`](../width) . Обратите внимание, что`CellWidth` нельзя использовать с[`Width`](../width) .
