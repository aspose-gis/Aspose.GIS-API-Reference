## WarpOptions.Height property

Задает высоту выходного растра в пикселях и столбцах. Если установлено значение 0, высота вычисляется автоматически. Значение по умолчанию — «0».

```csharp
public int Height { get; set; }
```

### Примечания

Если для высоты установлено значение 0, значение будет взято из исходной высоты или вычислено из[`CellHeight`](../cellheight). Обратите внимание, что`Height`нельзя использовать с[`CellHeight`](../cellheight).
