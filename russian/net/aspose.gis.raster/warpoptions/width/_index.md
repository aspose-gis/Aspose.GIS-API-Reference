## WarpOptions.Width property

Определяет ширину выходного растра в пикселях и столбцах. Если установлено значение 0, ширина вычисляется автоматически. Значение по умолчанию — «0».

```csharp
public int Width { get; set; }
```

### Примечания

Если для ширины установлено значение 0, значение будет взято из исходной ширины или вычислено из[`CellWidth`](../cellwidth/) . Обратите внимание, что`Width` нельзя использовать с[`CellWidth`](../cellwidth/) .
