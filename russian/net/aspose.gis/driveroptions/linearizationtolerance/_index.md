## DriverOptions.LinearizationTolerance property

Допуск, используемый для линеаризации геометрии кривой.

```csharp
public double LinearizationTolerance { get; set; }
```

### Стоимость имущества

Допуск для перехода к[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/) перед добавлением геометрии.

### Примечания

Это параметры создания - не влияет на открытие. Если драйвер не поддерживает нелинейные геометрии, они автоматически линеаризуются. Это свойство задает аргумент для[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/) метод, который используется для линеаризации.
