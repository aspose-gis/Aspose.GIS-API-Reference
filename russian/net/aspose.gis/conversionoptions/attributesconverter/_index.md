## ConversionOptions.AttributesConverter property

Пользовательский преобразователь атрибутов. Это позволяет нам переименовывать или исключать атрибуты назначения. Если нет`null` он вызывается для каждого атрибута исходного слоя и, как ожидается, изменит его при необходимости.

```csharp
public IAttributesConverter AttributesConverter { get; set; }
```

### Примечания

Также можно изменить тип данных атрибута.
