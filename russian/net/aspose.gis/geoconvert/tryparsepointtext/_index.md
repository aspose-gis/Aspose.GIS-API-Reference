## GeoConvert.TryParsePointText method

Преобразует строку, содержащую координаты, в объект IPoint. Возвращаемое значение указывает, успешно ли выполнено преобразование.

```csharp
public static bool TryParsePointText(string text, out IPoint point)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| text | String | Строка, содержащая координаты для преобразования. Строка должна содержать координаты широты и долготы. Координаты должны быть разделены пробелом, запятой или точкой с запятой. |
| point | IPoint& | Когда этот метод возвращается, содержит объект IPoint с проанализированными координатами, если преобразование прошло успешно, или null, если преобразование завершилось неудачно. |

### Возвращаемое значение

Истинно, если текст был проанализирован успешно, в противном случае — Ложь.

### Примечания

Примеры: "80° 151°", "74°50,82', 172°08,21'", "80°;151°", "2CMB", "2CMB6682893142", "2C MB 66828 93142", "WMAQ12405535".
