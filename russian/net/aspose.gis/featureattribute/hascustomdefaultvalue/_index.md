## FeatureAttribute.HasCustomDefaultValue property

Получает значение, указывающее, было ли предопределенное значение по умолчанию для этого атрибута переопределено пользовательским значением.

```csharp
public bool HasCustomDefaultValue { get; }
```

### Стоимость имущества

`истинный` если этот экземпляр имеет пользовательское значение по умолчанию; в противном случае,`ЛОЖЬ` .

### Примечания

Если толькоObject) был вызван, значение этого свойства равно`ЛОЖЬ` .
