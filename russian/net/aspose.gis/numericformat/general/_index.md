## NumericFormat.General method

Преобразует число в более компактную запись с фиксированной точкой или экспоненциальную запись, в зависимости от типа числа и наличия спецификатора точности. Рекомендуется использовать.

```csharp
public static NumericFormat General(int precision = 0)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| precision | Int32 | Точность определяет максимальное количество значащих цифр, которые могут появиться в строке результата. Если точность равна нулю, используется значение "15". Максимально доступная точность — «17». |

### Возвращаемое значение

Общий спецификатор формата.

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentOutOfRangeException | Количество значащих цифр меньше 0 или больше 17. |

### Примечания

Внутренний код генерирует числовые строки для WKT через: координата.ToString("G", CultureInfo.InvariantCulture).
