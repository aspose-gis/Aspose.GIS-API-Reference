## NumericFormat.RoundTrip property

Преобразует и пытается гарантировать, что числовое значение, преобразованное в строку, анализируется обратно в то же числовое значение.

```csharp
public static NumericFormat RoundTrip { get; }
```

### Примечания

Внутренний код генерирует числовые строки для WKT с помощью: координаты.ToString("R", CultureInfo.InvariantCulture) решения.
