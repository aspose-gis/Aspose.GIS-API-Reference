## AbstractPath.WithExtension method

Возвращает новый[`AbstractPath`](../) с расширением файла, измененным на указанное значение.

```csharp
public virtual AbstractPath WithExtension(string newExtension)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| newExtension | String | Новое расширение. |

### Возвращаемое значение

Новый[`AbstractPath`](../) , который указывает на файл в том же каталоге, но с новым расширением.

### Примечания

Обычно наследник не должен переопределять этот метод. Реализация по умолчанию заменяет расширение и вызывает[`WithLocation`](../withlocation/) .
