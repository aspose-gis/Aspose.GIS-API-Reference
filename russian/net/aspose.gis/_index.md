`Aspose.Gis` пространство имен предоставляет классы для создания, преобразования и изменения данных ГИС.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [AbstractPath](./abstractpath/) | Ан`АннотацияПуть` является базовым классом для классов, указывающих уникальное местоположение в среде, похожей на файловую систему, , например, на локальную файловую систему, удаленное файловое хранилище или ZIP-архив и т. д. |
| [AttributesConverterActions](./attributesconverteractions/) | Дополнительные действия с атрибутами целевого слоя. |
| [ConversionOptions](./conversionoptions/) | Параметры преобразования данных между форматами. |
| [DatabaseDriver](./databasedriver/) | Драйвер для определенного формата базы данных. |
| [DatabaseDriverOptions](./databasedriveroptions/) | Параметры для[`DatabaseDriver`](../aspose.gis/databasedriver/) . |
| [Dataset](./dataset/) | Набор данных — это набор[`VectorLayer`](../aspose.gis/vectorlayer/) экземпляры. |
| [Driver](./driver/) | Базовый класс для драйверов к данным ГИС. |
| [DriverOptions](./driveroptions/) | Параметры для[`Driver`](../aspose.gis/driver/) . |
| [Drivers](./drivers/) | Драйверы для всех поддерживаемых форматов. |
| [Extent](./extent/) | Двумерная пространственная ограничивающая рамка. |
| [Feature](./feature/) | Географический объект, состоящий из геометрии и определенных пользователем атрибутов. |
| [FeatureAttribute](./featureattribute/) | Атрибут[`Feature`](../aspose.gis/feature/) . |
| [FeatureAttributeCollection](./featureattributecollection/) | А[`FeatureAttributeCollection`](../aspose.gis/featureattributecollection/) определяет, какие атрибуты доступны для[`Feature`](../aspose.gis/feature/) . |
| [FeaturesSequence](./featuressequence/) | [`FeaturesSequence`](../aspose.gis/featuressequence/) представляет собой набор векторных объектов. |
| [FeatureStyle](./featurestyle/) | Абстрактный корневой класс иерархии стилей пространственных объектов. |
| [FileDriver](./filedriver/) | Драйвер для определенного формата файла. |
| [GeoConvert](./geoconvert/) | Преобразует координаты в/из различных форматов. |
| [GisException](./gisexception/) | Исключение, возникающее при возникновении ошибки при обработке данных ГИС. |
| [License](./license/) | Предоставляет методы лицензирования компонента. |
| [Metered](./metered/) | Предоставляет методы для установки измеренного ключа. |
| [NumericFormat](./numericformat/) | [`NumericFormat`](../aspose.gis/numericformat/) используются для форматирования общих числовых типов в тексте. |
| [PrecisionModel](./precisionmodel/) | [`PrecisionModel`](../aspose.gis/precisionmodel/) указывает количество значащих цифр в координате. |
| [RasterDriver](./rasterdriver/) | Драйвер для определенного растрового формата. |
| [RasterDriverOptions](./rasterdriveroptions/) | Параметры для[`RasterDriver`](../aspose.gis/rasterdriver/) . |
| [SavingOptions](./savingoptions/) | Варианты сохранения[`FeaturesSequence`](../aspose.gis/featuressequence/) в файл. |
| [VectorLayer](./vectorlayer/) | Представляет собой векторный слой. Векторный слой представляет собой набор географических объектов, хранящихся в файле. |
