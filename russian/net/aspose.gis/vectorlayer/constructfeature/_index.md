## VectorLayer.ConstructFeature method

Создает (но не добавляет к слою) новый объект с атрибутами, соответствующими набору атрибутов этого слоя. Когда вы закончите настройку данных для объекта, используйте[`Add`](../add/) чтобы добавить объект в слой.

```csharp
public Feature ConstructFeature()
```

### Возвращаемое значение

Новая функция.

### Примечания

Заменяется на[`Attributes`](../attributes/) не допускаются после вызова этого метода.
