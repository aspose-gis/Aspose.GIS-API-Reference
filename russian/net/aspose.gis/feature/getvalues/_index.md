## Feature.GetValues method

Возвращает значения всех атрибутов в массиве.

```csharp
public int GetValues(object[] values, object defaultValue = null)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| values | Object[] | Массив, в который копируются значения атрибутов. |
| defaultValue | Object | Возвращаемое значение, если значение атрибута отсутствует (не задано). Значение по умолчанию`null`. Рассмотрите возможность использования 'DBNull.Value» для разделения «не установлено» и «`null` значения. |

### Возвращаемое значение

Скопировано несколько атрибутов.

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | Аргумент`null`. |
| InvalidOperationException | Атрибут не заблокирован. |

### Примечания

Атрибуты значений объекта копируются в массив значений, который передается в качестве параметра. Для атрибутов с неустановленным значением возвращается указанный параметр 'unsetValue'.  Длина массива значений не обязательно должна совпадать с количеством атрибутов в объекте. Если длина массива больше, чем количество атрибутов, все значения атрибутов копируются в массив; если меньше, в массив копируется только число значений атрибута длины массива, начиная со значения атрибута с порядковым номером 0.
