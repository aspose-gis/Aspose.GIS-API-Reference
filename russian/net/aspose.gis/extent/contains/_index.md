```

| Параметр | Тип | Описание |

Значение, указывающее, содержит ли данный экстент аргумент.

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | Аргумент:`null`. |
| ArgumentException | [`SpatialReferenceSystem`](../spatialreferencesystem)этого экстента и аргумента не являются`null`и не равны друг другу. |

### Примечания

Координаты, расположенные на границах этого[`Extent`](../../extent)являются считается содержащимся в этом[`Extent`](../../extent). По этой причине считается, что равные экстенты содержат друг друга.
