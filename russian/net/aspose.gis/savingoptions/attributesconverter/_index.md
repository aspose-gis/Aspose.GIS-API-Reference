## SavingOptions.AttributesConverter property

Пользовательский преобразователь атрибутов. Если нет`null` , он вызывается для каждого атрибута исходного слоя и, как ожидается, изменит его при необходимости.

```csharp
public IAttributesConverter AttributesConverter { get; set; }
```

### Примечания

Имя атрибута и тип данных изменить нельзя.
