## GmlOptions.RestoreSchema property

Определяет, разрешено ли Aspose.GIS анализировать атрибуты в файле Gml, в котором схема XML отсутствует или не может быть загружена. Если установлено значение`true` , Для чтения Aspose.GIS не требуется наличие XML-схемы. Значение по умолчанию:`false` .

```csharp
public bool RestoreSchema { get; set; }
```

### Примечания

Драйвер пытается автоматически проанализировать классы объектов и связанные с ними свойства путем сканирования файла и поиска «известных» объектов для определения организации. Хотя этот подход подвержен ошибкам, его преимущество заключается в том, что он работает с файлами GML даже , если связанные XML-схема утеряна или не может быть загружена из Интернета.
