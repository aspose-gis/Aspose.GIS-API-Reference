## GmlOptions.RestoreSchema property

Определяет, разрешено ли Aspose.GIS анализировать атрибуты в файле Gml, в котором схема XML отсутствует или не может быть загружена. Если установлено значение`true`, для чтения Aspose.GIS не требуется наличие XML-схемы. По умолчанию:`false`.

```csharp
public bool RestoreSchema { get; set; }
```

### Примечания

Драйвер пытается автоматически проанализировать классы объектов и связанные с ними свойства с помощью сканирование файла и поиск «известных» объектов для определения организации. Хотя этот подход подвержен ошибкам, он имеет то преимущество, что работает с файлами GML, даже если соответствующая XML-схема потеряна или не может быть загружена из Интернета.
