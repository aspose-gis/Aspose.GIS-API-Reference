## FileGdbOptions.ZTolerance property

Допуск привязки Z.

```csharp
public double? ZTolerance { get; set; }
```

### Примечания

Это опция создания и не влияет на чтение. Этот параметр управляет допуском привязки, используемым для расширенных функций ArcGIS. Это не влияет на поведение Aspose.GIS, но может использоваться ArcGIS. Единицей параметра является единица пространственной системы отсчета. Если установлено значение`null`, используется значение по умолчанию. Если пространственная система отсчета неизвестна или имеет менее трех измерений, значение по умолчанию равно 0,001. Если пространственная система отсчета имеет три измерения , значение по умолчанию равно 0,001 метру, преобразованному в единицу измерения третьего измерения.
