## FileGdbOptions.XYTolerance property

Допуск привязки по осям X и Y.

```csharp
public double? XYTolerance { get; set; }
```

### Примечания

Это параметр создания, и он не влияет на чтение. Этот параметр управляет допуском привязки, используемым для расширенных функций ArcGIS. Он не влияет на поведение Aspose.GIS, но может использоваться ArcGIS. Единица параметра единица пространственной системы отсчета. Если установлено значение`null` используется значение по умолчанию. Значение по умолчанию зависит от системы пространственной привязки и равно 0,000000008983153 градуса для географических систем или 0,001 метра для проекционных систем (значения преобразуются в единицы системы пространственной привязки). Если система пространственной привязки неизвестна, значение по умолчанию равно 0,001.
