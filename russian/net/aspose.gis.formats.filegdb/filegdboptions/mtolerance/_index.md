## FileGdbOptions.MTolerance property

Допуск привязки M.

```csharp
public double? MTolerance { get; set; }
```

### Примечания

Это параметр создания, и он не влияет на чтение. Этот параметр управляет допуском привязки, используемым для расширенных функций ArcGIS. Он не влияет на поведение Aspose.GIS, но может использоваться ArcGIS. Единица параметра единица пространственной системы отсчета. Если установлено значение`null` , используется значение по умолчанию. По умолчанию 0.001.
