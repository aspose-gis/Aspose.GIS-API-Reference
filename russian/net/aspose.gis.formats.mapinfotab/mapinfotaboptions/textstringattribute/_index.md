## MapInfoTabOptions.TextStringAttribute property

Задает имя атрибута, представляющего текст графического объекта «Текст».

```csharp
public string TextStringAttribute { get; set; }
```

### Примечания

Формат вкладки MapInfo определяет графический объект типа «Текст». Графический объект «Текст» представляет собой метку на карте. Мы экспортируем графические объекты «Текст» как a[`Feature`](../../../aspose.gis/feature)with[`Polygon`](../../../aspose.gis.geometries/polygon)геометрия который ограничивает метку. Текст метки экспортируется как[`FeatureAttribute`](../../../aspose.gis/featureattribute). Это свойство указывает имя атрибута, который используется для экспорта текста этикетки. Значение по умолчанию:` "текстовая строка"` .
