## CsvOptions.ColumnWkt property

Получает или задает имя столбца, содержащего общеизвестный текст для представления геометрии. Значение по умолчанию:`null` .

```csharp
public string ColumnWkt { get; set; }
```

### Примечания

Если значение столбца`null` или`empty string` геометрия, основанная на общеизвестном тексте, не будет создана.
