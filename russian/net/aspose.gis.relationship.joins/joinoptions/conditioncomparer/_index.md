## JoinOptions.ConditionComparer property

Определяет, как объекты двух слоев связаны в запросе, указывая логику, которая будет использоваться при сравнении значений объектов.

```csharp
public IEqualityComparer<object> ConditionComparer { get; set; }
```

### Примечания

По умолчанию определяет, равны ли два объекта:

```csharp
EqualityComparer<object>.Default;  
```
