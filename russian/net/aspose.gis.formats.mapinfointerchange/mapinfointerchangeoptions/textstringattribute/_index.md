## MapInfoInterchangeOptions.TextStringAttribute property

Указывает имя атрибута, представляющего текст графического объекта Text.

```csharp
public string TextStringAttribute { get; set; }
```

### Примечания

Формат обмена MapInfo задает графический объект типа «Текст». Графический объект «Текст» представляет собой метку на карте. Мы экспортируем графические объекты «Текст» как a[`Feature`](../../../aspose.gis/feature/) с[`Polygon`](../../../aspose.gis.geometries/polygon/) геометрия, ограничивающая этикетку. Текст этикетки экспортируется как[`FeatureAttribute`](../../../aspose.gis/featureattribute/) . Это свойство указывает имя атрибута, используемого для экспорта текста этикетки. Значение по умолчанию:`"текстовая строка"` .
