## TopoJsonOptions.ObjectNameAttribute property

Имя атрибута, отражающее имя объекта, содержащего признак.

```csharp
public string ObjectNameAttribute { get; set; }
```

### Примечания

TopoJSON может содержать любое количество именованных объектов. Каждый такой объект может содержать несколько функций. Обратитесь к спецификации TopoJSON для более подробной информации об именованных объектах. При чтении TopoJSON это свойство указывает, какой атрибут должен отражать имя объекта, содержащего функцию. При написании TopoJSON это свойство указывает, какой атрибут следует использовать для группировки функций. в объектах. По умолчанию используется "topojson_object_name".
