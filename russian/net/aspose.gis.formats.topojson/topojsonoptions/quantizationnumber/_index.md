## TopoJsonOptions.QuantizationNumber property

Указывает номер квантования, используемый для квантования координат и дельта-кодирования дуг в выходных данных TopoJSON.

```csharp
public int? QuantizationNumber { get; set; }
```

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentOutOfRangeException | Аргумент меньше двух. |

### Примечания

Это опция записи - не влияет на чтение. Эта опция взаимоисключающая с[`Transform`](../transform)- только одна из этих двух опций может быть не`null`. Если это не`null`- выходные координаты TopoJSON квантуются, а дуги дельта-кодируются с указанным номером квантизации. Число квантования определяет максимальное количество выражаемых значений на размерность в результирующих квантованных координатах; обычно выбирается степень десяти. По умолчанию`null`.
