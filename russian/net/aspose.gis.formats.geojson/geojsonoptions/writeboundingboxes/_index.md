## GeoJsonOptions.WriteBoundingBoxes property

Определяет, следует ли включать в объекты GeoJSON информацию о диапазоне координат для своих геометрий. Если установлено значение`true` , элемент "bbox" создается для каждой геометрии (не нулевой), когда она добавляется к слою. Значение по умолчанию:`false` .

```csharp
public bool WriteBoundingBoxes { get; set; }
```

### Примечания

Это вариант создания - не влияет на открытие.
