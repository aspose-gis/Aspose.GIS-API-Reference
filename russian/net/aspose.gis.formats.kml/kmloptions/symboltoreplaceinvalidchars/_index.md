## KmlOptions.SymbolToReplaceInvalidChars property

Определяет, какой символ будет использоваться для замены недопустимых символов при чтении. Замена пропускается, если значение равно '\0'. По умолчанию значение равно '\0' char.

```csharp
public char SymbolToReplaceInvalidChars { get; set; }
```
