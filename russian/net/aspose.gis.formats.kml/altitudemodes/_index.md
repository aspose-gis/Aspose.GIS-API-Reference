## AltitudeModes enumeration

Режим высоты для геометрии KML.

```csharp
public enum AltitudeModes
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Без высотного режима. |
| RelativeToGround | `1` | Интерпретирует высоту как значение в метрах над землей. |
| ClampToGround | `2` | Для камеры этот параметр также размещает камеру относительно земли, , так как размещение камеры точно на высоте местности будет означать, что глаз будет пересекать местность (и обзор будет заблокирован). |
| Absolute | `3` | Интерпретирует высоту как значение в метрах над уровнем моря. |
