## LinearRing.IsClockwise method

Определяет, имеет ли кольцо обмотку по часовой стрелке.

```csharp
public bool IsClockwise()
```

### Возвращаемое значение

`true` если по часовой стрелке; в противном случае`false`.

### Примечания

Если линейное кольцо неверно (например, самопересекающееся), то результат функции не определен.
