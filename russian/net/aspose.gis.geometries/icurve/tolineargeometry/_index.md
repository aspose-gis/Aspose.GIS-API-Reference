## ToLinearGeometry() {#tolineargeometry}

Получает приблизительную или эквивалентную некривую версию этой геометрии, используя допуск по умолчанию .

```csharp
public ILineString ToLinearGeometry()
```

### Возвращаемое значение

A[`ILineString`](../../ilinestring)что приблизительно или эквивалент этой кривой. Это эквивалент[`ToLinearGeometry`](../tolineargeometry)с по умолчанию` допуск` . Значение по умолчанию` допуска` s зависит от[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem) этой геометрии: &lt;ul&gt;&lt;li&gt; Для проектируемого SRS Допуск составляет 0,001 метра (в единицах SRS) &lt;/li&gt;&lt;li&gt; Для географического SRS Допуск составляет` 1e-5` градусов (в единицах SRS) &lt;/li&gt;&lt;li&gt; Для неизвестного SRS Допуск составляет` 1e-5` &lt;/li&gt;&lt;/ ul&gt; Подробнее о применяемых преобразованиях см. в спецификации[`ToLinearGeometry`](../tolineargeometry).

### Исключения

| исключение | условие |

| InvalidOperationException | Эта геометрия недействительна таким образом, что операция не может быть завершенный. |

## ToLinearGeometry(double) {#tolineargeometry_1}

Получает приблизительную или эквивалентную некривую версию этой геометрии с использованием указанного допуска .

```csharp
public ILineString ToLinearGeometry(double tolerance)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| допуск | Double | Допуск использовать. Результат гарантированно будет меньше` допуска` вдали от криволинейной геометрии, если только количество точек, необходимых для линеаризации геометрии, не превышает максимум на квадрант, в настоящее время равно 10000 точек. |

### Возвращаемое значение

A[`ILineString`](../../ilinestring)что приблизительно или эквивалент этой кривой: &lt;ul&gt; Если этот объект[`ILineString`](../../ilinestring)сам результат эквивалентен этот объект Если этот объект[`ICircularString`](../../icircularstring)результатом является круговая строка, линеаризованная с указанным*tolerance* Если этот объект[`ICompoundCurve`](../../icompoundcurve)- все кривые из него линеаризуются, а затем объединяются в[`ILineString`](../../ilinestring)&lt;/ul&gt;

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentOutOfRangeException | ` допуск` меньше больше или равно` 0` . |
| InvalidOperationException | Эта геометрия недействительна таким образом, что операция не может быть завершенный. |
