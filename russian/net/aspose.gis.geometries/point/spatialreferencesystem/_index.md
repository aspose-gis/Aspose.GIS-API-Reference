## Point.SpatialReferenceSystem property

Получает SpatialReferenceSystem этого экземпляра. Это свойство может быть`null` , is SpatialReferenceSystem is unknown. Назначение новой SpatialReferenceSystem не будет выполнять никакого преобразования координат, изменится только привязка.

```csharp
public override SpatialReferenceSystem SpatialReferenceSystem { get; set; }
```
