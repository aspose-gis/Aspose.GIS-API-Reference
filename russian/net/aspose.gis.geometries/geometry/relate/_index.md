## Geometry.Relate method

Определяет, соответствует ли DE-9IM матрица пересечения этой геометрии и указанной геометрии заданному шаблону.

```csharp
public bool Relate(IGeometry other, string intersectionPatternMatrix)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| other | IGeometry | Геометрия. |
| intersectionPatternMatrix | String | Шаблон для сопоставления. Это должна быть строка длиной 9. Каждый символ строки представляет ожидаемый размер пересечения: &lt;ul&gt;&lt;li&gt; символ 0 - между внутренностями геометрий.&lt;/li&gt;&lt;li&gt;символ 1 - между внутренностью этой геометрии и границей другой геометрии.&lt;/li&gt;&lt;li&gt;символ 2 - между внутренностью этой геометрии и экстерьером другой геометрии .&lt;/li&gt;&lt;li&gt;символ 3 - между границей этой геометрии и внутренней частью другой геометрии.&lt;/li&gt;&lt;li&gt;символ 4 - между границами геометрий.&lt;/li&gt;&lt;li&gt;символ 5 - между границей этой геометрии и экстерьером другой геометрии.&lt;/li&gt;&lt;li&gt;символ 6 - между экстерьером этой геометрии и внутренностью другой геометрии.&lt;/li&gt;&lt;li&gt;символ 7 - между экстерьером этой геометрии и границей другой геометрии .&lt;/li&gt;&lt;li&gt;символ 8 - между экстерьерами геометрий.&lt;/li&gt;&lt;/ul&gt; Возможные значения каждого символа: &lt;ul&gt;&lt; li&gt;* - любое значение;&lt;/li&gt;&lt;li&gt;F - нет пересечения;&lt;/li&gt;&lt;li&gt;T - любое пересечение;&lt;/li&gt;&lt;li&gt;0 - точка пересечения (например, общая точка);&lt;/li&gt;&lt;li&gt;1 - пересечение линий (например, общий отрезок линии);&lt;/li&gt;&lt;li&gt;2 - площадь пересечение (например, общая часть многоугольника);&lt;/li&gt;&lt;/ul&gt; Например, шаблон пересечения "F0*******" означает, что не должно быть пересечения между геометриями интерьеры и пересечение границ геометрий должно быть точкой. См. спецификацию простых функций OpenGIS для получения более подробной информации о шаблоне матрицы пересечения. |

### Возвращаемое значение

`true`если эта матрица пересечений соответствует шаблону;`false`иначе.

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentNullException | *other*is`null`. |
| ArgumentException | Одна из геометрий недействительна, поэтому операция не может быть завершена. |
| ArgumentException | [`SpatialReferenceSystem`](../../igeometry/spatialreferencesystem)геометрий не эквивалентны. Вы можете использовать[`SpatialReferenceSystemTransformation`](../../../aspose.gis.spatialreferencing/spatialreferencesystemtransformation)для преобразования геометрии в одну и ту же пространственную систему отсчета. |

### Примечания

Этот метод строит матрицу пересечений DE-9IM и сопоставляет ее с образцом См. Спецификацию простых функций OpenGIS для получения более подробной информации о матрице пересечения DE-9IM.
