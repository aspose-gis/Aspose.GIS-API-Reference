## CompoundCurve.SpatialReferenceSystem property

Получает SpatialReferenceSystem этого экземпляра. Это свойство может быть`null` , если SpatialReferenceSystem не задана. Назначение новой SpatialReferenceSystem не приведет к преобразованию координат, изменится только привязка.

```csharp
public SpatialReferenceSystem SpatialReferenceSystem { get; set; }
```
