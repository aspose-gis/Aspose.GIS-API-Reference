## CompoundSpatialReferenceSystem.AsProjected property

Возвращает спроецированное представление этой SRS. Если эта составная SRS действительно представляет спроецированную SRS, результатом будет трехмерная спроецированная SRS (с размерами X, Y, высоты). В противном случае будет выдано исключение.

public override ProjectedSpatialReferenceSystem AsProjected { get; }
