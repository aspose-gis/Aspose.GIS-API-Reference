## ProjectedSpatialReferenceSystemParameters.ProjectionMethodIdentifier property

Идентификатор метода проецирования. Значение по умолчанию отсутствует, вы можете установить этот параметр не`null` value, если вы хотите прикрепить идентификатор к проекции. Если вы это сделаете, вы должны убедиться, что идентификатор в проекции согласован с именем method (имя метода проекции не изменится, когда вы установите это свойство).

```csharp
public Identifier ProjectionMethodIdentifier { get; set; }
```
