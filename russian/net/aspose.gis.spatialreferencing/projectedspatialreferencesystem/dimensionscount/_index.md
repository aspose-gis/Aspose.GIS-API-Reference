## ProjectedSpatialReferenceSystem.DimensionsCount property

Возвращает количество измерений в этом SRS. Для проектируемого СКР это может быть: два - если это один проектируемый СКР. три - если это составная СКР, состоящая из одинарной, двумерной, проекционной ССО и вертикальной ССО, то добавляется третье измерение.

public abstract int DimensionsCount { get; }
