## ProjectedSpatialReferenceSystem.DimensionsCount property

Возвращает количество измерений в этой SRS. Для проекционной СКР это может быть: два - если это одиночная проекционная СКР. три - если это составная СКР, которая состоит из одинарной, двумерной, спроецированной ССО и вертикальной ССО, добавляющей третье измерение.

```csharp
public abstract int DimensionsCount { get; }
```
