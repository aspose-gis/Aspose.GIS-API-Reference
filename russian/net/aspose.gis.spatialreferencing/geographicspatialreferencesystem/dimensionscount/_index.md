## GeographicSpatialReferenceSystem.DimensionsCount property

Возвращает количество измерений в этом SRS. Для географической СГД это может быть: два - если это одна географическая СГД. три - если это составная ССО, состоящая из одномерной, двумерной, географической ССО и вертикальной ССО, то добавляется третье измерение.

public abstract int DimensionsCount { get; }
