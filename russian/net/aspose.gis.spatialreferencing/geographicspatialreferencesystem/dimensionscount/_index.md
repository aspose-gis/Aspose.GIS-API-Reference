## GeographicSpatialReferenceSystem.DimensionsCount property

Возвращает количество измерений в этой SRS. Для географической SRS это может быть: два - если это одиночная географическая SRS. три - если это составная SRS, которая состоит из одиночной, двумерной, географической SRS и вертикальной SRS, что добавляет третье измерение.

public abstract int DimensionsCount { get; }
