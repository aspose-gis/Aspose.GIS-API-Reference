### Примеры

WGS 84 Пространственная система отсчета имеет код EPSG 4326, поэтому может содержать идентификатор: Эллипсоид WGS 84 имеет код EPSG 7030 и может содержать идентификатор:

```csharp
new  {  = "EPSG",  = 4326 };
```

new  {  = "EPSG",  = 7030 };
