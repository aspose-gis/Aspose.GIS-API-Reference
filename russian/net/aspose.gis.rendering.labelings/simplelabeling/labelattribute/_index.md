## SimpleLabeling.LabelAttribute property

Имя атрибута для использования в качестве источника меток. Игнорируется, если[`LabelExpression`](../labelexpression/) установлен. Либо`LabelAttribute` или[`LabelExpression`](../labelexpression/) необходимо установить перед рендерингом; InvalidOperationException иначе бросается.

```csharp
public string LabelAttribute { get; set; }
```
