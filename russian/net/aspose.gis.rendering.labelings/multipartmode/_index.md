## MultipartMode enumeration

Указывает, как отображаются метки для объектов, содержащих составную геометрию.

```csharp
public enum MultipartMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| All | `0` | Помещает метку рядом с каждой частью геометрии, если рядом с частью есть место. |
| Any | `1` | Размещает одну метку для всей геометрии. |
| Largest | `2` | Размещает метку для самой большой части геометрии, если есть место для метки. |
