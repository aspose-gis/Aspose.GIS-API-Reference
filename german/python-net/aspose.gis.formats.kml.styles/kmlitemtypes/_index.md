Gibt an, wie ein kml:Feature und dessen Inhalte als Elemente in einer Listenansicht angezeigt werden sollen.

**Module:** [aspose.gis.formats.kml.styles](/psd/python-net/aspose.gis.formats.kml.styles/)

**Full Name:** aspose.gis.formats.kml.styles.KmlItemTypes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Membername** | **Beschreibung** |
| :- | :- |
| CHECK | Die Sichtbarkeit ist an das Kontrollkästchen des Elements gebunden |
| CHECK_HIDE_CHILDREN | Verwenden Sie ein normales Kontrollkästchen für die Sichtbarkeit, aber zeigen Sie die untergeordneten Elemente des Items nicht in der Listenansicht an. |
| CHECK_OFF_ONLY | Verhindert, dass alle Elemente gleichzeitig sichtbar gemacht werden – das heißt,<br/>            kann der Benutzer alles in der Containergruppe ausschalten, aber nicht alles gleichzeitig einschalten. |
| RADIO_FOLDER | Es darf jeweils nur ein Element sichtbar sein. |
