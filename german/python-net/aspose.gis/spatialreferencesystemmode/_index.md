Gibt einen Modus für das Schreiben des Spatial Reference System (SRS) in die Datenbank an<br/>            falls es ein unbekanntes SRS ist.

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.SpatialReferenceSystemMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Membername** | **Beschreibung** |
| :- | :- |
| SETUP_TO_ZERO | Setze die SRID einer Geometrie auf 'null', wenn es ein unbekanntes SRS für die Datenbank ist. |
| THROW_EXCEPTION | Wirf Ausnahmen, wenn es ein unbekanntes SRS für die Datenbank ist. |
| WRITE_IN_SYSTEM_TABLE | Schreibe SRS-Informationen in die Systemtabelle, wenn es ein unbekanntes SRS für die Datenbank ist. |
