Gibt an, wie Beschriftungen für Features gerendert werden, die mehrteilige Geometrien enthalten.

**Module:** [aspose.gis.rendering.labelings](/psd/python-net/aspose.gis.rendering.labelings/)

**Full Name:** aspose.gis.rendering.labelings.MultipartMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Membername** | **Beschreibung** |
| :- | :- |
| ALL | Platziert ein Etikett in der Nähe jedes Teils der Geometrie, solange Platz in der Nähe des Teils vorhanden ist. |
| ANY | Platziert ein Etikett für die gesamte Geometrie. |
| LARGEST | Platziert ein Etikett für das größte Teil der Geometrie, solange Platz für das Etikett vorhanden ist. |
