`Aspose.Gis.Rendering.Symbolizers` Namespace stellt Klassen zum Konfigurieren von Rendering-Vektorfunktionen bereit.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [GeometryGenerator](./geometrygenerator/) | Dekorieren Sie einen Symbolisierer, um die Geometrie des Features vor dem Rendern zu ändern. |
| [LayeredSymbolizer](./layeredsymbolizer/) | Ein Symbolisierer, der mehrere andere Symbolisierer rendert. |
| [MarkerCluster](./markercluster/) | Marker-Cluster-Symbolisierer. |
| [MarkerLine](./markerline/) | Markierungsliniensymbolisierer. |
| [MarkerPatternFill](./markerpatternfill/) | Markierungsmuster-Füllsymbolisierer. |
| [MixedGeometrySymbolizer](./mixedgeometrysymbolizer/) | Wendet den korrekten Symbolisierer auf eine Feature-Geometrie gemäß ihrem Geometrietyp an. |
| [NullVectorSymbolizer](./nullvectorsymbolizer/) | Die`NullSymbolizer` zeichnet nichts und überspringt effektiv das Rendern einer Geometrie, auf die es angewendet wird. |
| [RasterImageMarker](./rasterimagemarker/) | Dieser Symbolisierer rendert ein bereitgestelltes Rasterbild. |
| [Rule](./rule/) | Eine benutzerdefinierte Regel für[`RuleBasedSymbolizer`](../aspose.gis.rendering.symbolizers/rulebasedsymbolizer/) . |
| [RuleBasedSymbolizer](./rulebasedsymbolizer/) | Wendet einen Symbolisierer gemäß benutzerdefinierten Regeln auf Feature-Geometrien an. |
| [SimpleFill](./simplefill/) | Einfacher Polygonsymbolisierer. |
| [SimpleLine](./simpleline/) | Einfacher Liniensymbolisierer. |
| [SimpleMarker](./simplemarker/) | Einfacher Punktsymbolisierer. |
| [VectorSymbolizer](./vectorsymbolizer/) | Die abstrakte Stammklasse für die Symbolisierer, die Vektor-Features rendern. |
