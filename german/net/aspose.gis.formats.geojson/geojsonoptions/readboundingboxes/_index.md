## GeoJsonOptions.ReadBoundingBoxes property

Legt fest, ob Bounding Boxes ('bbox') als Attribute mit Namen 'bbox_0', 'bbox_1' usw. gelesen werden sollen. Standardwert ist`false` . Die[`NestedPropertiesSeparator`](../nestedpropertiesseparator/) string wird verwendet in bbox_0, bbox_1,.. names.

```csharp
public bool ReadBoundingBoxes { get; set; }
```

### Bemerkungen

Dies ist eine Leseoption – sie wirkt sich nicht auf das Schreiben aus.
