## DriverOptions.LinearizationTolerance property

Eine Toleranz zum Linearisieren von Kurvengeometrien.

```csharp
public double LinearizationTolerance { get; set; }
```

### Eigentumswert

Die zu übergebende Toleranz[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry) vor dem Hinzufügen von Geometrien.

### Bemerkungen

Dies ist eine Erstellungsoption – sie wirkt sich nicht auf das Öffnen aus. Wenn der Treiber keine nichtlinearen Geometrien unterstützt, werden sie automatisch linearisiert. Diese Eigenschaft gibt ein Argument für die an[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry) Methode, die zur Linearisierung verwendet wird.
