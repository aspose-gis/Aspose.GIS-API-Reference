## DriverOptions.ValidateGeometriesOnWrite property

Legt fest, ob Geometrien validiert werden sollen, wenn sie der Ebene hinzugefügt werden. Wenn festgelegt auf`true` ,[`IsValid`](../../../aspose.gis.geometries/geometry/isvalid)wird für each -Geometrie aufgerufen, wenn sie der Ebene hinzugefügt wird und wenn die Validierung fehlschlägt ([`IsValid`](../../../aspose.gis.geometries/geometry/isvalid) ist`false` ),[`GisException`](../../gisexception) wird geworfen.

```csharp
public bool ValidateGeometriesOnWrite { get; set; }
```

### Bemerkungen

Dies ist eine Erstellungsoption – sie wirkt sich nicht auf das Öffnen aus.
