## CurvePolygon.GetPointOnSurface method

Findet einen Punkt, der sich garantiert auf diesem gekrümmten Polygon befindet.

```csharp
public override IPoint GetPointOnSurface()
```

### Rückgabewert

Ein Punkt auf diesem gekrümmten Polygon. Ein leerer Punkt, wenn dieses gekrümmte Polygon kein Inneres hat.

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| InvalidOperationException | Das gekrümmte Polygon ist ungültig, sodass der Vorgang nicht abgeschlossen werden kann. |
