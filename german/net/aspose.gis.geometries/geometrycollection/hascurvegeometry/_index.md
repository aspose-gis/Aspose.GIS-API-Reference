## GeometryCollection.HasCurveGeometry property

Ruft einen Wert ab, der angibt, ob diese Geometrie Kurvengeometrie (nicht linear) ist oder enthält.

```csharp
public override bool HasCurveGeometry { get; }
```

### Eigentumswert
