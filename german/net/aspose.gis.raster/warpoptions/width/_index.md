## WarpOptions.Width property

Gibt die Breite des Ausgaberasters in Pixeln und Spalten an. Wenn der Wert auf 0 gesetzt ist, wird die Breite automatisch berechnet. Der Standardwert ist "0".

```csharp
public int Width { get; set; }
```

### Bemerkungen

Wenn die Breite auf 0 gesetzt ist, wird der Wert von der ursprünglichen Breite genommen oder daraus berechnet[`CellWidth`](../cellwidth) . Beachten Sie das`Width` kann nicht mit verwendet werden[`CellWidth`](../cellwidth) .
