## AxisDirection enumeration

Achsrichtung definiert die Richtung, in die die Achse zeigt.

```csharp
public enum AxisDirection
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Invalid | `0` | Standardwert. |
| North | `1` | Achse zeigt nach Norden. |
| South | `2` | Achse zeigt nach Süden. |
| East | `3` | Achse zeigt nach Osten. |
| West | `4` | Achse zeigt nach Westen. |
| Up | `5` | Achse zeigt nach oben. |
| Down | `6` | Achse zeigt nach unten. |
| Other | `7` | Achse, die in eine andere Richtung zeigt. Dies könnte die 'X'-Achse im geozentrischen SRS sein (sie zeigt auf den Nullmeridian). |
