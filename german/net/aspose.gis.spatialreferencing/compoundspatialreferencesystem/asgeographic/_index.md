## CompoundSpatialReferenceSystem.AsGeographic property

Geben Sie die geografische Darstellung dieses SRS zurück. Wenn diese zusammengesetzte SRS tatsächlich eine geografische SRS darstellt, ist das Ergebnis eine dreidimensionale geografische SRS (mit Längen-, Breiten- und Höhendimensionen). Andernfalls wird eine Ausnahme ausgelöst.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
