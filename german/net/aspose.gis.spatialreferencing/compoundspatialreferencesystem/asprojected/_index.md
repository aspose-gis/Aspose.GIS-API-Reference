## CompoundSpatialReferenceSystem.AsProjected property

Projizierte Darstellung dieses SRS zurückgeben. Wenn diese zusammengesetzte SRS tatsächlich eine projizierte SRS darstellt, ist das Ergebnis eine dreidimensionale projizierte SRS (mit X-, Y-, Höhendimensionen). Andernfalls wird eine Ausnahme ausgelöst.
