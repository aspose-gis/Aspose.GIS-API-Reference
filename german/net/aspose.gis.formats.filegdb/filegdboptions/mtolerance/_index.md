## FileGdbOptions.MTolerance property

M Fangtoleranz.

```csharp
public double? MTolerance { get; set; }
```

### Bemerkungen

Dies ist eine Erstellungsoption und wirkt sich nicht auf das Lesen aus. Dieser Parameter steuert eine Fangtoleranz, die für erweiterte ArcGIS-Features verwendet wird. Er wirkt sich nicht auf das Verhalten von Aspose.GIS aus, kann aber von ArcGIS verwendet werden. Die Einheit des Parameters ist die Einheit des Raumbezugssystems. Wenn eingestellt auf`null` , der Standardwert wird verwendet. Der Standardwert ist 0,001.
