## FileGdbOptions.ZTolerance property

Z-Fangtoleranz.

```csharp
public double? ZTolerance { get; set; }
```

### Bemerkungen

Dies ist eine Erstellungsoption und wirkt sich nicht auf das Lesen aus. Dieser Parameter steuert eine Fangtoleranz, die für erweiterte ArcGIS-Features verwendet wird. Er wirkt sich nicht auf das Verhalten von Aspose.GIS aus, kann aber von ArcGIS verwendet werden. Die Einheit des Parameters ist die Einheit des Raumbezugssystems. Wenn eingestellt auf`null` , der Standardwert wird verwendet. Wenn das Raumbezugssystem unknown ist oder weniger als drei Dimensionen hat, ist der Standardwert 0,001. Wenn das Raumbezugssystem drei Dimensionen hat , ist der Standardwert 0,001 Meter, konvertiert in die Einheit der dritten Dimension.
