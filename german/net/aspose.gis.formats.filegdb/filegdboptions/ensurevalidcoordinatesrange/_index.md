## FileGdbOptions.EnsureValidCoordinatesRange property

Ob Koordinaten im gültigen Bereich liegen sollen.

```csharp
public bool EnsureValidCoordinatesRange { get; set; }
```

### Bemerkungen

Dies ist eine Erstellungsoption und wirkt sich nicht auf das Lesen aus. Wenn auf eingestellt`true` Beim Versuch, eine Koordinate mit Werten außerhalb des gültigen Bereichs zu schreiben, wird eine Ausnahme ausgelöst. Wenn eingestellt`false` solche Koordinate wird still geschrieben. Gültiger Bereich wird definiert durch[`CoordinatePrecisionGrid`](../coordinateprecisiongrid) . Beziehen auf[`CoordinatePrecisionGrid`](../coordinateprecisiongrid) Dokumentation, um zu lesen, wie der gültige Bereich aus dem Koordinatengenauigkeitsgitter bestimmt wird. Standard ist`false` .
