## MarkerCluster.FeaturesBasedConfiguration property

Μια επιστροφή κλήσης που χρησιμοποιείται για τη διαμόρφωση αυτού του συμβολοποιητή πριν από την απόδοση ενός κέντρου συμπλέγματος.

```csharp
public Action<IEnumerable<Feature>, MarkerCluster> FeaturesBasedConfiguration { get; set; }
```

### Παρατηρήσεις

Αυτή η επανάκληση καλείται πριν από την απόδοση κάθε κέντρου συμπλέγματος. Αποδέχεται ένα χαρακτηριστικό που πρόκειται να αποδοθεί και έναν κλώνο αυτού του συμβολιστή. Αλλάζοντας τις ιδιότητες του κλώνου, είναι δυνατό να ενημέρωση της συμπεριφοράς του συμβολιστή με βάση τα χαρακτηριστικά των χαρακτηριστικών.
