## GmlDriver.CanCreateDatasets property

```csharp
public override bool CanCreateDatasets { get; }
```

### Αξία περιουσίας

`true` , εάν αυτό το πρόγραμμα οδήγησης μπορεί να δημιουργήσει σύνολα δεδομένων. σε διαφορετική περίπτωση,`false` .
