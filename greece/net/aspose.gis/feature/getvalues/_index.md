## Feature.GetValues method

Επιστρέφει τις τιμές για όλα τα χαρακτηριστικά ενός πίνακα.

```csharp
public int GetValues(object[] values, object defaultValue = null)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| values | Object[] | Ο πίνακας στον οποίο θα αντιγραφούν οι τιμές των χαρακτηριστικών. |
| defaultValue | Object | Η τιμή που πρέπει να επιστραφεί εάν η τιμή του χαρακτηριστικού λείπει (δεν έχει οριστεί). Η προεπιλεγμένη τιμή είναι`null`. Σκεφτείτε να χρησιμοποιήσετε το 'DBNull.Τιμή' για τον διαχωρισμό των 'unset' και '`null` τιμές. |

### Επιστρεφόμενη Αξία

Αντιγράφηκε ένας αριθμός χαρακτηριστικών.

### Εξαιρέσεις

| εξαίρεση | κατάσταση |
| --- | --- |
| ArgumentNullException | Το επιχείρημα είναι`null`. |
| InvalidOperationException | Το χαρακτηριστικό δεν είναι κλειδωμένο. |

### Παρατηρήσεις

Τα χαρακτηριστικά τιμών του χαρακτηριστικού αντιγράφονται στον πίνακα τιμών που μεταβιβάζεται ως παράμετρος. Για χαρακτηριστικά με μη καθορισμένη τιμή, επιστρέφεται η καθορισμένη παράμετρος 'unsetValue'.  Το μήκος του πίνακα τιμών δεν χρειάζεται να ταιριάζει με τον αριθμό των χαρακτηριστικών στο χαρακτηριστικό. Εάν το μήκος του πίνακα είναι μεγαλύτερο από τον αριθμό των χαρακτηριστικών, όλες οι τιμές των χαρακτηριστικών αντιγράφονται στον πίνακα· εάν είναι μικρότερο, μόνο ο αριθμός του μήκους του πίνακα των τιμών των χαρακτηριστικών αντιγράφεται στον πίνακα, ξεκινώντας από την τιμή χαρακτηριστικού με το τακτικό 0.
