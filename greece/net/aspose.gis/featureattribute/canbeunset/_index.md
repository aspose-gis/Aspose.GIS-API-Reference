## FeatureAttribute.CanBeUnset property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν η τιμή για αυτό το χαρακτηριστικό μπορεί να παραλειφθεί.

```csharp
public bool CanBeUnset { get; set; }
```

### Αξία περιουσίας

`true` εάν αυτή η περίπτωση μπορεί να καταργηθεί. σε διαφορετική περίπτωση,`false` .

### Εξαιρέσεις

| εξαίρεση | κατάσταση |
| --- | --- |
| InvalidOperationException | Το χαρακτηριστικό είναι κλειδωμένο. |
