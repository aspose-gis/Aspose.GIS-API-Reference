## SpatialReferenceSystemMode enumeration

Καθορίζει έναν τρόπο λειτουργίας του Συστήματος Χωρικής Αναφοράς (SRS) που γράφει στη βάση δεδομένων εάν πρόκειται για άγνωστο SRS.

```csharp
public enum SpatialReferenceSystemMode
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| ThrowException | `0` | Περάστε εξαιρέσεις εάν πρόκειται για άγνωστο SRS για τη βάση δεδομένων. |
| WriteInSystemTable | `1` | Γράψτε πληροφορίες SRS στον πίνακα συστήματος εάν είναι άγνωστο SRS για τη βάση δεδομένων. |
| SetupToZero | `2` | Ρυθμίστε το SRID μιας γεωμετρίας στο «μηδέν» εάν είναι ένα άγνωστο SRS για τη βάση δεδομένων. |
