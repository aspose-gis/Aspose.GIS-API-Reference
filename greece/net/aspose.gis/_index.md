`Aspose.Gis` Ο χώρος ονομάτων παρέχει κλάσεις για τη δημιουργία, τη μετατροπή και την τροποποίηση δεδομένων GIS.

| Τάξη | Περιγραφή |
| --- | --- |
| [AbstractPath](./abstractpath/) | Αν`AbstractPath` είναι μια βασική κλάση για κλάσεις που καθορίζουν μια μοναδική τοποθεσία σε ένα περιβάλλον παρόμοιο με ένα σύστημα αρχείων, όπως ένα τοπικό σύστημα αρχείων, μια απομακρυσμένη αποθήκευση αρχείων ή ένα αρχείο ZIP, μεταξύ άλλων. |
| [AttributesConverterActions](./attributesconverteractions/) | Προαιρετικές ενέργειες με χαρακτηριστικά του επιπέδου προορισμού. |

| [DatabaseDriver](./databasedriver/) | Ένα πρόγραμμα οδήγησης για μια συγκεκριμένη μορφή που βασίζεται σε βάση δεδομένων. |
| [DatabaseDriverOptions](./databasedriveroptions/) | Επιλογές για α[`DatabaseDriver`](../aspose.gis/databasedriver/) . |

| [Feature](./feature/) | Ένα γεωγραφικό χαρακτηριστικό που αποτελείται από μια γεωμετρία και χαρακτηριστικά που καθορίζονται από τον χρήστη. |
| [FeatureAttribute](./featureattribute/) | Ένα χαρακτηριστικό του α[`Feature`](../aspose.gis/feature/) . |

| [GeoConvert](./geoconvert/) | Μετατρέπει τις συντεταγμένες σε/από τις διαφορετικές μορφές. |
| [GisException](./gisexception/) | Η εξαίρεση που προκύπτει όταν παρουσιαστεί σφάλμα κατά την επεξεργασία δεδομένων GIS. |
| [License](./license/) | Παρέχει μεθόδους για την άδεια χρήσης του στοιχείου. |

| [NumericFormat](./numericformat/) | [`NumericFormat`](../aspose.gis/numericformat/) χρησιμοποιούνται για τη μορφοποίηση κοινών αριθμητικών τύπων σε κείμενο. |

| [VectorLayer](./vectorlayer/) | Αντιπροσωπεύει ένα διανυσματικό επίπεδο. Ένα διανυσματικό επίπεδο είναι μια συλλογή από γεωγραφικά χαρακτηριστικά, αποθηκευμένα σε ένα αρχείο. |
