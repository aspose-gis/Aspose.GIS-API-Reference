## WarpOptions.Height property

Καθορίζει το ύψος εξόδου ράστερ σε pixel και στήλες. Εάν η τιμή οριστεί σε 0, το ύψος υπολογίζεται αυτόματα. Η προεπιλεγμένη τιμή είναι "0".

```csharp
public int Height { get; set; }
```

### Παρατηρήσεις

Εάν το ύψος έχει οριστεί στο 0, η τιμή θα ληφθεί από το αρχικό ύψος ή θα υπολογιστεί από[`CellHeight`](../cellheight/) . Σημειώστε ότι`Height` δεν μπορεί να χρησιμοποιηθεί με[`CellHeight`](../cellheight/) .
