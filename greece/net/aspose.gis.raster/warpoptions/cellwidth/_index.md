## WarpOptions.CellWidth property

Καθορίζει ένα νέο πλάτος του κελιού ράστερ (σε μονάδες γεωαναφοράς στόχου). Εάν η τιμή οριστεί σε 0, το`CellWidth` υπολογίζεται αυτόματα. Η προεπιλεγμένη τιμή είναι "0".

```csharp
public double CellWidth { get; set; }
```

### Παρατηρήσεις

Εάν το πλάτος κελιού οριστεί σε 0, η τιμή θα ληφθεί από το αρχικό πλάτος κελιού ή θα υπολογιστεί από[`Width`](../width/) . Σημειώστε ότι`CellWidth` δεν μπορεί να χρησιμοποιηθεί με[`Width`](../width/) .
