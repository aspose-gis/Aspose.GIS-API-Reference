## GeoJsonOptions.WriteBoundingBoxes property

Καθορίζει εάν τα αντικείμενα GeoJSON θα πρέπει να περιλαμβάνονται πληροφορίες σχετικά με το εύρος συντεταγμένων για τις Γεωμετρίες του. Εάν έχει οριστεί σε`true` , ένα μέλος "bbox" δημιουργείται για κάθε γεωμετρία (όχι null) όταν προστίθεται στο επίπεδο. Η προεπιλεγμένη τιμή είναι`false` .

```csharp
public bool WriteBoundingBoxes { get; set; }
```
