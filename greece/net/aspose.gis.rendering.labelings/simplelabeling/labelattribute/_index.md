## SimpleLabeling.LabelAttribute property

Όνομα χαρακτηριστικού για χρήση ως πηγή ετικετών. Αγνοήθηκε αν[`LabelExpression`](../labelexpression/) έχει οριστεί. Είτε`LabelAttribute` ή[`LabelExpression`](../labelexpression/) πρέπει να ρυθμιστεί πριν από την απόδοση; InvalidOperationException πετιέται αλλιώς.

```csharp
public string LabelAttribute { get; set; }
```
