## SimpleLabeling.FeatureBasedConfiguration property

Μια επιστροφή κλήσης που χρησιμοποιείται για τη διαμόρφωση αυτής της ετικέτας πριν από το χειρισμό μιας δυνατότητας.

```csharp
public Action<Feature, SimpleLabeling> FeatureBasedConfiguration { get; set; }
```

### Παρατηρήσεις

Αυτή η επανάκληση καλείται πριν από την επισήμανση κάθε δυνατότητας. Αποδέχεται ένα χαρακτηριστικό που πρόκειται να χαρακτηριστεί και έναν κλώνο αυτής της ετικέτας. Αλλάζοντας τις ιδιότητες του κλώνου, είναι δυνατό να ενημέρωση της συμπεριφοράς της ετικέτας με βάση τα χαρακτηριστικά του χαρακτηριστικού.
