## CsvOptions.HasAttributeNames property

Καθορίζει εάν υπάρχει μια σειρά κεφαλίδας με ονόματα χαρακτηριστικών. Η προεπιλογή είναι`true` .

```csharp
public bool HasAttributeNames { get; set; }
```

### Παρατηρήσεις

Σε περίπτωση`true` ένα[`Attributes`](../../../aspose.gis/vectorlayer/attributes/) θα ονομαστεί σύμφωνα με τη σειρά κεφαλίδας. Διαφορετικά θα ονομαστεί ως:`πεδίο_1, πεδίο_2, κ.λπ.`
