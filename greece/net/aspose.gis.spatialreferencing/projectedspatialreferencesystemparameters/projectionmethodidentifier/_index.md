## ProjectedSpatialReferenceSystemParameters.ProjectionMethodIdentifier property

Αναγνωριστικό μεθόδου προβολής. Δεν υπάρχει προεπιλεγμένη τιμή, μπορείτε να ορίσετε αυτήν την παράμετρο σε όχι`null` τιμή, εάν θέλετε να επισυνάψετε αναγνωριστικό στην προβολή. Εάν το κάνετε - εξαρτάται από εσάς να διασφαλίσετε ότι το αναγνωριστικό στο όνομα συνεπούς προβολής method (το όνομα της μεθόδου προβολής δεν θα αλλάξει όταν ορίσετε αυτήν την ιδιότητα).

```csharp
public Identifier ProjectionMethodIdentifier { get; set; }
```
