## ProjectedSpatialReferenceSystemParameters.AddProjectionParameter method

Προσθέτει την παράμετρο προβολής σε αυτό το SRS. Εάν έχει ήδη προστεθεί παράμετρος με τέτοιο όνομα - ενημερώστε την.

```csharp
public void AddProjectionParameter(string parameterName, double value)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| parameterName | String | Όνομα παραμέτρου προβολής. |
| value | Double | Τιμή παραμέτρου. Η μονάδα αξίας πρέπει να είναι μέσα[`LinearUnit`](../linearunit/) ή[`AngularUnit`](../../geographicspatialreferencesystem/angularunit/) του[`Base`](../base/) . |

### Εξαιρέσεις

| εξαίρεση | κατάσταση |
| --- | --- |
| ArgumentNullException | *parameterName* είναι μηδενικό. |
