## CompoundSpatialReferenceSystem.AsProjected property

Επιστρέψτε την προβλεπόμενη αναπαράσταση αυτού του SRS. Εάν αυτή η ένωση SRS αντιπροσωπεύει πράγματι ένα προβαλλόμενο SRS, το αποτέλεσμα θα είναι τρισδιάστατο προβαλλόμενο SRS (με X, Y, διαστάσεις ύψους). Διαφορετικά θα γίνει εξαίρεση.

```csharp
public override ProjectedSpatialReferenceSystem AsProjected { get; }
```
