## CompoundSpatialReferenceSystem.AsGeographic property

Επιστροφή γεωγραφικής αναπαράστασης αυτού του SRS. Εάν αυτή η ένωση SRS αντιπροσωπεύει πράγματι ένα γεωγραφικό SRS, το αποτέλεσμα θα είναι τρισδιάστατο γεωγραφικό SRS (με γεωγραφικό μήκος, γεωγραφικό πλάτος, διαστάσεις ύψους). Διαφορετικά θα γίνει εξαίρεση.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
