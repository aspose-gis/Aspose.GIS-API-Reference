`Aspose.Gis.SpatialReferencing` Ο χώρος ονομάτων παρέχει κλάσεις για εργασία με χωρικές αναφορές (συστήματα αναφοράς συντεταγμένων).

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [Axis](./axis/) | Ένας άξονας περιγράφει μια διάσταση του SRS. |

| [CompoundSpatialReferenceSystem](./compoundspatialreferencesystem/) | Η ένωση SRS ενώνει δύο υποκείμενα SRS, κανένα από τα οποία δεν μπορεί να είναι σύνθετο. |

| [GeocentricSpatialReferenceSystemParameters](./geocentricspatialreferencesystemparameters/) | Παράμετροι για τη δημιουργία γεωκεντρικών SRS. Οι παράμετροι έχουν λογικές προεπιλογές, επομένως θα πρέπει να εκχωρήσετε μόνο μερικές από αυτές. Εάν εκχωρήσετε`null` σε οποιαδήποτε παράμετρο, θα χρησιμοποιηθεί μια προεπιλεγμένη τιμή. |
| [GeographicDatum](./geographicdatum/) | Το γεωγραφικό δεδομένο συσχετίζει το γεωγραφικό μήκος και γεωγραφικό πλάτος με συγκεκριμένο σημείο στη γη. |
| [GeographicSpatialReferenceSystem](./geographicspatialreferencesystem/) | Ένα γεωγραφικό SRS είναι ένα SRS που βασίζεται στο γεωγραφικό μήκος και γεωγραφικό πλάτος. Ένα γεωγραφικό SRS μπορεί να είναι δισδιάστατο ή τρισδιάστατο. Εάν το γεωγραφικό SRS είναι τρισδιάστατο, τότε είναι στην πραγματικότητα ένα σύνθετο SRS δισδιάστατου SRS και κάθετου SRS. |

| [Identifier](./identifier/) | Αντιπροσωπεύει ένα αναγνωριστικό - μια αναφορά σε εξωτερική περιγραφή ενός αντικειμένου. Εάν δημιουργήσετε ένα SRS από το WKT,[`Identifier`](../aspose.gis.spatialreferencing/identifier/) αντιστοιχεί στη λέξη-κλειδί "AUTHORITY". |
| [LocalDatum](./localdatum/) | Υποδεικνύει τη μέθοδο που χρησιμοποιείται για μετρήσεις σε τοπικό χωρικό σύστημα αναφοράς. |

| [PrimeMeridian](./primemeridian/) | Ο PrimeMeridian αντιπροσωπεύει έναν μεσημβρινό στον οποίο το γεωγραφικό μήκος ορίζεται να είναι 0. |
| [ProjectedSpatialReferenceSystem](./projectedspatialreferencesystem/) | Το Προβαλλόμενο SRS είναι αποτέλεσμα εφαρμογής μιας προβολής σε γεωγραφικό SRS. Ένα προβαλλόμενο SRS μπορεί να είναι δισδιάστατο ή τρισδιάστατο. Εάν το προβαλλόμενο SRS είναι τρισδιάστατο, τότε είναι στην πραγματικότητα ένα σύνθετο SRS από δισδιάστατο προβαλλόμενο SRS και μονοδιάστατο κατακόρυφο SRS. |
| [ProjectedSpatialReferenceSystemParameters](./projectedspatialreferencesystemparameters/) | Παράμετροι για τη δημιουργία προβαλλόμενου SRS. Ορισμένες από τις παραμέτρους έχουν προεπιλογές. Ορισμένες παράμετροι έχουν λογικές προεπιλογές, επομένως δεν χρειάζεται να τις εκχωρήσετε μόνο. Εάν εκχωρήσετε`null` σε αυτές τις παραμέτρους, θα χρησιμοποιηθεί μια προεπιλεγμένη τιμή. [`ProjectionMethodName`](../aspose.gis.spatialreferencing/projectedspatialreferencesystemparameters/projectionmethodname/) και[`Base`](../aspose.gis.spatialreferencing/projectedspatialreferencesystemparameters/base/) δεν έχετε προεπιλογές - πρέπει να εκχωρήσετε κάποια μη`null` τιμή σε αυτές τις ιδιότητες. |
| [Projection](./projection/) | Αντιπροσωπεύει μια μέθοδο προβολής με παραμέτρους, που μετατρέπει (γεωγραφικό μήκος, γεωγραφικό πλάτος) σε (x, y). |
| [SpatialReferenceSystem](./spatialreferencesystem/) | Το χωρικό σύστημα αναφοράς χαρτογραφεί συντεταγμένες σε μέρη στη Γη. Υπάρχουν διάφοροι τύποι SRS, βλ.[`Type`](../aspose.gis.spatialreferencing/spatialreferencesystem/type/) . Επιπλέον, εάν είναι τύπος SRSGeographic ή Projected το SRS μπορεί να είναι σύνθετο ή απλό, βλ[`IsCompound`](../aspose.gis.spatialreferencing/spatialreferencesystem/iscompound/) . |
| [SpatialReferenceSystemTransformation](./spatialreferencesystemtransformation/) | Ο μετασχηματισμός χωρικού συστήματος αναφοράς μετασχηματίζει γεωμετρίες από σύστημα χωρικής αναφοράς πηγής σε σύστημα χωρικής αναφοράς στόχου. |
| [TransformationException](./transformationexception/) | Η εξαίρεση μετασχηματισμού δημιουργείται όταν παρουσιάζεται σφάλμα κατά τον μετασχηματισμό της συντεταγμένης ή κατά τη δημιουργία του μετασχηματισμού. |
