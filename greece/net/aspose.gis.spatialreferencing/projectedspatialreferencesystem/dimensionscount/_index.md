## ProjectedSpatialReferenceSystem.DimensionsCount property

Επιστρέφει τον αριθμό των διαστάσεων σε αυτό το SRS. Για προβαλλόμενο SRS αυτό μπορεί να είναι: δύο - εάν αυτό είναι μονό προβαλλόμενο SRS. τρία - εάν αυτό είναι σύνθετο SRS, το οποίο αποτελείται από μονό, δισδιάστατο, προβαλλόμενο SRS και κάθετο SRS, που προσθέτει τρίτη διάσταση.

public abstract int DimensionsCount { get; }
