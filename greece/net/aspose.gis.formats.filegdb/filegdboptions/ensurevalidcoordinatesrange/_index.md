## FileGdbOptions.EnsureValidCoordinatesRange property

Εάν οι συντεταγμένες πρέπει να βρίσκονται σε έγκυρο εύρος.

```csharp
public bool EnsureValidCoordinatesRange { get; set; }
```

### Παρατηρήσεις

Αυτή είναι μια επιλογή δημιουργίας και δεν επηρεάζει την ανάγνωση. Εάν έχει οριστεί σε`true` θα γίνει μια εξαίρεση στην προσπάθεια να γραφτεί μια συντεταγμένη με τιμές εκτός έγκυρου εύρους. Εάν έχει οριστεί σε`false` μια τέτοια συντεταγμένη θα γραφτεί σιωπηλά. Το έγκυρο εύρος ορίζεται από[`CoordinatePrecisionGrid`](../coordinateprecisiongrid/) . Αναφέρομαι σε[`CoordinatePrecisionGrid`](../coordinateprecisiongrid/) τεκμηρίωση για να διαβάσετε πώς προσδιορίζεται το έγκυρο εύρος από το πλέγμα ακριβείας συντεταγμένων. Η προεπιλογή είναι`false` .
