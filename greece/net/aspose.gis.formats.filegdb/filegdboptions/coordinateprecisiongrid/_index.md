## FileGdbOptions.CoordinatePrecisionGrid property

Ένα πλέγμα ακριβείας συντεταγμένων για χρήση στο νέο στρώμα.

```csharp
public FileGdbCoordinatePrecisionGrid CoordinatePrecisionGrid { get; set; }
```

### Παρατηρήσεις

Αυτή είναι μια επιλογή δημιουργίας και δεν επηρεάζει την ανάγνωση. Το πλέγμα ακριβείας συντεταγμένων ορίζει τον έγκυρο τομέα και την ανάλυση των συντεταγμένων στο FileGDB. Όταν αυτή η ιδιότητα έχει οριστεί σε`null` χρησιμοποιούνται προεπιλεγμένες τιμές. Ανατρ[`FileGdbCoordinatePrecisionGrid`](../../filegdbcoordinateprecisiongrid/) τεκμηρίωση για περισσότερες λεπτομέρειες σχετικά με τις συντεταγμένες πλέγματα ακριβείας.
