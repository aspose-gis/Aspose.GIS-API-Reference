## FileGdbCoordinatePrecisionGrid.XYScale property

Λαμβάνει ή ορίζει την κλίμακα των συντεταγμένων X και Y. Εάν έχει οριστεί σε`null` χρησιμοποιείται η προεπιλογή.

```csharp
public double? XYScale { get; set; }
```
