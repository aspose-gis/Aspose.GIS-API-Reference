## FileGdbCoordinatePrecisionGrid.MScale property

Λαμβάνει ή ορίζει την κλίμακα της συντεταγμένης M. Εάν έχει οριστεί σε`null` χρησιμοποιείται η προεπιλογή.

```csharp
public double? MScale { get; set; }
```

### Παρατηρήσεις
