Modalità di altitudine per le geometrie KML.

**Module:** [aspose.gis.formats.kml](/psd/python-net/aspose.gis.formats.kml/)

**Full Name:** aspose.gis.formats.kml.AltitudeModes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nome membro** | **Descrizione** |
| :- | :- |
| ABSOLUTE | Interpreta l'altitudine come valore in metri sopra il livello del mare. |
| CLAMP_TO_GROUND | Per una fotocamera, questa impostazione posiziona anche la fotocamera relativeToGround, <br/>            poiché posizionare la fotocamera esattamente all'altezza del terreno significherebbe che <br/>            l'occhio intersecherebbe il terreno (e la visuale sarebbe bloccata). |
| NESSUNO | Senza modalità di altitudine. |
| RELATIVE_TO_GROUND | Interpreta l'altitudine come valore in metri sopra il terreno. |
