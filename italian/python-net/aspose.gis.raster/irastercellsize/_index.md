**Summary:** Describes the size, scale and rotation of a raster cell on a map.

**Module:** [aspose.gis.raster](/psd/python-net/aspose.gis.raster/)

**Full Name:** aspose.gis.raster.IRasterCellSize

**Aspose.PSD Version:** 25.9.0

## **Properties**
| **Name** | **Type** | **Access** | **Descrizione** |
| :- | :- | :- | :- |
| altezza | double | r | Restituisce l'altezza della cella o del pixel, sempre positiva. |
| scale_x | double | r | Restituisce la componente x della larghezza della cella o del pixel (x-scale). |
| scale_y | double | r | Restituisce la componente y dell'altezza della cella o del pixel (y-scale), tipicamente negativa. |
| skew_x | double | r | Restituisce la componente x dell'altezza della cella o del pixel (x-skew). |
| skew_y | double | r | Restituisce la componente y della larghezza della cella o del pixel (y-skew). |
| width | double | r | Restituisce la larghezza della cella o del pixel, sempre positiva. |
