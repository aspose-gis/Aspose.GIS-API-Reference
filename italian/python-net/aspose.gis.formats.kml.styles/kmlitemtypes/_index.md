Specifica come un kml:Feature e i suoi contenuti devono essere visualizzati come elementi in una vista elenco.

**Module:** [aspose.gis.formats.kml.styles](/psd/python-net/aspose.gis.formats.kml.styles/)

**Full Name:** aspose.gis.formats.kml.styles.KmlItemTypes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nome membro** | **Descrizione** |
| :- | :- |
| CHECK | La visibilità è legata alla casella di controllo dell'elemento |
| CHECK_HIDE_CHILDREN | Utilizza una casella di controllo normale per la visibilità ma non visualizzare i figli dell'elemento nella vista elenco. |
| CHECK_OFF_ONLY | Previene che tutti gli elementi vengano resi visibili contemporaneamente—cioè,<br/>            l'utente può disattivare tutto nel Container Group ma non può attivare tutto allo stesso tempo. |
| RADIO_FOLDER | Solo uno degli elementi deve essere visibile alla volta. |
