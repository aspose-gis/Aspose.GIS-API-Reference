Specifica come le etichette sono renderizzate per le feature che contengono geometrie multipart.

**Module:** [aspose.gis.rendering.labelings](/psd/python-net/aspose.gis.rendering.labelings/)

**Full Name:** aspose.gis.rendering.labelings.MultipartMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nome membro** | **Descrizione** |
| :- | :- |
| ALL | Posiziona un'etichetta vicino a ogni parte della geometria finché c'è spazio vicino alla parte. |
| ANY | Posiziona un'etichetta per l'intera geometria. |
| LARGEST | Posiziona un'etichetta per la parte più grande della geometria finché c'è spazio per l'etichetta. |
