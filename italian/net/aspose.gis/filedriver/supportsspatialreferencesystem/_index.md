## FileDriver.SupportsSpatialReferenceSystem method

Determina se il sistema di riferimento spaziale specificato è supportato dal driver.

```csharp
public abstract bool SupportsSpatialReferenceSystem(SpatialReferenceSystem spatialReferenceSystem)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| spatialReferenceSystem | SpatialReferenceSystem | Sistema di riferimento spaziale. |

### Valore di ritorno

Valore booleano, che indica se il sistema di riferimento spaziale specificato è supportato dal driver. `null` è considerato supportato da qualsiasi driver.

### Osservazioni

Se il sistema di riferimento spaziale non è supportato e lo si passa a[`CreateLayer`](../createlayer) metodo, aNotSupportedException verrà lanciato.
