```

| --- | --- | --- |
| path | String | Percorso del set di dati. |
| driver | FileDriver | Driver da usare. |

### Valore di ritorno

### Eccezioni

| eccezione | condizione |

| IOException | Si è verificato un errore di I/O. |
| NotSupportedException | Il driver non può aprire i set di dati. |
