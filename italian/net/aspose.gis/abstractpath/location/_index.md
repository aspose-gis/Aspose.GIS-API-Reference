## AbstractPath.Location property

Ottiene una rappresentazione di stringa della posizione di this`AbstractPath` .

```csharp
public abstract string Location { get; }
```

### Osservazioni

Due qualsiasi[`AbstractPath`](../) è uguale`Location` puntano allo stesso file o directory. La rappresentazione di stringa deve utilizzare[`Separator`](../separator/) per separare i livelli di directory. Ad esempio, per il file system locale una posizione è un percorso del file system, per i BLOB di Azure può essere un percorso del BLOB all'interno di un contenitore BLOB.
