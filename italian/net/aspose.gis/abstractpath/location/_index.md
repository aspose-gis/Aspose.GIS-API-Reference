### Osservazioni

Due qualsiasi[`AbstractPath`](../../abstractpath) è uguale`Location` punta allo stesso file o directory. La rappresentazione di stringa deve essere utilizzata[`Separator`](../separator) per separare i livelli di directory. Ad esempio, per il file system locale una posizione è un percorso del file system, per i BLOB di Azure può essere un percorso del BLOB all'interno di un contenitore BLOB.
