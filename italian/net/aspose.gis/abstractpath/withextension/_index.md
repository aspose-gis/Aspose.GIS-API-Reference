## AbstractPath.WithExtension method

Restituisce un nuovo[`AbstractPath`](../) con l'estensione del file modificata nel valore specificato.

```csharp
public virtual AbstractPath WithExtension(string newExtension)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| newExtension | String | Una nuova estensione. |

### Valore di ritorno

Un nuovo[`AbstractPath`](../) , che punta a un file nella stessa directory, ma con una nuova estensione.

### Osservazioni

Di solito, un erede non dovrebbe sovrascrivere questo metodo. L'implementazione predefinita sostituisce l'interno e le chiamate[`WithLocation`](../withlocation/) .
