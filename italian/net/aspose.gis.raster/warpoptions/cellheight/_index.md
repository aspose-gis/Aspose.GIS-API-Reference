## WarpOptions.CellHeight property

Specifica una nuova altezza della cella raster (in unità target georeferenziate). Se il valore è impostato su 0, il`CellHeight` viene calcolato automaticamente. Il valore predefinito è "0".

```csharp
public double CellHeight { get; set; }
```

### Osservazioni

Se l'altezza della cella è impostata su 0, il valore sarà preso dall'altezza della cella originale o calcolato da[`Height`](../height) . Nota che`CellHeight` non può essere utilizzato con[`Height`](../height) .
