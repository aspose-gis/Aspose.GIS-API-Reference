## WarpOptions.Height property

Specifica l'altezza raster di output in pixel e colonne. Se il valore è impostato su 0, l'altezza viene calcolata automaticamente. Il valore predefinito è "0".

```csharp
public int Height { get; set; }
```

### Osservazioni

Se l'altezza è impostata su 0, il valore sarà preso dall'altezza originale o calcolato da[`CellHeight`](../cellheight) . Nota che`Height` non può essere utilizzato con[`CellHeight`](../cellheight) .
