`Aspose.Gis.Rendering.Symbolizers` namespace fornisce classi per la configurazione delle funzionalità del vettore di rendering.

## Classi

| Classe | Descrizione |
| --- | --- |
| [GeometryGenerator](./geometrygenerator/) | Decora un simbolo per modificare la geometria dell'elemento prima del rendering. |
| [LayeredSymbolizer](./layeredsymbolizer/) | Un simbolo che rende molti altri simboli. |
| [MarkerCluster](./markercluster/) | Simbolo del gruppo di marcatori. |
| [MarkerLine](./markerline/) | Simbolo della linea di marcatura. |
| [MarkerPatternFill](./markerpatternfill/) | Simbolo di riempimento del modello di marcatore. |
| [MixedGeometrySymbolizer](./mixedgeometrysymbolizer/) | Applica il simbolo corretto alla geometria di una feature in base al suo tipo di geometria. |
| [NullVectorSymbolizer](./nullvectorsymbolizer/) | Il`Simbolo nullo` non disegna nulla ed effettivamente salta il rendering di una geometria a cui è applicato. |
| [RasterImageMarker](./rasterimagemarker/) | Questo simbolo visualizza un'immagine raster fornita. |
| [Rule](./rule/) | Una regola definita dall'utente per[`RuleBasedSymbolizer`](../aspose.gis.rendering.symbolizers/rulebasedsymbolizer/) . |
| [RuleBasedSymbolizer](./rulebasedsymbolizer/) | Applica un simbolo alle geometrie delle caratteristiche in base alle regole definite dall'utente. |
| [SimpleFill](./simplefill/) | Simbolo poligono semplice. |
| [SimpleLine](./simpleline/) | Simbolo di linea semplice. |
| [SimpleMarker](./simplemarker/) | Simbolo punto semplice. |
| [VectorSymbolizer](./vectorsymbolizer/) | La classe radice astratta per i simboli che visualizzano le caratteristiche vettoriali. |
