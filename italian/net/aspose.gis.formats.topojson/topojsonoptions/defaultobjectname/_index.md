## TopoJsonOptions.DefaultObjectName property

Nome dell'oggetto in cui vengono inserite le caratteristiche per impostazione predefinita.

```csharp
public string DefaultObjectName { get; set; }
```

### Osservazioni

Questa è un'opzione di scrittura - non influisce sulla lettura. TopoJSON può contenere un numero qualsiasi di oggetti con nome. Ciascuno di questi oggetti può contenere più funzioni. Per specificare in quale oggetto mettere la tua caratteristica, usa [`ObjectNameAttribute`](../objectnameattribute) proprietà. Se attributo con nome[`ObjectNameAttribute`](../objectnameattribute) è`null` non impostato per alcune funzionalità, questa funzionalità viene aggiunta all'oggetto con il nome`DefaultObjectName` . Se attributo con nome[`ObjectNameAttribute`](../objectnameattribute) non è presente in[`Attributes`](../../../aspose.gis/vectorlayer/attributes) , tutte le caratteristiche vengono messe in oggetto con il nome[`ObjectNameAttribute`](../objectnameattribute) . Il valore predefinito è "senza nome".
