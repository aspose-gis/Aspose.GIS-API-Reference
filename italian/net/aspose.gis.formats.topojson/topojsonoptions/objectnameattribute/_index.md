## TopoJsonOptions.ObjectNameAttribute property

Nome dell'attributo, che riflette il nome dell'oggetto che contiene una caratteristica.

```csharp
public string ObjectNameAttribute { get; set; }
```

### Osservazioni

TopoJSON può contenere un numero qualsiasi di oggetti con nome. Ciascuno di questi oggetti può contenere più funzioni. Fare riferimento alla specifica TopoJSON per maggiori dettagli sugli oggetti denominati. Durante la lettura di TopoJSON, questa proprietà specifica quale attributo deve riflettere il nome dell'oggetto che contiene una funzionalità. Quando si scrive TopoJSON, questa proprietà specifica quale attributo deve essere utilizzato per raggruppare le funzionalità negli oggetti. L'impostazione predefinita è "topojson_object_name".
