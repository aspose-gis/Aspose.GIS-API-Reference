## ProjectedSpatialReferenceSystemParameters.ProjectionMethodIdentifier property

Identificativo del metodo di proiezione. Non esiste un valore predefinito, è possibile impostare questo parametro su not`null` value, se vuoi allegare l'identificatore alla proiezione. Se lo fai, sta a te assicurarti che l'identificatore sia coerente con il nome del metodo di proiezione (il nome del metodo di proiezione non cambierà quando imposti questa proprietà).

```csharp
public Identifier ProjectionMethodIdentifier { get; set; }
```
