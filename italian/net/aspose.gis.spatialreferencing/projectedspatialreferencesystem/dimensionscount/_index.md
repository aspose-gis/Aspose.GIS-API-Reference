## ProjectedSpatialReferenceSystem.DimensionsCount property

Restituisce il conteggio delle dimensioni in questo SRS. Per SRS proiettato questo può essere: due - se si tratta di SRS proiettato singolo. tre - se si tratta di SRS composto, che consiste in SRS singolo, bidimensionale, proiettato e SRS verticale, che aggiunge la terza dimensione.

```csharp
public abstract int DimensionsCount { get; }
```
