## CompoundSpatialReferenceSystem.AsGeographic property

Restituisce la rappresentazione geografica di questo SRS. Se questo SRS composto rappresenta effettivamente un SRS geografico, il risultato sarà sarà un SRS geografico tridimensionale (con dimensioni di longitudine, latitudine, altezza). In caso contrario verrà generata un'eccezione.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
