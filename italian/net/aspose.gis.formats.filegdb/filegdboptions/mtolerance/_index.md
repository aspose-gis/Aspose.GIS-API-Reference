## FileGdbOptions.MTolerance property

Tolleranza snap M.

```csharp
public double? MTolerance { get; set; }
```

### Osservazioni

Questa è un'opzione di creazione e non influisce sulla lettura. Questo parametro controlla una tolleranza di aggancio utilizzata per le funzionalità avanzate di ArcGIS. Non influisce sul comportamento di Aspose.GIS, ma può essere utilizzato da ArcGIS. L'unità del parametro è l'unità del sistema di riferimento spaziale. Se impostato su`null` , viene utilizzato il valore predefinito. Il valore predefinito è 0.001.
