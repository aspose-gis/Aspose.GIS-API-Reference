## GeoJsonOptions.WriteBoundingBoxes property

Determina se gli oggetti GeoJSON devono includere informazioni sull'intervallo di coordinate per le relative geometrie. Se impostato su`true` , viene generato un membro "bbox" per ciascuna geometria (non null) quando viene aggiunta al livello. Il valore predefinito è`false` .

```csharp
public bool WriteBoundingBoxes { get; set; }
```
