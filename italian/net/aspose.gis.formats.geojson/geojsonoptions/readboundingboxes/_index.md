## GeoJsonOptions.ReadBoundingBoxes property

Determina se i Bounding Box ('bbox') devono essere letti come attributi con un nome 'bbox_0', 'bbox_1', ecc. Il valore predefinito è`false` . Il[`NestedPropertiesSeparator`](../nestedpropertiesseparator/) la stringa è usata in bbox_0, bbox_1,.. names.

public bool ReadBoundingBoxes { get; set; }
