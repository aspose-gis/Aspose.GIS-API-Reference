## CsvOptions.HasAttributeNames property

Determina se esiste una riga di intestazione con i nomi degli attributi. Il valore predefinito è`true` .

```csharp
public bool HasAttributeNames { get; set; }
```

### Osservazioni

In caso di`true` UN[`Attributes`](../../../aspose.gis/vectorlayer/attributes/) sarà denominato in base alla riga di intestazione. Altrimenti verrà denominato come:`campo_1, campo_2, ecc.`
