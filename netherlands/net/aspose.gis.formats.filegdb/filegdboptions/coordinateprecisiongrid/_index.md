## FileGdbOptions.CoordinatePrecisionGrid property

Een coördinatenprecisieraster om te gebruiken in een nieuwe laag.

```csharp
public FileGdbCoordinatePrecisionGrid CoordinatePrecisionGrid { get; set; }
```

### Opmerkingen

Dit is een aanmaakoptie en heeft geen invloed op het lezen. Het coördinatenprecisieraster definieert het geldige domein en de resolutie van coördinaten in FileGDB. Wanneer deze eigenschap is ingesteld op`null` standaardwaarden worden gebruikt. Zie[`FileGdbCoordinatePrecisionGrid`](../../filegdbcoordinateprecisiongrid/) documentatie voor meer details over coördinaten precisierasters.
