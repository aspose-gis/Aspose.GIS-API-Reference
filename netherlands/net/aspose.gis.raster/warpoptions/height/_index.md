## WarpOptions.Height property

Specificeert de hoogte van het uitvoerraster in pixels en kolommen. Als de waarde is ingesteld op 0, wordt de hoogte automatisch berekend. De standaardwaarde is "0".

```csharp
public int Height { get; set; }
```

### Opmerkingen

Als de hoogte is ingesteld op 0, wordt de waarde genomen van de oorspronkelijke hoogte of berekend[`CellHeight`](../cellheight/) . Merk op dat`Height` kan niet mee gebruikt worden[`CellHeight`](../cellheight/) .
