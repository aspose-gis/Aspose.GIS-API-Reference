## WarpOptions.Width property

Specificeert uitvoerrasterbreedte in pixels en kolommen. Als de waarde is ingesteld op 0, wordt de breedte automatisch berekend. De standaardwaarde is "0".

```csharp
public int Width { get; set; }
```

### Opmerkingen

Als de breedte is ingesteld op 0, wordt de waarde overgenomen van de oorspronkelijke breedte of berekend op basis van[`CellWidth`](../cellwidth/) . Merk op dat`Width` kan niet mee gebruikt worden[`CellWidth`](../cellwidth/) .
