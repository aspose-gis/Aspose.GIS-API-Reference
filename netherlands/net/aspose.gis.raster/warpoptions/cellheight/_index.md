## WarpOptions.CellHeight property

Specificeert een nieuwe hoogte van de rastercel (in eenheden met geoverwijzingen). Als de waarde is ingesteld op 0, wordt de`CellHeight` wordt automatisch berekend. De standaardwaarde is "0".

```csharp
public double CellHeight { get; set; }
```

### Opmerkingen

Als de celhoogte is ingesteld op 0, wordt de waarde overgenomen van de oorspronkelijke celhoogte of berekend op basis van[`Height`](../height/) . Merk op dat`CellHeight` kan niet mee gebruikt worden[`Height`](../height/) .
