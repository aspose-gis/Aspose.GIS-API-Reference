## WarpOptions.CellWidth property

Specificeert een nieuwe breedte van de rastercel (in eenheden met geoverwijzingen). Als de waarde is ingesteld op 0, wordt de`CellWidth` wordt automatisch berekend. De standaardwaarde is "0".

```csharp
public double CellWidth { get; set; }
```

### Opmerkingen

Als de celbreedte is ingesteld op 0, wordt de waarde overgenomen van de oorspronkelijke celbreedte of berekend op basis van[`Width`](../width/) . Merk op dat`CellWidth` kan niet mee gebruikt worden[`Width`](../width/) .
