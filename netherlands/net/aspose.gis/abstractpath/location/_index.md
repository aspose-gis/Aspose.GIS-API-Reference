## AbstractPath.Location property

Krijgt een tekenreeksrepresentatie van de locatie hiervan`AbstractPath` .

```csharp
public abstract string Location { get; }
```

### Opmerkingen

Elke twee[`AbstractPath`](../) dat is gelijk`Location` verwijzen naar hetzelfde bestand of dezelfde map. De tekenreeksrepresentatie moet gebruiken[`Separator`](../separator/) om mapniveaus te scheiden. Voor het lokale bestandssysteem is een locatie bijvoorbeeld een bestandssysteempad, voor Azure-blobs kan het een pad zijn naar de blob binnen een blob-container.
