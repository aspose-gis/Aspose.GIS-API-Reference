## SpatialReferenceSystemMode enumeration

Specificeert een modus voor het schrijven van Spatial Reference System (SRS) in database als het een onbekende SRS is.

```csharp
public enum SpatialReferenceSystemMode
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| ThrowException | `0` | Uitzonderingen genereren als het een onbekende SRS voor de database is. |
| WriteInSystemTable | `1` | Schrijf SRS-info in de systeemtabel als het een onbekende SRS voor de database is. |
| SetupToZero | `2` | Stel de SRID van een geometrie in op 'nul' als het een onbekende SRS is voor de database. |
