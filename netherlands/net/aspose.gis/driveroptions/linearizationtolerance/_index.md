## DriverOptions.LinearizationTolerance property

Een tolerantie om te gebruiken om krommegeometrieën te lineariseren.

```csharp
public double LinearizationTolerance { get; set; }
```

### Eigendoms-waarde

De tolerantie om naar toe te gaan[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/) voordat u geometrieën toevoegt.

### Opmerkingen

Dit is een aanmaakoptie - het heeft geen invloed op het openen. Als het stuurprogramma geen niet-lineaire geometrieën ondersteunt, worden ze automatisch gelineariseerd. Deze eigenschap specificeert een argument voor de[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/) methode die wordt gebruikt voor linearisatie.
