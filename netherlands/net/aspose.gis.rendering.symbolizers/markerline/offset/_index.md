## MarkerLine.Offset property

Specificeert de offset vanaf de oorspronkelijke lijn. Voor een positieve afstand zal de offset zich aan de linkerkant van de invoerlijn bevinden (ten opzichte van de richting van de lijn). Voor een negatieve afstand zal deze aan de rechterkant zijn.

```csharp
public Measurement Offset { get; set; }
```
