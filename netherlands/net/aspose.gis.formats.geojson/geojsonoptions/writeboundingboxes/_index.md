## GeoJsonOptions.WriteBoundingBoxes property

Bepaalt of GeoJSON-objecten informatie moeten bevatten over het coördinatenbereik voor zijn geometrieën. Indien ingesteld op`true` , wordt een lid "bbox" gegenereerd voor elke geometrie (niet null) wanneer deze wordt toegevoegd aan de laag. Standaardwaarde is`false` .

```csharp
public bool WriteBoundingBoxes { get; set; }
```

### Opmerkingen

Dit is een aanmaakoptie - het heeft geen invloed op het openen.
