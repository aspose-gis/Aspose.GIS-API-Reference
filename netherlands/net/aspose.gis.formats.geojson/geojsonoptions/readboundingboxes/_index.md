## GeoJsonOptions.ReadBoundingBoxes property

Bepaalt of begrenzingsvakken ('bbox') moeten worden gelezen als attributen met een naam 'bbox_0', 'bbox_1', etc. Standaardwaarde is`false` . De[`NestedPropertiesSeparator`](../nestedpropertiesseparator/) string wordt gebruikt in bbox_0, bbox_1,.. namen.

```csharp
public bool ReadBoundingBoxes { get; set; }
```

### Opmerkingen

Dit is een leesoptie - het heeft geen invloed op schrijven.
