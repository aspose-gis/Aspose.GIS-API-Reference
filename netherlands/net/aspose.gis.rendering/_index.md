`Aspose.Gis.Rendering` naamruimte biedt klassen voor het weergeven van kaarten.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [Map](./map/) | Map is een verzameling van lagen die over elkaar kunnen worden weergegeven via[`Renderer`](../aspose.gis.rendering/renderer/) . |
| [MapLayer](./maplayer/) | A`Kaartlaag` is een basisklasse voor lagen binnen de[`Map`](../aspose.gis.rendering/map/). |
| [RasterMapLayer](./rastermaplayer/) | Een laag binnen[`Map`](../aspose.gis.rendering/map/) dat vertegenwoordigt een rasterlaag data. |
| [RasterMapResampling](./rastermapresampling/) | Deze klasse beschrijft hoe een rasterlaag opnieuw moet worden bemonsterd bij het renderen van een kaart. |
| [Renderer](./renderer/) | Een basisklasse voor renderers. |
| [Renderers](./renderers/) | Renderers voor alle ondersteunde formaten. |
| [VectorMapLayer](./vectormaplayer/) | Een laag binnen[`Map`](../aspose.gis.rendering/map/) dat vertegenwoordigt gegevens van een vectorlaag. |
