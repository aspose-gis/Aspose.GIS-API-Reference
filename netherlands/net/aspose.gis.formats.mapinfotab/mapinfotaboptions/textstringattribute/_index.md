## MapInfoTabOptions.TextStringAttribute property

Specificeert de naam van het attribuut dat de tekst van het grafische object 'Tekst' vertegenwoordigt.

```csharp
public string TextStringAttribute { get; set; }
```

### Opmerkingen

MapInfo Tab Formaat specificeert een grafisch object van het type 'Tekst'. 'Tekst' grafisch object vertegenwoordigt een label op een kaart. We exporteren 'Tekst' grafische objecten as a[`Feature`](../../../aspose.gis/feature/) met[`Polygon`](../../../aspose.gis.geometries/polygon/) geometrie die het label begrenst. Tekst van het label wordt geëxporteerd als[`FeatureAttribute`](../../../aspose.gis/featureattribute/) . Deze eigenschap specificeert de naam van het kenmerk dat wordt gebruikt om tekst van het label te exporteren. Standaardwaarde is`"tekst reeks"` .
