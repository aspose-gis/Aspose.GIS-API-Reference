## SpatialReferenceSystem.TryCreateFromEpsg method

Maak een ruimtelijk referentiesysteem op basis van de opgegeven EPSG-code.

```csharp
public static bool TryCreateFromEpsg(int epsg, out SpatialReferenceSystem value)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| epsg | Int32 | EPSG-code van het ruimtelijke referentiesysteem. |
| value | SpatialReferenceSystem& | Wanneer deze methode terugkeert`true` , bevat een SRS met de opgegeven EPSG-code; anders bevat `null` . |

### Winstwaarde

`true` als de opgegeven EPSG-code bekend is en SRS is gemaakt;`false` anders.
