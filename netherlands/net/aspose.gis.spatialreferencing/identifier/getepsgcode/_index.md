## Identifier.GetEpsgCode method

Als dit object een geldige EPSG-identificatie vertegenwoordigt (bijv. de naam van de autoriteit is "EPSG" en de unieke identificatie van de autoriteit is een geheel getal) - geef het terug. Anders - retourneer -1.

```csharp
public int GetEpsgCode()
```

### Winstwaarde

EPSG-ID vertegenwoordigd door dit object. Als dit object geen EPSG-identificatie vertegenwoordigt, retourneer dan -1.
