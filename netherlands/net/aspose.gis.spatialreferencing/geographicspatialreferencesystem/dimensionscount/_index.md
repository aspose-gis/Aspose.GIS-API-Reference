## GeographicSpatialReferenceSystem.DimensionsCount property

Retourneert het aantal dimensies in deze SRS. Voor geografische SRS kan dit zijn: twee - als dit een enkele geografische SRS is. drie - als dit een samengestelde SRS is, die bestaat uit een enkele, tweedimensionale, geografische SRS en verticale SRS, dat voegt een derde dimensie toe.

```csharp
public abstract int DimensionsCount { get; }
```
