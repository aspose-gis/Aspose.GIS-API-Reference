## CompoundSpatialReferenceSystem.AsProjected property

Geprojecteerde weergave van deze SRS retourneren. Als deze samengestelde SRS inderdaad een geprojecteerde SRS vertegenwoordigt, zal het resultaat een driedimensionale geprojecteerde SRS zijn (met X, Y, hoogte-afmetingen). Anders wordt er een uitzondering gegenereerd.

```csharp
public override ProjectedSpatialReferenceSystem AsProjected { get; }
```
