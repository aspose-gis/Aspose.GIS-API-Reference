## CompoundSpatialReferenceSystem.AsGeographic property

Geef de geografische representatie van deze SRS terug. Als deze samengestelde SRS inderdaad een geografische SRS vertegenwoordigt, zal het resultaat een driedimensionale geografische SRS zijn (met lengte-, breedte- en hoogtedimensies). Anders wordt er een uitzondering gegenereerd.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
