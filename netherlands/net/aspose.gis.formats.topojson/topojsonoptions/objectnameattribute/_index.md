## TopoJsonOptions.ObjectNameAttribute property

Naam van het attribuut, dat de naam weergeeft van het object dat een feature bevat.

```csharp
public string ObjectNameAttribute { get; set; }
```

### Opmerkingen

TopoJSON kan een onbeperkt aantal benoemde objecten bevatten. Elk dergelijk object kan meerdere functies bevatten. Raadpleeg de TopoJSON-specificatie voor meer details over benoemde objecten. Bij het lezen van TopoJSON specificeert deze eigenschap welk attribuut de naam van het object moet weerspiegelen dat een feature bevat. Bij het schrijven van TopoJSON specificeert deze eigenschap welk attribuut moet worden gebruikt om objecten te groeperen in objects. Standaard is "topojson_object_name".
