`Aspose.Gis.Rendering.Labelings` naamruimte biedt klassen voor het configureren van labeling van vectorobjecten.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [Labeling](./labeling/) | De abstracte hoofdklasse voor labelingen, klassen die beschrijven hoe labels op objecten moeten worden geplaatst. |
| [LabelingRule](./labelingrule/) | Een door de gebruiker gedefinieerde regel voor[`RuleBasedLabeling`](../aspose.gis.rendering.labelings/rulebasedlabeling/) . |
| [LabelPlacement](./labelplacement/) | Labelplaatsing geeft aan hoe labels worden geplaatst ten opzichte van de geometrie van het object. |
| [LineLabelPlacement](./linelabelplacement/) | Lijnlabelplaatsing plaatst labels langs lijn. |
| [NullLabeling](./nulllabeling/) | De[`NullLabeling`](../aspose.gis.rendering.labelings/nulllabeling/) slaat het labelen van een geometrie over waarop het is toegepast. |
| [PointLabelPlacement](./pointlabelplacement/) | Plaatsing van puntenlabels plaatst labels in de buurt van geometriecentra. |
| [RuleBasedLabeling](./rulebasedlabeling/) | Past een label toe op een object volgens door de gebruiker gedefinieerde regels. |
| [SimpleLabeling](./simplelabeling/) | Een eenvoudige labeling plaatst een label op elke functie. |
