Specificeert hoe labels worden gerenderd voor features die multipart-geometrieën bevatten.

**Module:** [aspose.gis.rendering.labelings](/psd/python-net/aspose.gis.rendering.labelings/)

**Full Name:** aspose.gis.rendering.labelings.MultipartMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Lidnaam** | **Beschrijving** |
| :- | :- |
| ALL | Plaatst een label dicht bij elk deel van de geometrie zolang er ruimte bij het deel is. |
| ANY | Plaatst één label voor de gehele geometrie. |
| LARGEST | Plaatst een label voor het grootste deel van de geometrie zolang er ruimte voor het label is. |
