## WarpOptions.CellHeight property

Tarama hücresinin yeni yüksekliğini belirtir (hedef coğrafi referanslı birimlerde). Değer 0 olarak ayarlanırsa,`CellHeight` otomatik olarak hesaplanır. Varsayılan değer "0"dır.

```csharp
public double CellHeight { get; set; }
```

### Notlar

Hücre yüksekliği 0 olarak ayarlanırsa, değer orijinal hücre yüksekliğinden alınır veya şu değerden hesaplanır:[`Height`](../height/) . Şuna dikkat edin`CellHeight` ile kullanılamaz[`Height`](../height/) .
