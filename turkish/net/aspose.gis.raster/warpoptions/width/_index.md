## WarpOptions.Width property

Çıktı raster genişliğini piksel ve sütun cinsinden belirtir. Değer 0 olarak ayarlanırsa, genişlik otomatik olarak hesaplanır. Varsayılan değer "0"dır.

```csharp
public int Width { get; set; }
```

### Notlar

Genişlik 0 olarak ayarlanırsa, değer orijinal genişlikten alınır veya şu değerden hesaplanır:[`CellWidth`](../cellwidth/) . Şuna dikkat edin`Width` ile kullanılamaz[`CellWidth`](../cellwidth/) .
