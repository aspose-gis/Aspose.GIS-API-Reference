## WarpOptions.CellWidth property

Tarama hücresinin yeni genişliğini belirtir (hedef coğrafi referanslı birimlerde). Değer 0 olarak ayarlanırsa,`CellWidth` otomatik olarak hesaplanır. Varsayılan değer "0"dır.

```csharp
public double CellWidth { get; set; }
```

### Notlar

Hücre genişliği 0 olarak ayarlanırsa, değer orijinal hücre genişliğinden alınır veya şu değerden hesaplanır:[`Width`](../width/) . Şuna dikkat edin`CellWidth` ile kullanılamaz[`Width`](../width/) .
