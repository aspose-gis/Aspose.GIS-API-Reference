## WarpOptions.Height property

Çıktı raster yüksekliğini piksel ve sütun cinsinden belirtir. Değer 0 olarak ayarlanırsa, yükseklik otomatik olarak hesaplanır. Varsayılan değer "0"dır.

```csharp
public int Height { get; set; }
```

### Notlar

Yükseklik 0 olarak ayarlanırsa, değer orijinal yükseklikten alınır veya şu değerden hesaplanır:[`CellHeight`](../cellheight/) . Şuna dikkat edin`Height` ile kullanılamaz[`CellHeight`](../cellheight/) .
