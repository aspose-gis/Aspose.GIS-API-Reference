## GpxOptions.ZAttribute property

Ara noktaların, rota noktalarının ve izleme noktalarının 'Z' koordinatı olarak hangi GPX özniteliğinin dışa aktarılacağını belirler. Eğer`null` hiçbir öznitelik 'Z' koordinatı olarak dışa aktarılmayacaktır. Varsayılanlar "ele"dir. Olası değerler, çift olarak temsil edilebilecek tüm GPX XML özniteliklerinin adlarıdır (örn. "hız", "magvar", "geoidheight" vb. )

```csharp
public string ZAttribute { get; set; }
```
