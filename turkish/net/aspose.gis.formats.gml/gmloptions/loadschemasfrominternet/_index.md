## GmlOptions.LoadSchemasFromInternet property

Aspose.GIS'in İnternet'ten XML şeması yüklemesine izin verilip verilmediğini belirler. olarak ayarlanırsa`false` , 'file://' ile başlamayan mutlak URI'lere sahip şemalar yüklenmez. Varsayılan`false` .

```csharp
public bool LoadSchemasFromInternet { get; set; }
```

### Notlar

Aspose.GIS, oluşturmak için GML dosyasının XML şemasını kullanır.[`FeatureAttributeCollection`](../../../aspose.gis/featureattributecollection) . XML şema dosyaları, GML dosyalarıyla birlikte dağıtılabilir veya İnternet'te bulunabilir. Önceki durumda, bu seçeneği şu şekilde ayarlamanız gerekir:`true` Aspose.GIS'in XML şemasını yüklemesine izin vermek için.
