## CompoundSpatialReferenceSystem.AsProjected property

Bu SRS'nin tahmini gösterimini döndürür. Bu bileşik SRS gerçekten de yansıtılmış bir SRS'yi temsil ediyorsa, sonuç üç boyutlu yansıtılmış SRS olacaktır (X, Y, yükseklik boyutları ile). Aksi takdirde bir istisna atılacaktır.

```csharp
public override ProjectedSpatialReferenceSystem AsProjected { get; }
```
