## CompoundSpatialReferenceSystem.AsGeographic property

Bu SRS'nin coğrafi temsilini döndürür. Bu bileşik SRS gerçekten de bir coğrafi SRS'yi temsil ediyorsa, sonuç üç boyutlu coğrafi SRS olacaktır (boylam, enlem, yükseklik boyutları ile). Aksi takdirde bir istisna atılacaktır.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
