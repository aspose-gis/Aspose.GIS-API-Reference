 kml:Feature とその内容がリストビューの項目としてどのように表示されるかを指定します。

**Module:** [aspose.gis.formats.kml.styles](/psd/python-net/aspose.gis.formats.kml.styles/)

**Full Name:** aspose.gis.formats.kml.styles.KmlItemTypes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **メンバー名** | **説明** |
| :- | :- |
| CHECK | 可視性はその項目のチェックボックスに紐付けられます |
| CHECK_HIDE_CHILDREN | 可視性には通常のチェックボックスを使用しますが、リストビューに項目の子要素は表示しません。 |
| CHECK_OFF_ONLY | すべての項目が一度に表示されるのを防ぎます—つまり、<br/>            ユーザーはコンテナ グループ内のすべてをオフにできますが、同時にすべてをオンにすることはできません。 |
| RADIO_FOLDER | 一度に表示される項目は1つだけです。 |
