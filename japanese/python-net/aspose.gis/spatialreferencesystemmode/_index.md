データベースにおける空間参照系 (SRS) の書き込みモードを指定します<br/>未知の SRS の場合です。

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.SpatialReferenceSystemMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **メンバー名** | **説明** |
| :- | :- |
| SETUP_TO_ZERO | ジオメトリの SRID をデータベースで未知の SRS の場合、'zero' に設定します。 |
| THROW_EXCEPTION | データベースで未知の SRS の場合、例外をスローします。 |
| WRITE_IN_SYSTEM_TABLE | データベースで未知の SRS の場合、システムテーブルに SRS 情報を書き込みます。 |
