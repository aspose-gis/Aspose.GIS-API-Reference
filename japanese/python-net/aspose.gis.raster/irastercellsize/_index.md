**Summary:** Describes the size, scale and rotation of a raster cell on a map.

**Module:** [aspose.gis.raster](/psd/python-net/aspose.gis.raster/)

**Full Name:** aspose.gis.raster.IRasterCellSize

**Aspose.PSD Version:** 25.9.0

## **Properties**
| **Name** | **Type** | **Access** | **説明** |
| :- | :- | :- | :- |
| 高さ | double | r | セルまたはピクセルの高さを取得します（常に正の値）。 |
| scale_x | double | r | セルまたはピクセルの幅の x 成分 (x-スケール) を取得します。 |
| scale_y | double | r | セルまたはピクセルの高さの y 成分 (y-スケール) を取得します。通常は負の値です。 |
| skew_x | double | r | セルまたはピクセルの高さの x 成分 (x-スキュー) を取得します。 |
| skew_y | double | r | セルまたはピクセルの幅の y 成分 (y-スキュー) を取得します。 |
| width | double | r | セルまたはピクセルの幅を取得します（常に正の値）。 |
