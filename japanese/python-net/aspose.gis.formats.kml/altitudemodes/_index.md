KML ジオメトリの高度モード。

**Module:** [aspose.gis.formats.kml](/psd/python-net/aspose.gis.formats.kml/)

**Full Name:** aspose.gis.formats.kml.AltitudeModes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **メンバー名** | **説明** |
| :- | :- |
| ABSOLUTE | 高度を海面上のメートル単位の値として解釈します。 |
| CLAMP_TO_GROUND | カメラの場合、この設定はカメラを relativeToGround に配置します。<br/>            カメラを地形の高さに正確に置くと、視点が地形と交差し（ビューが遮断されます）。 |
| なし | 高度モードなし。 |
| RELATIVE_TO_GROUND | 高度を地面上のメートル単位の値として解釈します。 |
