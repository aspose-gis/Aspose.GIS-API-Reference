## FileGdbCoordinatePrecisionGrid.XYScale property

X 座標と Y 座標のスケールを取得または設定します。に設定した場合`null`デフォルトが使用されます。

```csharp
public double? XYScale { get; set; }
```

### 例外

| 例外 | 調子 |
| --- | --- |
| ArgumentOutOfRangeException | 引数が正ではありません。 |

### 備考

デフォルト値は`1e9`ために[`VectorLayer`](../../../aspose.gis/vectorlayer/)地理的[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) と`XY スケール = 1 / XY 公差 * 10`ために[`VectorLayer`](../../../aspose.gis/vectorlayer/)with 投影[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) .
