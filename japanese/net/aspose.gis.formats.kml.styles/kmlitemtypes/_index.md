## KmlItemTypes enumeration

kml:Feature とそのコンテンツをアイテムとしてリスト ビューに表示する方法を指定します。

```csharp
public enum KmlItemTypes
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| RadioFolder | `0` | 一度に表示できるアイテムは 1 つだけです。 |
| Check | `1` | 可視性はアイテムのチェックボックスに関連付けられています |
| CheckHideChildren | `2` | 可視性のために通常のチェックボックスを使用しますが、リスト ビューに項目の子を表示しません。 |
| CheckOffOnly | `3` | すべてのアイテムが一度に表示されるのを防ぎます。つまり、 ユーザーはコンテナー グループ内のすべてをオフにすることはできますが、同時にすべてをオンにすることはできません. |
