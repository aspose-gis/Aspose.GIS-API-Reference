## KmlFeatureStyle class

スタイルはジオメトリの表示方法に影響します. これらのスタイルはドキュメント セクションでエンコードされ、各スタイルに一意の識別子があります. これらは kml 仕様で「共有スタイル」として知られています.

```csharp
public class KmlFeatureStyle : FeatureStyle
```
