`Aspose.Gis.Rendering.Symbolizers`名前空間は、レンダリング ベクター機能を構成するためのクラスを提供します。

## クラス

| クラス | 説明 |
| --- | --- |
| [GeometryGenerator](./geometrygenerator/) | シンボライザーを装飾して、レンダリング前にフィーチャのジオメトリを変更します。 |
| [LayeredSymbolizer](./layeredsymbolizer/) | 他のいくつかのシンボライザーをレンダリングするシンボライザー. |
| [MarkerCluster](./markercluster/) | マーカー クラスター シンボライザー. |
| [MarkerLine](./markerline/) | マーカー ライン シンボライザー. |
| [MarkerPatternFill](./markerpatternfill/) | マーカー パターン塗りつぶしシンボライザー. |
| [MixedGeometrySymbolizer](./mixedgeometrysymbolizer/) | ジオメトリ タイプに従ってフィーチャ ジオメトリに正しいシンボライザーを適用します。 |
| [NullVectorSymbolizer](./nullvectorsymbolizer/) | `NullSymbolizer`何も描画せず、適用されるジオメトリのレンダリングを効果的にスキップします. |
| [RasterImageMarker](./rasterimagemarker/) | このシンボライザーは、提供されたラスター イメージをレンダリングします。 |
| [Rule](./rule/) | のユーザー定義ルール[`RuleBasedSymbolizer`](../aspose.gis.rendering.symbolizers/rulebasedsymbolizer/) . |
| [RuleBasedSymbolizer](./rulebasedsymbolizer/) | ユーザー定義のルールに従って、フィーチャ ジオメトリにシンボライザーを適用します。 |
| [SimpleFill](./simplefill/) | シンプルなポリゴン シンボライザー. |
| [SimpleLine](./simpleline/) | シンプルなライン シンボライザー. |
| [SimpleMarker](./simplemarker/) | シンプルなポイント シンボライザー. |
| [VectorSymbolizer](./vectorsymbolizer/) | ベクター フィーチャをレンダリングするシンボライザーの抽象ルート クラス。 |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [HorizontalAnchor](./horizontalanchor/) | 水平に揃える辺を指定します。 |
| [MarkerShapeType](./markershapetype/) | マーカーの形状タイプ. |
| [RenderingOrder](./renderingorder/) | レンダリング順序を決定します。 |
| [VerticalAnchor](./verticalanchor/) | 垂直に揃える辺を指定します。 |
