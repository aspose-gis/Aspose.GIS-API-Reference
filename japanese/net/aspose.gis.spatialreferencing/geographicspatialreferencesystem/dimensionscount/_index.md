## GeographicSpatialReferenceSystem.DimensionsCount property

この SRS の次元数を返します。地理的 SRS の場合、これは次のようになります: 2 - これが単一の地理的 SRS の場合。 3 - 単一の 2 次元の地理的 SRS と垂直 SRS で構成され、3 番目の次元を追加する複合 SRS の場合。

public abstract int DimensionsCount { get; }
