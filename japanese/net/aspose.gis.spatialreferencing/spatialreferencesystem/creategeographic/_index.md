## SpatialReferenceSystem.CreateGeographic method

カスタム パラメータから地理的 SRS を作成します。

```csharp
public static GeographicSpatialReferenceSystem CreateGeographic(
    GeographicSpatialReferenceSystemParameters parameters, Identifier identifier = null)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| parameters | GeographicSpatialReferenceSystemParameters | 作成元のパラメーター。 |
| identifier | Identifier | SRS に添付される識別子。識別子をアタッチしても、他の SRS パラメータは変更されません. 識別子と SRS パラメータの一貫性を確保するのはあなた次第です. |

### 戻り値

新しい地理的 SRS。
