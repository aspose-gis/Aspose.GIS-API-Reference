`Aspose.Gis`名前空間は、GIS データを生成、変換、および変更するためのクラスを提供します.

## クラス

| クラス | 説明 |
| --- | --- |
| [AbstractPath](./abstractpath/) | アン`抽象パス`ローカルファイルシステム、リモートファイルストレージ、ZIPアーカイブなどのファイルシステム に似た環境で一意の場所を指定するクラスの基本クラスです. |
| [AttributesConverterActions](./attributesconverteractions/) | 宛先レイヤーの属性を持つオプションのアクション. |
| [ConversionOptions](./conversionoptions/) | フォーマット間でデータを変換するためのオプション. |
| [DatabaseDriver](./databasedriver/) | 特定のデータベース ベースのフォーマット用のドライバー。 |
| [DatabaseDriverOptions](./databasedriveroptions/) | のオプション[`DatabaseDriver`](../aspose.gis/databasedriver/) . |
| [Dataset](./dataset/) | データセットは、[`VectorLayer`](../aspose.gis/vectorlayer/)インスタンス. |
| [Driver](./driver/) | GIS データへのドライバーの基本クラス。 |
| [DriverOptions](./driveroptions/) | のオプション[`Driver`](../aspose.gis/driver/) . |
| [Drivers](./drivers/) | サポートされているすべてのフォーマットのドライバー. |
| [Extent](./extent/) | 2 次元の空間バウンディング ボックス。 |
| [Feature](./feature/) | ジオメトリとユーザー定義の属性で構成される地理的特徴. |
| [FeatureAttribute](./featureattribute/) | の属性[`Feature`](../aspose.gis/feature/) . |
| [FeatureAttributeCollection](./featureattributecollection/) | A[`FeatureAttributeCollection`](../aspose.gis/featureattributecollection/)で使用できる属性を定義します。[`Feature`](../aspose.gis/feature/) . |
| [FeaturesSequence](./featuressequence/) | [`FeaturesSequence`](../aspose.gis/featuressequence/)ベクトル機能のセットを表します. |
| [FeatureStyle](./featurestyle/) | フィーチャ スタイル階層の抽象ルート クラス。 |
| [FileDriver](./filedriver/) | 特定のファイル ベースのフォーマット用のドライバー。 |
| [GeoConvert](./geoconvert/) | 座標をさまざまな形式に変換します。 |
| [GisException](./gisexception/) | GIS データ処理中にエラーが発生した場合にスローされる例外。 |
| [License](./license/) | コンポーネントのライセンスを取得する方法を提供します。 |
| [Metered](./metered/) | メータリング キーを設定するメソッドを提供します。 |
| [NumericFormat](./numericformat/) | [`NumericFormat`](../aspose.gis/numericformat/) text. の一般的な数値型をフォーマットするために使用されます |
| [PrecisionModel](./precisionmodel/) | [`PrecisionModel`](../aspose.gis/precisionmodel/)座標の有効桁数を指定します. |
| [RasterDriver](./rasterdriver/) | 特定のラスタ ベース フォーマット用のドライバ。 |
| [RasterDriverOptions](./rasterdriveroptions/) | のオプション[`RasterDriver`](../aspose.gis/rasterdriver/) . |
| [SavingOptions](./savingoptions/) | 保存のオプション[`FeaturesSequence`](../aspose.gis/featuressequence/)ファイルへ. |
| [VectorLayer](./vectorlayer/) | ベクター レイヤーを表します。 ベクター レイヤーは、ファイルに保存された地理的特徴のコレクションです。 |
