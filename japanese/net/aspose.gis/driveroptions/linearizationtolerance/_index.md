## DriverOptions.LinearizationTolerance property

曲線ジオメトリを線形化するために使用する許容差.

```csharp
public double LinearizationTolerance { get; set; }
```

### プロパティ値

渡す許容値[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/)ジオメトリを追加する前.

### 備考

これは作成オプションです - 開くことには影響しません. ドライバーが非線形ジオメトリをサポートしていない場合、それらは自動的に線形化されます.[`ToLinearGeometry`](../../../aspose.gis.geometries/geometry/tolineargeometry/)線形化のために が使用されるメソッド.
