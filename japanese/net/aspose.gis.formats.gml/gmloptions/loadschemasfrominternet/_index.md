## GmlOptions.LoadSchemasFromInternet property

Aspose.GIS がインターネットから XML スキーマをロードできるかどうかを決定します。 に設定されている場合`false`、'file://' で始まらない絶対 URI を持つスキーマはロードされません。 デフォルトは`false` .

```csharp
public bool LoadSchemasFromInternet { get; set; }
```

### 備考

Aspose.GIS は、GML ファイルの XML スキーマを使用して、[`FeatureAttributeCollection`](../../../aspose.gis/featureattributecollection/) . XML スキーマ ファイルは、GML ファイルと一緒に配布したり、インターネット上に配置したりできます。 前者の場合、このオプションを次のように設定する必要があります。`true` Aspose.GIS が XML スキーマをロードできるようにします。
