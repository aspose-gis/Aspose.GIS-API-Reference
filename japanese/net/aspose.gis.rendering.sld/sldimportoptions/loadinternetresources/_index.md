## SldImportOptions.LoadInternetResources property

Aspose.GIS がインターネットからソースをロードできるかどうかを決定します。 に設定されている場合`false`、'file://' で始まらない絶対 URI を持つスキーマはロードされません。 デフォルトは`false` .

public bool LoadInternetResources { get; set; }
