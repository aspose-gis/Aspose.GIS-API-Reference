## RasterLayer.GetBand method

指定されたインデックスでバンドを取得します。

```csharp
public abstract IRasterBand GetBand(int index = 0)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| index | Int32 | バンド番号は 0 から始まり、指定されていない場合、バンドは 0 と見なされます。 |

### 戻り値

ラスター バンドに関する基本的なメタ データを返します。

### 備考

ラスターにバンドが含まれていない場合、エラーがスローされます。
