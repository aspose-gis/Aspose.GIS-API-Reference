## WarpOptions.Height property

出力ラスターの高さをピクセルと列で指定します。 値が 0 に設定されている場合、高さは自動的に計算されます。デフォルト値は「0」です。

```csharp
public int Height { get; set; }
```

### 備考

高さが 0 に設定されている場合、値は元の高さから取得されるか、またはから計算されます。[`CellHeight`](../cellheight/) . 注意してください`Height`と一緒に使用することはできません[`CellHeight`](../cellheight/) .
