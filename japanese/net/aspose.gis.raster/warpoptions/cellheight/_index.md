## WarpOptions.CellHeight property

ラスター セルの新しい高さを (ターゲットの地理参照単位で) 指定します。 値が 0 に設定されている場合、`CellHeight`自動的に計算されます。デフォルト値は「0」です。

```csharp
public double CellHeight { get; set; }
```

### 備考

セルの高さが 0 に設定されている場合、値は元のセルの高さから取得されるか、またはセルの高さから計算されます。[`Height`](../height/) . 注意してください`CellHeight`と一緒に使用することはできません[`Height`](../height/) .
