## IGeometry.HasCurveGeometry property

このジオメトリが曲線 (線形ではない) ジオメトリであるか、曲線を含むかどうかを示す値を取得します。

```csharp
public bool HasCurveGeometry { get; }
```

### プロパティ値

`false`このジオメトリとそれに含まれるすべてのジオメトリが線形の場合。さもないと、`true` .
