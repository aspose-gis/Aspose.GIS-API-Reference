## CompoundSpatialReferenceSystem.AsGeographic property

Mengembalikan representasi geografis dari SRS ini. Jika SRS majemuk ini benar-benar mewakili SRS geografis, hasilnya akan menjadi SRS geografis tiga dimensi (dengan dimensi bujur, lintang, tinggi). Kalau tidak, pengecualian akan dilemparkan.

```csharp
public override GeographicSpatialReferenceSystem AsGeographic { get; }
```
