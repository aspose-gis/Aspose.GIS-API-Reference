## SimpleLabeling.FeatureBasedConfiguration property

Callback yang digunakan untuk mengonfigurasi pelabelan ini sebelum menangani fitur.

```csharp
public Action<Feature, SimpleLabeling> FeatureBasedConfiguration { get; set; }
```

### Perkataan

Callback ini dipanggil sebelum melabeli setiap fitur. Itu menerima fitur yang akan diberi label dan tiruan dari pelabelan ini. Dengan mengubah properti klon, dimungkinkan untuk memperbarui perilaku pelabelan berdasarkan atribut fitur.
