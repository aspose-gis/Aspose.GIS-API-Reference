## FileGdbCoordinatePrecisionGrid.XYScale property

Mendapat atau mengatur skala koordinat X dan Y. Jika disetel ke`null` defaultnya adalah digunakan.

```csharp
public double? XYScale { get; set; }
```

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ArgumentOutOfRangeException | Argumen tidak positif. |

### Perkataan

Nilai defaultnya adalah`1e9` untuk[`VectorLayer`](../../../aspose.gis/vectorlayer/)dengan geografis[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) dan`XYScale = 1 / XYToleransi * 10` untuk[`VectorLayer`](../../../aspose.gis/vectorlayer/) with diproyeksikan[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) .
