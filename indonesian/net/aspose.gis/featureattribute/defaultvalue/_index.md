## FeatureAttribute.DefaultValue property

Mendapat atau menetapkan nilai untuk atribut, yang menunjukkan data yang hilang.

```csharp
public object DefaultValue { get; set; }
```

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| InvalidOperationException | Atribut terkunci. |
| InvalidOperationException | Atribut tidak mengizinkan`batal` nilai-nilai. |

### Perkataan

Ini adalah nilai yang mewakili informasi yang hilang, ketika atribut tidak mengizinkan`batal`value. Untuk atribut yang memungkinkan`batal` nilai ([`CanBeNull`](../canbenull/) ==`BENAR` ),`DefaultValue` adalah`batal` kecuali diubah secara eksplisit.
