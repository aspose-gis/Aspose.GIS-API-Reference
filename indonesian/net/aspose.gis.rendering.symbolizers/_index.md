`Aspose.Gis.Rendering.Symbolizers` namespace menyediakan kelas untuk mengonfigurasi fitur vektor rendering.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [GeometryGenerator](./geometrygenerator/) | Hiasi simbol untuk mengubah geometri fitur sebelum merender. |
| [LayeredSymbolizer](./layeredsymbolizer/) | Simbol yang merender beberapa simbol lainnya. |
| [MarkerCluster](./markercluster/) | Simbol cluster penanda. |
| [MarkerLine](./markerline/) | Simbol garis penanda. |
| [MarkerPatternFill](./markerpatternfill/) | Simbol isian pola penanda. |
| [MixedGeometrySymbolizer](./mixedgeometrysymbolizer/) | Menerapkan simbol yang benar ke geometri fitur menurut jenis geometrinya. |
| [NullVectorSymbolizer](./nullvectorsymbolizer/) | Itu`NullSymbolizer` tidak menggambar apa pun dan secara efektif melewatkan rendering geometri yang diterapkan. |
| [RasterImageMarker](./rasterimagemarker/) | Simbol ini merender gambar raster yang disediakan. |
| [Rule](./rule/) | Aturan yang ditentukan pengguna untuk[`RuleBasedSymbolizer`](../aspose.gis.rendering.symbolizers/rulebasedsymbolizer/) . |
| [RuleBasedSymbolizer](./rulebasedsymbolizer/) | Menerapkan simbol untuk menampilkan geometri menurut aturan yang ditentukan pengguna. |
| [SimpleFill](./simplefill/) | Simbol poligon sederhana. |
| [SimpleLine](./simpleline/) | Simbol garis sederhana. |
| [SimpleMarker](./simplemarker/) | Simbol titik sederhana. |
| [VectorSymbolizer](./vectorsymbolizer/) | Kelas root abstrak untuk simbol yang merender fitur vektor. |
