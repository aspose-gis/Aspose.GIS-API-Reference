## CsvOptions.HasAttributeNames property

Menentukan apakah ada baris header dengan nama atribut. Defaultnya adalah`true` .

```csharp
public bool HasAttributeNames { get; set; }
```

### Perkataan

Dalam kasus`true` A[`Attributes`](../../../aspose.gis/vectorlayer/attributes/) akan diberi nama sesuai dengan baris tajuk. Jika tidak maka akan diberi nama sebagai:`bidang_1, bidang_2, dll.`
