## GeoJsonOptions.WriteBoundingBoxes property

Menentukan apakah objek GeoJSON harus menyertakan informasi rentang koordinat untuk Geometrinya. Jika disetel ke`true` , anggota "bbox" dihasilkan untuk setiap geometri (bukan nol) saat ditambahkan ke lapisan. Nilai defaultnya adalah`false` .

public bool WriteBoundingBoxes { get; set; }
