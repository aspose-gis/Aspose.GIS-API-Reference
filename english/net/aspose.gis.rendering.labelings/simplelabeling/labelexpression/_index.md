## SimpleLabeling.LabelExpression property

Provides a way to customize and format label text. If set, overrides [`LabelAttribute`](../labelattribute/). Either [`LabelAttribute`](../labelattribute/) or `LabelExpression` must be set before rendering; InvalidOperationException is thrown otherwise.
