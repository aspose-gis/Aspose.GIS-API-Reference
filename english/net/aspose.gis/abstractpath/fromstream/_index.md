## AbstractPath.FromStream method

Creates an [`AbstractPath`](../../abstractpath) from a Stream.

```csharp
public static AbstractPath FromStream(Stream stream)
```

| Parameter | Type | Description |
| --- | --- | --- |
| stream | Stream | A stream to create an [`AbstractPath`](../../abstractpath) from. `Aspose.GIS` does not dispose the stream. |

### Return Value

An instance of [`AbstractPath`](../../abstractpath) with the specified Stream as its content.

### Remarks

This method is useful for passing a stream of data in a single-file format (such as GeoJSON or KML) to methods that expect an [`AbstractPath`](../../abstractpath) .
