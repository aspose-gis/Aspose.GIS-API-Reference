## VectorLayer.ConstructFeature method

Creates (but does not add to the layer) a new feature with attributes matching the collection of attributes of this layer. When done with setting data for the feature, use [`Add`](../add) to add the feature to the layer.

```csharp
public Feature ConstructFeature()
```

## Return Value

A new feature.

### Remarks

Changes to [`Attributes`](../attributes) are not allowed after calling this method.
