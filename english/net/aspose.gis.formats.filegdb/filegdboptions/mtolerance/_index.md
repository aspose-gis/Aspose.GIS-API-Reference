## FileGdbOptions.MTolerance property

M snapping tolerance.

```csharp
public double? MTolerance { get; set; }
```

## Remarks

This is a creation option and it does not affect reading and editing. This parameter controls a snapping tolerance used for advanced ArcGIS features. It does not affect Aspose.GIS behavior, but it can be used by ArcGIS. The unit of the parameter is the unit of spatial reference system. If set to `null`, the default is used. The default is 0.001.
