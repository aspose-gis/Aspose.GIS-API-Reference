## IRasterValues indexer

के रूप में बैंड मान प्राप्त करता है`double` टाइप.

```csharp
public double this[int bandIndex] { get; }
```

| पैरामीटर | विवरण |
| --- | --- |
| bandIndex | बैंड का सूचकांक। नंबरिंग 0 से शुरू होती है। |

### प्रतिलाभ की मात्रा

परिवर्तित मान।

### टिप्पणियों

यदि बैंड प्रकार हैRawBits उपयोग करने पर विचार करें[`AsRawBits`](../asrawbits/) विधि अपवादों से बचने के लिए.
