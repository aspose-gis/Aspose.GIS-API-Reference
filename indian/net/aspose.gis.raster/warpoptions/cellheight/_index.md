## WarpOptions.CellHeight property

रेखापुंज सेल की एक नई ऊंचाई निर्दिष्ट करता है (लक्ष्य भू-संदर्भित इकाइयों में)। यदि मान 0 पर सेट है, तो`CellHeight` स्वचालित रूप से गणना की जाती है। डिफ़ॉल्ट मान "0" है.

```csharp
public double CellHeight { get; set; }
```

### टिप्पणियों

यदि सेल की ऊंचाई 0 पर सेट है, तो मान मूल सेल की ऊंचाई से लिया जाएगा या इसकी गणना की जाएगी[`Height`](../height/) . ध्यान दें`CellHeight` के साथ प्रयोग नहीं किया जा सकता[`Height`](../height/) .
