## GmlOptions.RestoreSchema property

निर्धारित करता है कि क्या Aspose.GIS को Gml फ़ाइल में विशेषताओं को पार्स करने की अनुमति है जिसमें एक XML स्कीमा गायब है या लोड नहीं किया जा सकता है। यदि पर सेट है`true` , Aspose.GIS रीडर को XML स्कीमा की उपस्थिति की आवश्यकता नहीं है। डिफ़ॉल्ट है`false` .

```csharp
public bool RestoreSchema { get; set; }
```

### टिप्पणियों

ड्राइवर फ़ाइल को स्कैन करके और संगठन को निर्धारित करने के लिए "ज्ञात" वस्तुओं की तलाश करके सुविधा वर्गों और उनके संबद्ध गुणों को स्वचालित रूप से पार्स करने का प्रयास करता है। XML स्कीमा गुम हो गया है या इंटरनेट से लोड नहीं हो सकता.
