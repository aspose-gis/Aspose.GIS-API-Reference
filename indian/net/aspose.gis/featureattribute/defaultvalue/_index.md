## FeatureAttribute.DefaultValue property

विशेषता के लिए एक मान प्राप्त या सेट करता है, जो लापता डेटा को इंगित करता है।

```csharp
public object DefaultValue { get; set; }
```

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| InvalidOperationException | विशेषता लॉक है। |
| InvalidOperationException | विशेषता अनुमति नहीं देती है`व्यर्थ` मान। |

### टिप्पणियों

यह वह मान है जो जानकारी के लापता हिस्से को दर्शाता है, जब कोई विशेषता इसकी अनुमति नहीं देती है`व्यर्थ`value. उन विशेषताओं के लिए जो अनुमति देते हैं`व्यर्थ` मान ([`CanBeNull`](../canbenull/) ==`सत्य` ),`DefaultValue` है`व्यर्थ` जब तक कि स्पष्ट रूप से परिवर्तित न किया गया हो.
