`Aspose.Gis.GeoTools` 命名空间包含表示地理工具的类。

## 类

| 类 | 描述 |
| --- | --- |
| [GeneratorTiles](./generatortiles/) | 瓦片生成器。 |
| [GeneratorTilesRenderOptions](./generatortilesrenderoptions/) | 渲染瓦片的选项。 |
| [GeoGenerator](./geogenerator/) | 在给定平面上生成随机点、线和多边形的生成器。 |
| [GeometryOperations](./geometryoperations/) | 几何操作类为几何提供了额外的地理处理算法。 |
| [LineGeneratorOptions](./linegeneratoroptions/) | 在表面或区域上生成线的选项。 |
| [NearPointsCleanerOptions](./nearpointscleaneroptions/) | 删除彼此过于接近的点的选项。 |
| [PointGeneratorOptions](./pointgeneratoroptions/) | 在表面或区域上生成点的选项。 |
| [PolygonGeneratorOptions](./polygongeneratoroptions/) | 在表面或区域上生成线的选项。 |
| [SimplifySegmentsOptions](./simplifysegmentsoptions/) | 用于 [`SimplifySegments`](../aspose.gis.geotools/geometryoperations/simplifysegments/) 的选项。 |
| [StarGeneratorOptions](./stargeneratoroptions/) | 针对 [`ProducePolygons`](../aspose.gis.geotools/geogenerator/producepolygons/) 的生成器特定选项。 |
