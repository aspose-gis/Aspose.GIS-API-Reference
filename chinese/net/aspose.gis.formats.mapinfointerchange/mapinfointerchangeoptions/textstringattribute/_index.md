## MapInfoInterchangeOptions.TextStringAttribute property

指定表示“文本”图形对象的文本的属性名称。

```csharp
public string TextStringAttribute { get; set; }
```

### 评论

MapInfo 交换格式指定“文本”类型的图形对象。 'Text' 图形对象表示地图上的标签。我们将“文本”图形对象导出为 a[`Feature`](../../../aspose.gis/feature)和[`Polygon`](../../../aspose.gis.geometries/polygon)geometry这限制了标签。 标签文本导出为[`FeatureAttribute`](../../../aspose.gis/featureattribute)。 此属性指定用于导出标签文本的属性名称。 默认值为` "textstring"` 。
