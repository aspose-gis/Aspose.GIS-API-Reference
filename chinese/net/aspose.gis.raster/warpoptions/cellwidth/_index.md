## WarpOptions.CellWidth property

指定光栅单元的新宽度（以目标地理参考单位）。 如果该值设置为 0，则自动计算`CellWidth`。默认值为“0”。

```csharp
public double CellWidth { get; set; }
```

### 评论

如果单元格宽度设置为 0，则该值将取自原始单元格宽度或计算自[`Width`](../width)。 注意`CellWidth`不能与[`Width`](../width)一起使用.
