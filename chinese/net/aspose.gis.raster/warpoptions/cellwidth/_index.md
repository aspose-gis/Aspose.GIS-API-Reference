## WarpOptions.CellWidth property

指定栅格像元的新宽度（以目标地理参考单位为单位）。 如果该值设置为 0，则`CellWidth`是自动计算的。默认值为“0”.

```csharp
public double CellWidth { get; set; }
```

### 评论

如果单元格宽度设置为 0，则该值将从原始单元格宽度中获取或从计算[`Width`](../width/). 注意`CellWidth`不能与[`Width`](../width/).
