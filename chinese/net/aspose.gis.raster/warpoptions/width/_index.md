## WarpOptions.Width property

以像素和列为单位指定输出栅格宽度。 如果该值设置为 0，则会自动计算宽度。默认值为“0”.

```csharp
public int Width { get; set; }
```

### 评论

如果宽度设置为 0，则该值将从原始宽度中获取或从[`CellWidth`](../cellwidth/). 注意`Width`不能与[`CellWidth`](../cellwidth/).
