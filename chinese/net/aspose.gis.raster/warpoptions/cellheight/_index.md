## WarpOptions.CellHeight property

指定栅格像元的新高度（以目标地理参考单位）。 如果该值设置为 0，则`CellHeight`是自动计算的。默认值为“0”。

```csharp
public double CellHeight { get; set; }
```

### 评论

如果单元格高度设置为 0，则该值将取自原始单元格高度或计算[`Height`](../height). 请注意`CellHeight`不能与[`Height`](../height).
