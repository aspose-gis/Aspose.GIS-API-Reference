## GeoJsonOptions.WriteBoundingBoxes property

确定是否应包含 GeoJSON 对象的几何坐标范围信息。 如果设置为`true`，则在将每个几何图形（非空）添加到图层时会为其生成一个成员“bbox”。 默认值为`false`。

```csharp
public bool WriteBoundingBoxes { get; set; }
```

### 评论

这是一个创建选项 - 它不影响打开。
