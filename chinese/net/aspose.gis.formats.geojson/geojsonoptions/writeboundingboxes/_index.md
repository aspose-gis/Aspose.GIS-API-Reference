## GeoJsonOptions.WriteBoundingBoxes property

确定是否应在 GeoJSON 对象中包含其几何体的坐标范围信息。如果设置为 `true`，则在几何体添加到图层时为每个几何体（非 null）生成一个成员 "bbox"。默认值为 `false`。

```csharp
public bool WriteBoundingBoxes { get; set; }
```

## 备注

这是一个创建选项 - 它不影响打开。
