## GeoJsonOptions.WriteBoundingBoxes property

确定 GeoJSON 对象是否应包含有关其几何坐标范围的信息。 如果设置为`true`，当它被添加到图层时，为每个几何体（不为空）生成一个成员“bbox”。 默认值为`false`.

```csharp
public bool WriteBoundingBoxes { get; set; }
```

### 评论

这是一个创建选项 - 它不影响打开。
