## GeoJsonOptions.ReadBoundingBoxes property

确定是否将边界框（'bbox'）读取为属性，属性名为 'bbox_0'、'bbox_1' 等。默认值为 `false`。[`NestedPropertiesSeparator`](../nestedpropertiesseparator/) 字符串用于 bbox_0、bbox_1 等名称中。

```csharp
public bool ReadBoundingBoxes { get; set; }
```

## 备注

这是读取选项 - 不影响写入。
