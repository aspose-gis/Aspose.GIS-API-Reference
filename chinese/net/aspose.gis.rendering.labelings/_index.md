`Aspose.Gis.Rendering.Labelings` 命名空间提供用于配置矢量要素标注的类。

## 类

| 类 | 描述 |
| --- | --- |
| [Labeling](./labeling/) | 标注的抽象根类，描述如何在要素上放置标签的类。 |
| [LabelingRule](./labelingrule/) | 用于 [`RuleBasedLabeling`](../aspose.gis.rendering.labelings/rulebasedlabeling/) 的用户定义规则。 |
| [LabelPlacement](./labelplacement/) | 标签放置指定标签相对于要素几何的放置方式。 |
| [LineLabelPlacement](./linelabelplacement/) | 线标签放置沿线放置标签。 |
| [NullLabeling](./nulllabeling/) | [`NullLabeling`](../aspose.gis.rendering.labelings/nulllabeling/) 跳过对其所应用的几何体的标注。 |
| [PointLabelPlacement](./pointlabelplacement/) | 点标签放置在几何体中心附近。 |
| [RuleBasedLabeling](./rulebasedlabeling/) | 根据用户定义的规则对要素应用标注。 |
| [SimpleLabeling](./simplelabeling/) | 简单标注在每个要素上放置标签。 |
## 枚举

| 枚举 | 描述 |
| --- | --- |
| [FontStyle](./fontstyle/) | 指定要应用于文本的样式。 |
| [LineLabelAlignment](./linelabelalignment/) | 指定标签相对于线的对齐方式。 |
| [MultipartMode](./multipartmode/) | 指定如何为包含多部件几何的要素渲染标签。 |
