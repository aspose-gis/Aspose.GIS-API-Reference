## MultipartMode enumeration

指定如何为包含多部分几何的要素呈现标签。

```csharp
public enum MultipartMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| All | `0` | 在几何体的每个部分附近放置一个标签，只要该部分附近有空间即可。 |
| Any | `1` | 为整个几何放置一个标签。 |
| Largest | `2` | 为几何体的最大部分放置标签，只要标签有空间即可。 |
