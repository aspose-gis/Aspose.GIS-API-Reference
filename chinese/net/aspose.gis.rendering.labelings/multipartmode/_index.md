## MultipartMode enumeration

指定如何为包含多段几何的要素渲染标签。

```csharp
public enum MultipartMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| All | `0` | 在几何体的每个部件附近放置标签，只要该部件附近有空间。 |
| Any | `1` | 为整个几何体放置一个标签。 |
| Largest | `2` | 在几何体最大的部件上放置标签，只要有足够的空间放置标签。 |
