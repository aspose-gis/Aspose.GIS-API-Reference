`Aspose.Gis.Rendering` 命名空间提供用于渲染地图的类。

## 类

| 类 | 描述 |
| --- | --- |
| [Map](./map/) | Map 是一个图层集合，可以通过 [`Renderer`](../aspose.gis.rendering/renderer/) 叠加渲染。 |
| [MapLayer](./maplayer/) | `MapLayer` 是 [`Map`](../aspose.gis.rendering/map/) 内图层的基类。 |
| [RasterMapLayer](./rastermaplayer/) | [`Map`](../aspose.gis.rendering/map/) 中的一个图层，表示栅格图层数据。 |
| [RasterMapResampling](./rastermapresampling/) | 此类描述了在渲染地图时如何对栅格图层进行重采样。 |
| [Renderer](./renderer/) | 渲染器的基类。 |
| [Renderers](./renderers/) | 所有受支持格式的渲染器。 |
| [VectorMapLayer](./vectormaplayer/) | [`Map`](../aspose.gis.rendering/map/) 中的一个图层，表示矢量图层数据。 |
## Structures

| 结构 | 描述 |
| --- | --- |
| [Measurement](./measurement/) | 指示渲染度量的数字。 |
