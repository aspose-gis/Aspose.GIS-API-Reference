## Map.Extent property

指定要渲染的地图边界。 如果设置为`null`，则在渲染期间计算范围以包括所有图层中的所有几何图形。

```csharp
public Extent Extent { get; set; }
```

### 评论

如果范围的空间参考系统不是等于地图的空间参考系统，范围在渲染过程中被转换为 到目标空间参考系统。
