## GpxOptions.ZAttribute property

确定哪个 GPX 属性将导出为航点、路线点和轨迹点的 'Z' 坐标。如果为 `null`，则不会有属性导出为 'Z' 坐标。默认值为 "ele"。可能的取值是所有可以表示为 double 的 GPX XML 属性名称（例如 "speed"、"magvar"、"geoidheight" 等）。

```csharp
public string ZAttribute { get; set; }
```
