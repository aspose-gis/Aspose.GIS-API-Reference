## GpxOptions.ZAttribute property

确定哪个 GPX 属性将导出为航路点、路线点和跟踪点的“Z”坐标。 如果`null` 没有属性将导出为“Z”坐标。 默认为“ele”。 可能的值是可以表示为双精度的所有 GPX XML 属性的名称（例如“speed”、“magvar”、“geoidheight”等。 )

```csharp
public string ZAttribute { get; set; }
```
