## FromDefinitionDataSourceBuilder.IdentityAttribute method

必需的设置，可用于跟踪更改。

```csharp
public FromDefinitionDataSourceBuilder IdentityAttribute(string name, bool overwriteSameKey = true)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| 名称 | String | 特征的属性，将被视为唯一标识该特征。 |
| overwriteSameKey | Boolean | 如果将此标志设置为 true，则新添加的具有与图层中已存在的唯一标识符相同的特征将覆盖当前特征，并且如果发现差异，数据将被读取为已更新。否则，将抛出异常。 |
