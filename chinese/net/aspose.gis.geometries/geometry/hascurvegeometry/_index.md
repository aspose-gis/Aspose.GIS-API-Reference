## Geometry.HasCurveGeometry property

获取一个值，该值指示此几何图形是否为或包含曲线（非线性）几何图形。

```csharp
public virtual bool HasCurveGeometry { get; }
```

### 适当的价值

`false`如果这个几何图形和它包含的所有几何图形都是线性的；否则，`true`.
