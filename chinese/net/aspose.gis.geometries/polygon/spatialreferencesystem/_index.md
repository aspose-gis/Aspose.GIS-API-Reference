## Polygon.SpatialReferenceSystem property

获取此实例的 SpatialReferenceSystem。此属性可以为 `null`，表示 SpatialReferenceSystem 未知。分配新的 SpatialReferenceSystem 不会执行任何坐标转换，只会更改引用。

```csharp
public override SpatialReferenceSystem SpatialReferenceSystem { get; set; }
```
