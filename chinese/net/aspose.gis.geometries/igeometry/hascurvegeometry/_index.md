## IGeometry.HasCurveGeometry property

获取一个值，指示此几何体是否为或包含曲线（非线性）几何体。

```csharp
public bool HasCurveGeometry { get; }
```

### Property Value

`false` 如果此几何体及其包含的所有几何体都是线性的；否则，`true`。
