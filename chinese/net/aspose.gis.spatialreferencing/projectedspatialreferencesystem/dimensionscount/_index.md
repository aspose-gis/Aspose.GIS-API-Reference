## ProjectedSpatialReferenceSystem.DimensionsCount property

返回此 SRS 的维度计数。对于投影 SRS，这可以是：two - 如果这是单一投影 SRS；three - 如果这是复合 SRS，由单一二维投影 SRS 和垂直 SRS 组成，后者增加第三维度。

```csharp
public abstract int DimensionsCount { get; }
```
