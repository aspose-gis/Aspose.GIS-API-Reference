## CompoundSpatialReferenceSystem.Type property

此复合 SRS 的类型。如果该复合 SRS 由地理和垂直 SRS 组合而成，则为 Geographic；如果由投影和垂直 SRS 组合而成，则为 Projected。

```csharp
public override SpatialReferenceSystemType Type { get; }
```
