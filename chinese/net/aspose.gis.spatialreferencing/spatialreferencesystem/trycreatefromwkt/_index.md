## SpatialReferenceSystem.TryCreateFromWkt method

基于 WKT（Well-Known Text）字符串创建新的 `SpatialReferenceSystem`。

```csharp
public static bool TryCreateFromWkt(string wkt, out SpatialReferenceSystem value)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| wkt | 字符串 | WKT 字符串。 |
| 值 | SpatialReferenceSystem& | 当此方法返回 `true` 时，包含一个由 WKT 创建的 SRS；否则，包含 `null`。 |

### 返回值

`true` 表示 SRS 已成功创建；`false` 表示未成功。
