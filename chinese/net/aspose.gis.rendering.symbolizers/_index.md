`Aspose.Gis.Rendering.Symbolizers` 命名空间提供用于配置渲染矢量要素的类。

## 类

| 类 | 描述 |
| --- | --- |
| [GeometryGenerator](./geometrygenerator/) | 装饰符号器以在渲染前修改要素的几何形状。 |
| [LayeredSymbolizer](./layeredsymbolizer/) | 一个渲染多个其他符号器的符号器。 |
| [MarkerCluster](./markercluster/) | 标记聚类符号器。 |
| [MarkerLine](./markerline/) | 标记线符号器。 |
| [MarkerPatternFill](./markerpatternfill/) | 标记图案填充符号器。 |
| [MixedGeometrySymbolizer](./mixedgeometrysymbolizer/) | 根据要素几何类型为要素几何应用正确的符号器。 |
| [NullVectorSymbolizer](./nullvectorsymbolizer/) | 该 `NullSymbolizer` 什么也不绘制，并有效地跳过对其所应用的几何体的渲染。 |
| [RasterImageMarker](./rasterimagemarker/) | 此符号器渲染提供的栅格图像。 |
| [Rule](./rule/) | 用户定义的规则用于 [`RuleBasedSymbolizer`](../aspose.gis.rendering.symbolizers/rulebasedsymbolizer/)。 |
| [RuleBasedSymbolizer](./rulebasedsymbolizer/) | 根据用户定义的规则，将符号器应用于要素几何体。 |
| [SimpleFill](./simplefill/) | 简单多边形符号器。 |
| [SimpleLine](./simpleline/) | 简单线符号器。 |
| [SimpleMarker](./simplemarker/) | 简单点符号器。 |
| [VectorSymbolizer](./vectorsymbolizer/) | 抽象根类用于渲染矢量要素的符号器。 |
