## AltitudeModes enumeration

KML 几何体的高度模式。

```csharp
public enum AltitudeModes
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 无高度模式。 |
| RelativeToGround | `1` | 将海拔解释为地面以上的米值。 |
| ClampToGround | `2` | 对于相机，此设置还将相机放置在 relativeToGround，因为将相机恰好放在地形高度会导致视线与地形相交（视图将被遮挡）。 |
| Absolute | `3` | 将海拔解释为海平面以上的米值。 |
