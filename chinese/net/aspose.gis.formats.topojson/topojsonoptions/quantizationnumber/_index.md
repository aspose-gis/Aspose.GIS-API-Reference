## TopoJsonOptions.QuantizationNumber property

指定用于量化输出 TopoJSON 中的坐标和增量编码弧的量化数。

```csharp
public int? QuantizationNumber { get; set; }
```

### 例外

| 例外 | （健康）状况 |
| --- | --- |
| ArgumentOutOfRangeException | 参数小于两个。 |

### 评论

这是写选项——它不影响读。 这个选项是互斥的[`Transform`](../transform) 这两个选项中只有一个不能`null`. 如果不是这样`null` - 输出 TopoJSON 坐标是量化的，弧是用指定的 量化数字进行增量编码的。量化数决定了结果量化坐标中每个维度 的最大可表达值数；通常选择十的幂。 默认为`null`.
