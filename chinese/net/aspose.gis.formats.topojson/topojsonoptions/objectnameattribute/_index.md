## TopoJsonOptions.ObjectNameAttribute property

属性名称，反映包含特征的对象的名称。

```csharp
public string ObjectNameAttribute { get; set; }
```

### 评论

TopoJSON 可能包含任意数量的命名对象。每个这样的对象都可以包含 个特征。有关命名对象的更多详细信息，请参阅 TopoJSON 规范。 在读取 TopoJSON 时，此属性指定应反映包含特征的对象的名称的属性。 在编写 TopoJSON 时，此属性指定应使用哪些属性对特征进行分组在 objects. 默认为“topojson_object_name”.
