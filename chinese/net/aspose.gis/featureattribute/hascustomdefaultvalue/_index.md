## FeatureAttribute.HasCustomDefaultValue property

获取一个值，指示此属性的预定义默认值是否被自定义值覆盖。

```csharp
public bool HasCustomDefaultValue { get; }
```

### Property Value

如果此实例具有自定义默认值，则为 `true`；否则为 `false`。

## 备注

除非调用了 Object)，此属性的值为 `false`。
