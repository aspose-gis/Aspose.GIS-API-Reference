## DriverOptions.ValidateGeometriesOnWrite property

确定在将几何图形添加到图层时是否应对其进行验证。 如果设置为`true`,[`IsValid`](../../../aspose.gis.geometries/geometry/isvalid/)在将每个 几何图形添加到图层时调用该几何图形，并且如果验证失败（[`IsValid`](../../../aspose.gis.geometries/geometry/isvalid/)是`false`),[`GisException`](../../gisexception/)被抛出.

```csharp
public bool ValidateGeometriesOnWrite { get; set; }
```

### 评论

这是一个创建选项 - 它不影响打开。
