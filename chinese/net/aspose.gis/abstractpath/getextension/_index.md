## AbstractPath.GetExtension method

返回此 [`AbstractPath`](../) 的扩展名。

```csharp
public string GetExtension()
```

### 返回值

此 [`AbstractPath`](../) 的扩展名（包括句点 "."），如果 [`AbstractPath`](../) 没有扩展名则返回空字符串。

## 示例

对于 `AbstractPath`，其 [`Location`](../location/) 为 `"/directory/file.txt"`，此属性返回 `".txt"`。
