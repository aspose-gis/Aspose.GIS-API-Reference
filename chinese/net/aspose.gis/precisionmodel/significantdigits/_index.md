## PrecisionModel.SignificantDigits property

如果精度模型进行四舍五入，则获取其有效数字的位数。

```csharp
public abstract int SignificantDigits { get; }
```

### Property Value

如果这是四舍五入的精度模型，则为有效数字的数量；如果是精确的精度模型，则为 -1。
