## Feature.SetValues method

为所有属性设置新值。 还可以考虑使用[`CopyValues`](../copyvalues)方法来简化一次调用中的设置值。

```csharp
public int SetValues(object[] values)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| values | Object[] | 新值数组。 |

### 返回值

设置的属性值的数量。

### 例外

| 例外 | （健康）状况 |
| --- | --- |
| ArgumentNullException | 参数不能是`null`。 |
| ArgumentException | 此层中不存在具有此名称的属性。 |
| InvalidOperationException | 属性未锁定。 |
| InvalidCastException | 值的类型没有实现IConvertible。 |
| FormatException | 转换失败，因为值格式不正确。 |
| OverflowException | 转换因溢出而失败。 |

### 评论

此方法自动将每个值转换为属性的类型。 &lt;br /&gt; values 数组的长度不需要与特征中的属性个数相匹配。 如果数组长度大于属性个数，则将所有数组值复制到属性中； 如果小于，则仅将数组长度个数的值复制到属性中， 从序数为 0 的属性值开始。
