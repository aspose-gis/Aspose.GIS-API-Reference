`Aspose.Gis.Rendering.Colorizers` 命名空间提供用于配置渲染栅格单元的类。

## 类

| 类 | 描述 |
| --- | --- |
| [BandColor](./bandcolor/) | 关联指定波段索引的波段值和颜色分量。波段值在最小值和最大值之间将线性插值。 |
| [MultiBandColor](./multibandcolor/) | 多波段颜色化器为栅格指定红、绿、蓝分量。 |
| [NullColorizer](./nullcolorizer/) | `NullColorizer` 不绘制任何内容，并有效地跳过其所应用的栅格单元的渲染。 |
| [RasterColorizer](./rastercolorizer/) | 用于渲染栅格的颜色化器的抽象根类。 |
| [SingleBandColor](./singlebandcolor/) | 单波段颜色化器为栅格指定灰度分量。 |
