## XyzTiles.GetTile method

```csharp
public WebTile GetTile(int zoom, int x, int y)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| zoom | Int32 | 加载图块的缩放级别。最高缩放级别为 0。大多数磁贴提供程序有大约 22 个最大缩放级别。 |
| x | Int32 | 磁贴的 x 列。 |
| y | Int32 | 瓷砖的 y 行。 |

### 返回值
