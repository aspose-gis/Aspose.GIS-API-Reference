## CsvOptions.HasAttributeNames property

确定是否存在包含属性名称的标题行。默认值为 `true`。

```csharp
public bool HasAttributeNames { get; set; }
```

## 备注

如果为 `true`，则 [`Attributes`](../../../aspose.gis/vectorlayer/attributes/) 将根据标题行命名。否则将命名为：`field_1, field_2, etc.`
