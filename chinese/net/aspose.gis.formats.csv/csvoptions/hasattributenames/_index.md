## CsvOptions.HasAttributeNames property

确定是否存在具有属性名称的标题行。 默认为`true`.

```csharp
public bool HasAttributeNames { get; set; }
```

如果`true`A[`Attributes`](../../../aspose.gis/vectorlayer/attributes/)将根据标题行命名。 否则将命名为：`field_1、field_2 等`
