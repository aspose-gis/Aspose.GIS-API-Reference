## FileGdbCoordinatePrecisionGrid.XOrigin property

获取或设置 X 坐标的原点。如果设置为 `null`，则使用默认值。

```csharp
public double? XOrigin { get; set; }
```

## 备注

默认值为 -400，适用于使用地理 [`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) 的 [`VectorLayer`](../../../aspose.gis/vectorlayer/)，以及 -2147483647，适用于使用投影 [`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) 的 [`VectorLayer`](../../../aspose.gis/vectorlayer/)。
