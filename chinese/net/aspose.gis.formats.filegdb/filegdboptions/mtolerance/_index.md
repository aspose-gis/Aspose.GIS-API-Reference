## FileGdbOptions.MTolerance property

M 捕捉容差。

```csharp
public double? MTolerance { get; set; }
```

## 备注

这是一个创建选项，不影响读取和编辑。此参数控制用于高级 ArcGIS 功能的捕捉容差。它不影响 Aspose.GIS 的行为，但可被 ArcGIS 使用。该参数的单位与空间参考系统的单位相同。如果设置为 `null`，将使用默认值。默认值为 0.001。
