## FileGdbOptions.CoordinatePrecisionGrid property

用于新图层的坐标精度网格。

```csharp
public FileGdbCoordinatePrecisionGrid CoordinatePrecisionGrid { get; set; }
```

## 备注

这是一个创建选项，不影响读取和编辑。坐标精度网格定义了 FileGDB 中坐标的有效范围和分辨率。当此属性设置为 `null` 时，将使用默认值。有关坐标精度网格的更多详情，请参阅 [`FileGdbCoordinatePrecisionGrid`](../../filegdbcoordinateprecisiongrid/) 文档。
