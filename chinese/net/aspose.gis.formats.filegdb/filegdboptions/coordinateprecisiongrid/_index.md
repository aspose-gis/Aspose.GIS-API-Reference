## FileGdbOptions.CoordinatePrecisionGrid property

在新图层中使用的坐标精度网格。

```csharp
public FileGdbCoordinatePrecisionGrid CoordinatePrecisionGrid { get; set; }
```

### 评论

这是一个创建选项，不影响读取。 坐标精度网格定义了FileGDB中坐标的有效域和分辨率。 当该属性设置为`null`使用默认值。 参考[`FileGdbCoordinatePrecisionGrid`](../../filegdbcoordinateprecisiongrid/)有关坐标 精度网格的更多详细信息的文档。
