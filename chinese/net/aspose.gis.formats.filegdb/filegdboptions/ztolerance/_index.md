## FileGdbOptions.ZTolerance property

Z 捕捉公差。

```csharp
public double? ZTolerance { get; set; }
```

### 评论

这是一个创建选项，不影响读取。 此参数控制用于高级 ArcGIS 功能的捕捉容差。 它不会影响 Aspose.GIS 的行为，但它可以被 ArcGIS 使用。 参数单位为空间参照系单位。 如果设置为`null`，则使用默认值。如果空间参考系统未知 或少于三个维度，则默认值为 0.001。如果空间参照系有三维 默认为 0.001 米转换为三维单位。
