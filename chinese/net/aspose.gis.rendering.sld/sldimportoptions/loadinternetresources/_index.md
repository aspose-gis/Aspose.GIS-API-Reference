## SldImportOptions.LoadInternetResources property

确定是否允许 Aspose.GIS 从 Internet 加载资源。 如果设置为`false` 具有不以“file://”开头的绝对 URI 的模式将不会被加载。 默认为`false`.

```csharp
public bool LoadInternetResources { get; set; }
```
