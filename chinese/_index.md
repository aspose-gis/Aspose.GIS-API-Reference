## Aspose.GIS for .NET

{{% alert color="primary" %}} 

![Aspose.GIS for .NET 产品徽标](../home_1.png)

Aspose.GIS for .NET 专为使 .NET 开发人员轻松处理以各种文件格式存储的地理空间数据而设计。 它允许您创建、读取、转换地理数据和渲染地图，而无需安装任何其他 GIS 软件。 Aspose.GIS 支持 Shapefile、GeoJSON、KML、ESRI 文件地理数据库和其他格式。
