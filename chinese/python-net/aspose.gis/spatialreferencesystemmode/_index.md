指定在数据库中写入空间参考系统 (SRS) 的模式<br/>            如果它是未知的 SRS。

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.SpatialReferenceSystemMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **成员名称** | **Description** |
| :- | :- |
| SETUP_TO_ZERO | 如果数据库中的 SRS 未知，将几何体的 SRID 设置为 'zero'。 |
| THROW_EXCEPTION | 如果数据库中的 SRS 未知，则抛出异常。 |
| WRITE_IN_SYSTEM_TABLE | 如果数据库中的 SRS 未知，将 SRS 信息写入系统表。 |
