[PointFormats](/psd/python-net/aspose.gis/pointformats/) are used to convert coordinates in text.

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.PointFormats

**Aspose.PSD Version:** 25.9.0

## **Members**
| **成员名称** | **Description** |
| :- | :- |
| DECIMAL_DEGREES | 十进制度 (DD) 格式。 |
| DEGREE_DECIMAL_MINUTES | 度十进制分钟 (DDM) 格式。 |
| DEGREE_MINUTES_SECONDS | 度-分-秒 (DMS) 格式。 |
| GARS | 全球区域参考系统 |
| GEO_REF | 世界地理参考系统。 |
| MAIDENHEAD | Maidenhead 定位系统（又称 QTH 定位器和 IARU 定位器） |
| MGRS | 带有 WGS 84 基准的军事网格参考系统。 |
| PLUS_CODE | 开放位置代码（OLC）或 Plus Code |
| UPS | 通用极地立体投影坐标系 |
| USNG | 带有 WGS 84 基准的美国国家网格。 |
| UTM | 通用横轴墨卡托投影 |
