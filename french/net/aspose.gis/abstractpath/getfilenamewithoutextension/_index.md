## AbstractPath.GetFileNameWithoutExtension method

Renvoie le nom de fichier de ce[`AbstractPath`](../) sans l'extension.

```csharp
public string GetFileNameWithoutExtension()
```

### Return_Value

La chaîne renvoyée par[`GetFileName`](../getfilename/) moins le dernier point et tous les caractères qui le suivent.

### Exemples

Pour un`RésuméChemin` avec[`Location`](../location/) égal à`"/répertoire/fichier.txt"` et[`Separator`](../separator/) égal à`/'` , cette méthode retourne`"déposer"` .
