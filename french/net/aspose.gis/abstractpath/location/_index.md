## AbstractPath.Location property

Obtient une représentation sous forme de chaîne de l'emplacement de ce`RésuméChemin` .

```csharp
public abstract string Location { get; }
```

### Remarques

Deux quelconques[`AbstractPath`](../) c'est égal`Location` pointent vers le même fichier ou répertoire. La représentation sous forme de chaîne doit utiliser[`Separator`](../separator/) pour séparer les niveaux de répertoire. Par exemple, pour le système de fichiers local, un emplacement est un chemin de système de fichiers, pour les blobs Azure, il peut s'agir d'un chemin d'accès au blob dans un conteneur de blobs.
