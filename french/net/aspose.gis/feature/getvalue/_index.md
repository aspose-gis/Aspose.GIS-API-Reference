```csharp
public T GetValue<T>(string attributeName)
```

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentNullException | Le nom de l'attribut est`null`. |
| ArgumentException | L'attribut portant ce nom n'existe pas dans cette couche. |
| InvalidOperationException | L'attribut n'est pas verrouillé. |

### Remarques

Cette méthode convertit automatiquement la valeur dans le type demandé dans le paramètre de type générique.  Si la couche n'exige pas que ses entités aient des valeurs pour tous les attributs définis pour la couche, cette méthode peut échouer avecInvalidOperationExceptionlorsqu'une valeur manquante est demandée. Lorsque vous travaillez avec de telles couches, envisagez d'utiliser[`GetValueOrDefault`](../getvalueordefault) .
