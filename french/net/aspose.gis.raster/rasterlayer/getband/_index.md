## RasterLayer.GetBand method

Obtient une bande par l'index spécifié.

```csharp
public abstract IRasterBand GetBand(int index = 0)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| index | Int32 | Les numéros de bande commencent à 0 et la bande est supposée être 0 si elle n'est pas spécifiée. |

### Return_Value

Renvoie des métadonnées de base sur une bande raster.

### Remarques

Si le raster ne contient aucune bande, une erreur est renvoyée.
