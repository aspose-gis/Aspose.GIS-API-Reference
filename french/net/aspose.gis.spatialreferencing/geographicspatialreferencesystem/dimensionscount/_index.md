## GeographicSpatialReferenceSystem.DimensionsCount property

Renvoie le nombre de dimensions dans ce SRS. Pour un SRS géographique, cela peut être : deux - s'il s'agit d'un SRS géographique unique. trois - s'il s'agit d'un SRS composé, qui se compose d'un SRS géographique à deux dimensions et d'un SRS vertical, qui ajoute une troisième dimension.

```csharp
public abstract int DimensionsCount { get; }
```
