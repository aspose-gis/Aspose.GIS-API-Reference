## AxisDirection enumeration

Axis direction définit la direction dans laquelle l'axe pointe.

```csharp
public enum AxisDirection
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Invalid | `0` | Valeur par défaut. |
| North | `1` | Axe pointant vers le nord. |
| South | `2` | Axe pointant vers le sud. |
| East | `3` | Axe pointant vers l'est. |
| West | `4` | Axe pointant vers l'ouest. |
| Up | `5` | Axe pointant vers le haut. |
| Down | `6` | Axe pointant vers le bas. |
| Other | `7` | Axe pointant vers une autre direction. Cela pourrait être l'axe 'X' dans le SRS géocentrique (il pointe vers le méridien principal). |
