## ProjectedSpatialReferenceSystemParameters.ProjectionMethodIdentifier property

Identifiant de la méthode de projection. Il n'y a pas de valeur par défaut, vous pouvez définir ce paramètre sur non`null` value, si vous souhaitez attacher un identifiant à la projection. Si vous le faites, c'est à vous de vous assurer que l'identifiant est cohérent avec le nom de la méthode de projection (le nom de la méthode de projection ne changera pas lorsque vous définirez cette propriété).

```csharp
public Identifier ProjectionMethodIdentifier { get; set; }
```
