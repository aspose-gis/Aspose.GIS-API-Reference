## CompoundSpatialReferenceSystem.AsGeographic property

Renvoie la représentation géographique de ce SRS. Si ce SRS composé représente effectivement un SRS géographique, le résultat sera un SRS géographique tridimensionnel (avec des dimensions de longitude, latitude, hauteur). Sinon, une exception sera levée.

public override GeographicSpatialReferenceSystem AsGeographic { get; }
