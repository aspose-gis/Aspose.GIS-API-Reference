## CompoundSpatialReferenceSystem.AsProjected property

Renvoie la représentation projetée de ce SRS. Si ce SRS composé représente effectivement un SRS projeté, le résultat sera un SRS projeté en trois dimensions (avec X, Y, dimensions de hauteur). Sinon, une exception sera levée.

```csharp
public override ProjectedSpatialReferenceSystem AsProjected { get; }
```
