## FileGdbOptions.MTolerance property

Tolérance d'accrochage M.

```csharp
public double? MTolerance { get; set; }
```

### Remarques

Il s'agit d'une option de création qui n'affecte pas la lecture. Ce paramètre contrôle une tolérance d'accrochage utilisée pour les entités ArcGIS avancées. Il n'affecte pas le comportement d'Aspose.GIS, mais il peut être utilisé par ArcGIS. L'unité du paramètre est l'unité du système de référence spatiale. Si défini sur`null` , la valeur par défaut est utilisée. La valeur par défaut est 0.001.
