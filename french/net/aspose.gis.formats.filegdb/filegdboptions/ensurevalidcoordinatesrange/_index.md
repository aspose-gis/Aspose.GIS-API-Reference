## FileGdbOptions.EnsureValidCoordinatesRange property

Si les coordonnées doivent être dans une plage valide.

```csharp
public bool EnsureValidCoordinatesRange { get; set; }
```

### Remarques

Ceci est une option de création et n'affecte pas la lecture. Si défini sur`true` une exception sera levée lors de la tentative d'écriture d'une coordonnée avec values hors de la plage valide. Si réglé sur`false` cette coordonnée sera écrite silencieusement. La plage valide est définie par[`CoordinatePrecisionGrid`](../coordinateprecisiongrid) . Faire référence à[`CoordinatePrecisionGrid`](../coordinateprecisiongrid) documentation pour lire comment la plage valide est déterminée à partir de la grille de précision des coordonnées. La valeur par défaut est`false` .
