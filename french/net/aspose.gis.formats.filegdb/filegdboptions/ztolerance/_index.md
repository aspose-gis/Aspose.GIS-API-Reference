## FileGdbOptions.ZTolerance property

Tolérance d'accrochage Z.

```csharp
public double? ZTolerance { get; set; }
```

### Remarques

Il s'agit d'une option de création qui n'affecte pas la lecture. Ce paramètre contrôle une tolérance d'accrochage utilisée pour les entités ArcGIS avancées. Il n'affecte pas le comportement Aspose.GIS, mais il peut être utilisé par ArcGIS. L'unité du paramètre est l'unité du système de référence spatiale. Si défini sur`null` , la valeur par défaut est utilisée. Si le système de référence spatiale est unknown ou a moins de trois dimensions, la valeur par défaut est 0,001. Si le système de référence spatiale a trois dimensions , la valeur par défaut est 0,001 mètre converti dans l'unité de la troisième dimension.
