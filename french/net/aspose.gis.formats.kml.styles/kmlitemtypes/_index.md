## KmlItemTypes enumeration

Spécifie comment un kml:Feature et son contenu doivent être affichés en tant qu'éléments dans une vue de liste.

```csharp
public enum KmlItemTypes
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| RadioFolder | `0` | Un seul des éléments doit être visible à la fois. |
| Check | `1` | La visibilité est liée à la case à cocher de son élément |
| CheckHideChildren | `2` | Utilisez une case à cocher normale pour la visibilité, mais n'affichez pas les enfants de l'élément dans la vue de liste. |
| CheckOffOnly | `3` | Empêche que tous les éléments soient rendus visibles en même temps, c'est-à-dire que l'utilisateur peut tout désactiver dans le groupe de conteneurs mais ne peut pas tout activer en même temps. |
