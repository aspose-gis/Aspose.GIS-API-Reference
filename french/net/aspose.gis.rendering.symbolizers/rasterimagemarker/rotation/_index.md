## RasterImageMarker.Rotation property

Spécifie la rotation du symbole autour de son point central, en degrés décimaux. Les valeurs positives indiquent une rotation dans le sens des aiguilles d'une montre, les valeurs négatives indiquent une rotation dans le sens inverse des aiguilles d'une montre.

public double Rotation { get; set; }
