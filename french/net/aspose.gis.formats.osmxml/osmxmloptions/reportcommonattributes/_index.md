## OsmXmlOptions.ReportCommonAttributes property

Signaler les attributs OSM communs : visible, version, changeset, timestamp, user et uid. Les attributs communs seront signalés comme des attributs de fonctionnalité avec le préfixe "osm_", par exemple osm_user, osm_timestamp, etc.

public bool ReportCommonAttributes { get; set; }
