## GeoJsonOptions.ReadBoundingBoxes property

Détermine si les boîtes englobantes ('bbox') doivent être lues comme des attributs avec un nom 'bbox_0', 'bbox_1', etc. La valeur par défaut est`false` . Le[`NestedPropertiesSeparator`](../nestedpropertiesseparator/) chaîne est utilisée dans bbox_0, bbox_1,.. noms.

```csharp
public bool ReadBoundingBoxes { get; set; }
```

### Remarques

Ceci est une option de lecture - cela n'affecte pas l'écriture.
