Spécifie comment une kml:Feature et son contenu doivent être affichés en tant qu'éléments dans une vue en liste.

**Module:** [aspose.gis.formats.kml.styles](/psd/python-net/aspose.gis.formats.kml.styles/)

**Full Name:** aspose.gis.formats.kml.styles.KmlItemTypes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nom du membre** | **Description** |
| :- | :- |
| CHECK | La visibilité est liée à la case à cocher de l'élément |
| CHECK_HIDE_CHILDREN | Utilisez une case à cocher normale pour la visibilité mais n'affichez pas les enfants de l'élément dans la vue liste. |
| CHECK_OFF_ONLY | Empêche tous les éléments d'être rendus visibles en même temps—c’est-à-dire,<br/>            l'utilisateur peut désactiver tout le groupe de conteneurs mais ne peut pas activer tout en même temps. |
| RADIO_FOLDER | Un seul des éléments doit être visible à la fois. |
