**Parameters:**

| Paramètre | Type | Description |
| :- | :- | :- |

| actions | [AttributesConverterActions](/psd/python-net/aspose.gis/attributesconverteractions) | Spécifie des actions optionnelles avec les attributs de la couche de destination.<br/>            Par exemple, permet d'exclure notre attribut. |
