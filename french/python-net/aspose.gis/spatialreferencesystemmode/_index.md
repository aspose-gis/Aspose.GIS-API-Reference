Spécifie un mode d'écriture du système de référence spatiale (SRS) dans la base de données<br/>            s'il s'agit d'un SRS inconnu.

**Module:** [aspose.gis](/psd/python-net/aspose.gis/)

**Full Name:** aspose.gis.SpatialReferenceSystemMode

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nom du membre** | **Description** |
| :- | :- |
| SETUP_TO_ZERO | Configurez le SRID d'une géométrie à 'zero' s'il s'agit d'un SRS inconnu pour la base de données. |
| THROW_EXCEPTION | Lancez des exceptions si le SRS est inconnu pour la base de données. |
| WRITE_IN_SYSTEM_TABLE | Écrivez les informations SRS dans la table système si le SRS est inconnu pour la base de données. |
