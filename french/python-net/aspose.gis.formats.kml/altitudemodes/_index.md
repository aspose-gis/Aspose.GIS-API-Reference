Mode d'altitude pour les géométries KML.

**Module:** [aspose.gis.formats.kml](/psd/python-net/aspose.gis.formats.kml/)

**Full Name:** aspose.gis.formats.kml.AltitudeModes

**Aspose.PSD Version:** 25.9.0

## **Members**
| **Nom du membre** | **Description** |
| :- | :- |
| ABSOLUTE | Interprète l'altitude comme une valeur en mètres au-dessus du niveau de la mer. |
| CLAMP_TO_GROUND | Pour une caméra, ce paramètre place également la caméra relativeToGround, <br/>            car placer la caméra exactement à la hauteur du terrain signifierait que <br/>            l'œil intersecterait le terrain (et la vue serait bloquée). |
| AUCUN | Sans mode d'altitude. |
| RELATIVE_TO_GROUND | Interprète l'altitude comme une valeur en mètres au-dessus du sol. |
