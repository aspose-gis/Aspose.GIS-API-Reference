## TopoJsonOptions.ObjectNameAttribute property

기능을 포함하는 개체의 이름을 반영하는 특성의 이름입니다.

```csharp
public string ObjectNameAttribute { get; set; }
```

### 비고

TopoJSON에는 명명된 개체가 얼마든지 포함될 수 있습니다. 그러한 모든 개체는 여러 기능을 포함할 수 있습니다 . 명명된 개체에 대한 자세한 내용은 TopoJSON 사양을 참조하세요. TopoJSON을 읽을 때 이 속성은 기능을 포함하는 개체의 이름을 반영해야 하는 속성을 지정합니다. TopoJSON을 작성할 때 이 속성은 기능을 그룹화하는 데 사용해야 하는 속성을 지정합니다. in objects. 기본값은 "topojson_object_name"입니다.
