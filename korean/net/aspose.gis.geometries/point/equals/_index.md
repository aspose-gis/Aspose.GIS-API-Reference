## Equals(object) {#equals_1}

지정된 개체가 현재 개체와 같은지 여부를 확인합니다.

```csharp
public override bool Equals(object obj)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| obj | Object | 현재 개체와 비교할 개체입니다. |

### 반환 값

지정된 개체가 현재 개체와 같으면 true이고, 그렇지 않으면 false입니다. 그렇지 않으면 거짓입니다.
