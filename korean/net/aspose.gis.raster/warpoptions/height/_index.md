## WarpOptions.Height property

출력 래스터 높이를 픽셀 및 열 단위로 지정합니다. 값이 0으로 설정되면 높이가 자동으로 계산됩니다. 기본값은 "0"입니다.

```csharp
public int Height { get; set; }
```

### 비고

높이가 0으로 설정되면 값은 원래 높이에서 가져오거나[`CellHeight`](../cellheight/) . 참고`Height` 함께 사용할 수 없습니다[`CellHeight`](../cellheight/) .
