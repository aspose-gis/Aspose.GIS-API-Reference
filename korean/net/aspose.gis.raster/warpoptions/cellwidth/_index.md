## WarpOptions.CellWidth property

래스터 셀의 새 너비를 지정합니다(대상 지리 참조 단위). 값이 0으로 설정되면`CellWidth` 자동으로 계산됩니다. 기본값은 "0"입니다.

```csharp
public double CellWidth { get; set; }
```

### 비고

셀 너비가 0으로 설정되면 값은 원래 셀 너비에서 가져오거나[`Width`](../width/) . 참고`CellWidth` 함께 사용할 수 없습니다[`Width`](../width/) .
