## WarpOptions.CellHeight property

래스터 셀의 새 높이를 지정합니다(대상 지리 참조 단위). 값이 0으로 설정되면`CellHeight` 자동으로 계산됩니다. 기본값은 "0"입니다.

```csharp
public double CellHeight { get; set; }
```

### 비고

셀 높이가 0으로 설정되면 값은 원래 셀 높이에서 가져오거나[`Height`](../height/) . 참고`CellHeight` 함께 사용할 수 없습니다[`Height`](../height/) .
