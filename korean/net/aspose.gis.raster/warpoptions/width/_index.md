## WarpOptions.Width property

출력 래스터 너비를 픽셀 및 열 단위로 지정합니다. 값이 0으로 설정되면 너비가 자동으로 계산됩니다. 기본값은 "0"입니다.

```csharp
public int Width { get; set; }
```

### 비고

너비가 0으로 설정되면 값은 원래 너비에서 가져오거나[`CellWidth`](../cellwidth/) . 참고`Width` 함께 사용할 수 없습니다[`CellWidth`](../cellwidth/) .
