## SpatialReferenceSystem.CreateGeocentric method

맞춤형 매개변수에서 지구 중심 SRS를 생성합니다.

```csharp
public static GeocentricSpatialReferenceSystem CreateGeocentric(
    GeocentricSpatialReferenceSystemParameters parameters, Identifier identifier = null)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| parameters | GeocentricSpatialReferenceSystemParameters | 생성할 매개변수입니다. |
| identifier | Identifier | SRS에 첨부될 식별자. 식별자를 첨부해도 다른 SRS 매개변수는 수정되지 않습니다. 식별자와 SRS 매개변수의 일관성을 유지하는 것은 사용자의 책임입니다. |

### 반환 값

새로운 지오센트릭 SRS.
