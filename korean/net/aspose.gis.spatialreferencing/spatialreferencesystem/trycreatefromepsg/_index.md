## SpatialReferenceSystem.TryCreateFromEpsg method

지정된 EPSG 코드를 기반으로 공간 참조 시스템을 생성합니다.

```csharp
public static bool TryCreateFromEpsg(int epsg, out SpatialReferenceSystem value)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| epsg | Int32 | 공간 참조 시스템의 EPSG 코드입니다. |
| value | SpatialReferenceSystem& | 이 메서드가 반환될 때`true` , 지정된 EPSG 코드가 있는 SRS를 포함합니다. 그렇지 않으면 포함`null` . |

### 반환 값

`true` 지정된 EPSG 코드가 알려져 있고 SRS가 생성된 경우;`false` 그렇지 않으면.
