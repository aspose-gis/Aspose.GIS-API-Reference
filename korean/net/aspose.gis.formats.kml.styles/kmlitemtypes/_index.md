## KmlItemTypes enumeration

kml:Feature 및 해당 콘텐츠가 목록 보기의 항목으로 표시되는 방식을 지정합니다.

```csharp
public enum KmlItemTypes
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| RadioFolder | `0` | 한 번에 하나의 항목만 볼 수 있습니다. |
| Check | `1` | 가시성은 해당 항목의 확인란에 연결되어 있습니다. |
| CheckHideChildren | `2` | 가시성을 위해 일반 확인란을 사용하되 목록 보기에 항목의 하위 항목을 표시하지 마십시오. |
| CheckOffOnly | `3` | 모든 항목이 한 번에 표시되지 않도록 합니다. 즉, 사용자가 컨테이너 그룹의 모든 항목을 끌 수 있지만 동시에 모든 항목을 켤 수는 없습니다. |
