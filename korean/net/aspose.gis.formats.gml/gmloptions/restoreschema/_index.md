## GmlOptions.RestoreSchema property

Aspose.GIS가 XML 스키마가 누락되었거나 로드할 수 없는 Gml 파일의 속성을 구문 분석할 수 있는지 여부를 결정합니다. 다음으로 설정한 경우`true` , Aspose.GIS 리더는 XML 스키마의 존재를 요구하지 않습니다. 기본값은`false` .

```csharp
public bool RestoreSchema { get; set; }
```

### 비고

드라이버는 조직을 결정하기 위해 파일을 스캔하고 "알려진" 객체를 찾아 피쳐 클래스 및 관련 속성 를 자동으로 구문 분석하려고 시도합니다. XML 스키마가 손실되었거나 인터넷에서 로드할 수 없습니다.
