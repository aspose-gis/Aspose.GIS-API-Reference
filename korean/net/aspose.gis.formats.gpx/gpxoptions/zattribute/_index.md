## GpxOptions.ZAttribute property

웨이포인트, 루트 포인트 및 트랙 포인트의 'Z' 좌표로 내보낼 GPX 속성을 결정합니다. If`null` 어떤 속성도 'Z' 좌표로 내보내지지 않습니다. 기본값은 "ele"입니다. 가능한 값은 이중으로 표현될 수 있는 모든 GPX XML 속성의 이름입니다(예: "speed", "magvar", "geoidheight" 등). )

```csharp
public string ZAttribute { get; set; }
```
