```

| --- | --- | --- |

| ArgumentNullException | 경로는`null`. |
| [GisException](../../gisexception/) | 데이터 세트를 만드는 동안 오류가 발생했습니다. |
| IOException | I/O 오류가 발생했습니다. |
| NotSupportedException | 드라이버가 데이터 세트를 열 수 없습니다. |
| InvalidOperationException | 데이터세트가 이미 존재합니다. |
