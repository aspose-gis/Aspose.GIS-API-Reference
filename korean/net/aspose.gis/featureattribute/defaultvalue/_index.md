## FeatureAttribute.DefaultValue property

누락된 데이터를 나타내는 특성 값을 가져오거나 설정합니다.

```csharp
public object DefaultValue { get; set; }
```

### 예외

| 예외 | 상태 |
| --- | --- |
| InvalidOperationException | 속성이 잠겨 있습니다. |
| InvalidOperationException | 속성이 허용하지 않습니다.`없는` 가치. |

### 비고

속성이 허용하지 않는 경우 누락된 정보를 나타내는 값입니다.`없는`value. 허용하는 속성의 경우`없는` 값([`CanBeNull`](../canbenull/) ==`진실` ),`DefaultValue` ~이다`없는` 명시적으로 변경하지 않는 한.
