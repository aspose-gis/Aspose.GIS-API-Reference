## FeatureAttribute.CanBeUnset property

이 특성의 값을 생략할 수 있는지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool CanBeUnset { get; set; }
```

### 자산 가치

`true` 이 인스턴스를 설정 해제할 수 있는 경우; 그렇지 않으면,`false` .
