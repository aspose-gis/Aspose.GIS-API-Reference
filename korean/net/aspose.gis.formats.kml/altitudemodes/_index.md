## AltitudeModes enumeration

KML 지오메트리에 대한 고도 모드.

```csharp
public enum AltitudeModes
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | `0` | 고도 모드가 없습니다. |
| RelativeToGround | `1` | 고도를 지면 위의 미터 단위 값으로 해석합니다. |
| ClampToGround | `2` | 카메라의 경우 이 설정은 카메라를 지면에 상대적인 에 배치합니다. 카메라를 정확히 지형 높이에 놓으면 눈이 지형과 교차하고 보기가 차단됨을 의미하기 때문입니다. |
| Absolute | `3` | 고도를 해발 미터 단위의 값으로 해석합니다. |
