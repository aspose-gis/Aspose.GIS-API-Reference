## FileGdbOptions.MTolerance property

M 스냅 공차.

```csharp
public double? MTolerance { get; set; }
```

### 비고

이것은 생성 옵션이며 읽기에 영향을 미치지 않습니다. 이 매개변수는 고급 ArcGIS 기능에 사용되는 스냅 허용 오차를 제어합니다. Aspose.GIS 동작에는 영향을 미치지 않지만 ArcGIS에서 사용할 수 있습니다. 매개변수의 단위 는 공간 참조 시스템의 단위입니다. 로 설정된 경우`null` , 기본값이 사용됩니다. 기본값은 0.001. 입니다.
