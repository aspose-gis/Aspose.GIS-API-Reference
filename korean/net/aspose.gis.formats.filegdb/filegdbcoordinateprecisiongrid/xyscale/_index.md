## FileGdbCoordinatePrecisionGrid.XYScale property

X 및 Y 좌표의 배율을 가져오거나 설정합니다. 로 설정한 경우`null` 기본값이 사용됩니다.

```csharp
public double? XYScale { get; set; }
```

### 예외

| 예외 | 상태 |
| --- | --- |
| ArgumentOutOfRangeException | 인수가 긍정적이지 않습니다. |

### 비고

기본값은`1e9` ~을 위한[`VectorLayer`](../../../aspose.gis/vectorlayer/)지리적으로[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) 및`XY스케일 = 1 / XYTolerance * 10` ~을 위한[`VectorLayer`](../../../aspose.gis/vectorlayer/) with 예상됨[`SpatialReferenceSystem`](../../../aspose.gis.spatialreferencing/spatialreferencesystem/) .
