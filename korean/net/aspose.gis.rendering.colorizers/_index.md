`Aspose.Gis.Rendering.Colorizers` 네임스페이스는 렌더링 래스터 셀을 구성하기 위한 클래스를 제공합니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [BandColor](./bandcolor/) | 지정된 밴드 인덱스에 대한 밴드 값과 색상 구성 요소를 연결합니다. 최소값과 최대값 사이의 밴드 값이 선형으로 보간됩니다. |
| [MultiBandColor](./multibandcolor/) | 멀티밴드 컬러라이저는 래스터의 빨간색, 녹색 및 파란색 구성 요소를 지정합니다. |
| [NullColorizer](./nullcolorizer/) | `NullColorizer` 아무 것도 그리지 않고 적용된 래스터 셀의 렌더링을 효과적으로 건너뜁니다. |
| [RasterColorizer](./rastercolorizer/) | 래스터를 렌더링하는 colorizer의 추상 루트 클래스입니다. |
| [SingleBandColor](./singlebandcolor/) | 단일 밴드 컬러라이저는 래스터의 회색 구성 요소를 지정합니다. |
