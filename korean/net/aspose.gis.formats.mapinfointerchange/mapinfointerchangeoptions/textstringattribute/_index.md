## MapInfoInterchangeOptions.TextStringAttribute property

'텍스트' 그래픽 객체의 텍스트를 나타내는 속성의 이름을 지정합니다.

```csharp
public string TextStringAttribute { get; set; }
```

### 비고

MapInfo 교환 형식은 '텍스트' 유형의 그래픽 개체를 지정합니다. '텍스트' 그래픽 개체는 지도의 레이블을 나타냅니다. 우리는 '텍스트' 그래픽 개체를 a로 내보냅니다.[`Feature`](../../../aspose.gis/feature/) ~와 함께[`Polygon`](../../../aspose.gis.geometries/polygon/) label. 레이블의 텍스트를 다음과 같이 내보냅니다.[`FeatureAttribute`](../../../aspose.gis/featureattribute/) . 이 속성은 레이블의 텍스트를 내보내는 데 사용되는 속성의 이름을 지정합니다. 기본값은 다음과 같습니다.`"문자열"` .
